        听着女生这么一说，童筱颖顿时就明白这小丫头认错了，刚准备笑着解释，结果这小丫头又滔滔不绝起来。
<br/>
<br/>        “哎——小时候，我还发誓要嫁给大哥的。没想到，大哥现在都有未婚妻了！不过，我见你长得比我漂亮，那么，大哥会喜欢你，我也就无话可说了。”女生一边摇着头，一边唉声叹气地说。
<br/>
<br/>        虽然不知道这女生和大哥傅夜沉是什么关系，但听着和女生管傅夜沉也叫做“大哥”，童筱颖心里估摸着这女生八成是大哥傅夜沉的堂妹。
<br/>
<br/>        既然是堂妹，那也就是傅昼景的堂妹。
<br/>
<br/>        童筱颖想到这里，嘴角微扬，忍不住跟这小姑娘八卦起傅昼景以前的事情来：“你喜欢大哥啊？我觉得你二哥傅昼景人也挺帅的啊！那你为什么不喜欢你二哥？”
<br/>
<br/>        “我才不喜欢二哥那个虐猫狂！”女生撅起嘴来，嗤之以鼻。
<br/>
<br/>        童筱颖身心一怔，脸上的笑容渐敛。
<br/>
<br/>        一提起傅昼景，这个小女生似乎像是受了什么天大的委屈一般，开始控诉道：“每次我帮我们班的女生，给大哥送小礼物，都被二哥给没收了。而且，我还亲眼看到过二哥将大庄园里的那一窝被母野猫抛弃的小野猫给一只只的掐死了。二哥简直就是个大变态！我才不喜欢他那种大变态！”
<br/>
<br/>        “那时候，你二哥多大？”童筱颖讷讷地问。
<br/>
<br/>        小女生口无遮拦地说道：“不大也不小了，他们上高中的时候吧！大哥和二哥放暑假回来会住在傅家大庄园里，我们这群小的，都爱去傅家大庄园里找大哥玩。大哥虽然对待那些追求他的女生冷酷，但是对我们这群堂弟堂妹却是好得没话说。不像二哥，喜欢在背地里玩阴的。我记得有一次，大哥念初中的时候，是那年的暑假，我们缠着大哥，要他陪我们一起玩捉迷藏，我躲了许久未见到大哥来找我们，于是我自个儿去找大哥，结果亲眼看到二哥故意将用布带蒙着双眼的大哥推入了荷花池里。我大哥那时候还不会游泳，幸好二哥他良心发现，又把大哥救了上来，不然我大哥真的就差点被淹死了。事后，我偷偷地把这件事情告诉给大伯公（傅夜沉的爷爷）听，大伯公私底下还把二哥痛打了一顿！这事，也就只有我和大伯公知道。连大哥他自己都不知道！”
<br/>
<br/>        “不会的，昼景不是那样的人。”童筱颖听着这个女生的话，心里慌得厉害，替傅昼景辩解道。
<br/>
<br/>        “我说的都是真的！所以啊！大嫂，你以后要帮着大哥小心防着点二哥。我是看在你是大哥的女人的份上，我才跟你说的噢！”女生仰起下巴，噘着嘴，一本正经地叮嘱。
<br/>
<br/>        童筱颖却不以为然地回答道：“那都是小时候的事情了，现在大家都是成年人了。不会再做那些幼稚的事情。”
<br/>
<br/>        “你这个女人真奇怪，你怎么一心向着二哥啊？”女生忍不住朝童筱颖翻了个白眼，“我懒得跟你说了，我先走啦！”
<br/>
<br/>        童筱颖干笑着点了点头。
<br/>
<br/>        这个女生，应该是小叔公的孙女或者是外孙女吧！
<br/>
<br/>        不过，她说的关于昼景的那些事……
<br/>
<br/>        童筱颖微微摇了摇头。
<br/>
<br/>        一定是假的！
<br/>
<br/>        小时候的昼景，她也接触过啊！
<br/>
<br/>        只是，童筱颖忘了，被接去傅家大庄园的昼景，已经不再是她所认识的那个傲娇“小卫”了。
<br/>
<br/>        当童筱颖回到酒桌前时，傅昼景见她去了那么久，还以为她是吃坏肚子了，便温柔地说道：“筱颍，待会晚上，我带你去别处再吃一餐我们俩都爱吃的。”
<br/>
<br/>        “小卫也不喜欢吃这酒店的饭菜吗？”童筱颖忍俊不禁地问。
<br/>
<br/>        傅昼景凑到童筱颖的耳边，轻轻地说道：“其实了，不管是寿宴还是喜宴，一般来说，就算是再高档的酒店，厨师要在一定的时间内现做或者是提前做出这么多份出来，味道肯定都不会很好。”
<br/>
<br/>        “那待会我们去吃临海城的小吃？临海城有一条小吃街，超级有名的！”童筱颖笑着说道。
<br/>
<br/>        “你似乎对临海城，比我还了解。”傅昼景倒是很意外地看着童筱颖。
<br/>
<br/>        童筱颖抬起手来，轻轻地遮着嘴巴，微笑道：“因为，我在重新参加高考前，是在这里的一所大专学校读了一个学期的大一啊！所以，哪里的东西好吃又实惠，我都清楚！”
<br/>
<br/>        “那我们吃过小吃后，晚上我再带你去临海大学里逛逛，提前感受一下这数一数二的本科大学的校园风气。”傅昼景莞尔一笑。
