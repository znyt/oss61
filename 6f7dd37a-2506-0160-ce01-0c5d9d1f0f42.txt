　　说着曹春辉又继续道：“但是这世界上的丧尸也太多了，总是这么放着也不是办法，环境不好又影响呼吸的空气，所以还是尽可能的减少它们的数量。”
<br/>　　“感染病毒的解毒剂，当然是给那些替罗家办事的人用的！比方说那些培养出来的精英战士们，好不容易到了八·九阶，却一不小心感染了，这时候解毒剂就有用了啊！”
<br/>　　在曹春辉滔滔不绝的述说中，在场的楚云天却如遭雷击，因为他听到了一个名字，一瞬间牵动了他的情绪不可抑止的爆发。
<br/>　　楚涵？
<br/>　　曹春辉刚刚口中说的，是楚涵？！
<br/>　　胡梦浩一直注意着楚云天的神色，当下看到他面色大变，反应极快的在当下开口：“狼牙管辖范围内的安罗市，其城市的丧尸清理很早就已经完成，所以我猜丧尸引诱剂出现的时间，可能比我们知道的更早。”
<br/>　　“由此可见，楚涵其实早就将大量资源用在了狼牙研究部上，这个部门闷不吭声的崛起并非巧合，其内一定有很多能力超群的研究人员，而以狼牙研究部的能力，在解毒剂上也没任何消息说有进展，其能否研发出来的可能性还是微乎其微。”
<br/>　　胡梦浩说的安罗市而非当下的安罗城，这一细节上的失误无人察觉，但他却刻意如此，目的就是要提醒楚云天这个安罗市和狼牙的关系，让其确定听到的这个楚涵，就是他的儿子。
<br/>　　“你这话是什么意思？你是觉得我们曹氏研究院不如他狼牙研究部？！”曹春辉黑着脸，恨不得当场将胡梦浩砍死。
<br/>　　他原本已经将话题拉到了正确轨道，而此时胡梦浩的一打岔，又将事态拉回原状不说，甚至还倒黑了曹氏研究院一把。
<br/>　　胡梦浩有些受惊，瞥了一眼罗杰后不敢说话。
<br/>　　罗杰将两人的神色尽收眼底，随后冷笑：“这位副院长说的每次，我花了大价钱让你们曹氏研究院发展，却根本比不上楚涵的狼牙研究部。你曹春辉不是个厉害人物呢，找了这么多研究人员过来，各个都名气巨大，可是狼牙研究部内的那些人名声都不显，怎么就能研发出你们毫无头绪的东西呢？”
<br/>　　曹春辉哑口无言，无法应对的他在心中将胡梦浩骂了千百遍。
<br/>　　心思不纯的胡梦浩很懂把握节奏，这时候又看似鼓起勇气的样子开口道：“其实我觉得正如这位大人所说，解毒剂哪怕真能研发出来也没什么用，以大人的能力哪里需要这些东西？我们倒不如将研究的方向改为丧尸之间的交流原理，以及能否将其彻底控制，就像异种能做到的那样。”
<br/>　　此话一出，曹春辉呆愣原地，惊讶的望着胡梦浩，不明白这个平时沉默寡言的副院长，怎么今日忽然思维如此敏捷？
<br/>　　罗杰原本也没指望这些人能说出什么让他感兴趣的话题，但听到这一番话，也忍不住对胡梦浩高看一眼。
<br/>　　曹春辉的提议完全就是屁话，而胡梦浩的想法，才真正契合他罗家的野心。
<br/>　　解毒剂？
<br/>　　他要这种东西有什么用？普通人的死活关他什么事？
<br/>　　但若能统治丧尸，趋势这世上数量最多的活死人军团行动，他们罗家才能在四大神秘家族中地位更高。
<br/>　　“你叫什么名字？”罗杰望着这个断了双·腿，坐在轮椅上的年轻人。
<br/>　　“胡梦浩。”胡梦浩如实回答，随后又添加了一句：“是曹氏研究院的副院长，曹院长的副手。”
<br/>　　曹春辉猛地一个激灵，有种不好的预感冒出，他连忙开口：“罗杰少爷！这个胡梦浩一点能力都没有，这么长时间了在我交给他的研发项目上毫无进展，这次的想法也是他随口一说，恐怕连方向还没有！”
<br/>　　罗杰不满的看了曹春辉一眼：“我在说话的时候，什么时候轮得到你来指手画脚？”
<br/>　　曹春辉一惊，急忙闭嘴不敢再出声。
<br/>　　但经过了曹春辉的提醒，罗杰也在这时候起了一丝试探心思，他看着胡梦浩问道：“关于控制丧尸的研究，你有什么方向？”
<br/>　　这句问话完全外行，但回答的不好，也能轻而易举的将之前在罗杰心中树立起来的好形象颠覆。
<br/>　　曹春辉低着头，嘴角嘲讽的勾起，他可不认为胡梦浩真的有方向，毕竟这一临时提议，他们研究院在此之前根本没有想过，没有思考过的新项目，哪来的研究方向？
<br/>　　但让所有人都没想到的是，胡梦浩几乎没有什么拖延就回答：“是有的。”
<br/>　　曹春辉猛然抬头，眼中满是不可置信。
<br/>　　刚从爆发的情绪中回归镇定的楚云天，此时也忍不住被胡梦浩吸引。
<br/>　　罗杰眉头一挑：“说下去。”
<br/>　　胡梦浩虽然立即给出了肯定回答，但其实不过是在拖延时间，正如曹春辉弩定的猜测那般，刚刚那番话还真是他随口一说，目的压根不在此处，初衷不过是为了提醒楚云天，没想到却引来了罗家人的看中。
<br/>　　紧急之下，胡梦浩忽然脱口而出：“方向在楚涵，在狼牙那里。”
<br/>　　这没头没脑的话让屋内一静，罗杰面上的表情很不好，也即将失去耐心。
<br/>　　楚云天是在胡梦浩连着提到的‘楚涵’二字中，愈发肯定这个楚涵不是重名，恐怕就是他儿子，并且胡梦浩还是知情者，否则不会在如此关键的时刻反复提及。
<br/>　　曹春辉却咧嘴露出了一个阴狠笑容，道：“胡梦浩，想不出来就想不出来，别在这里故弄玄虚。”
<br/>　　胡梦浩心脏跳动的有些快，但依然维持着表明镇定，直视着前方的罗杰开口：“我说的一切属实，大人您虽然贵不可攀，但在研究这一方面太外行，没能明白我的意思情有可原，但曹春辉院长竟然也没明白，就让我有些失望了。”
<br/>　　没有头绪，也不忘再黑曹春辉一把，胡梦浩这一手直接将故弄玄虚玩的登峰造极。
<br/>　　被说是外行，罗杰当场大怒：“你胆子大的可以啊！好啊，那你倒是换个说法解释解释，让我明白你所说的方向在楚涵那里，是何意？”
