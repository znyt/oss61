<br/>　　敢情您老是来给我们讲你年轻时的风流事？
<br/>　　赵家人自己也懵了，老爷子是今天受刺激受多了？
<br/>　　倏然地，周围的人都悄悄的交头接耳起来，要不是赵老爷子威望高，搁一般人，早就起哄了。
<br/>　　但赵老爷子已经做了心里准备，目光远望，继续接下来说：“很快，欣兰她怀孕了，怀了我的孩子，我就将她安置在了赵家的后院中”
<br/>　　“可那时候的我是有老婆的，我老婆发现了我跟欣兰的事情，也知道她怀孕，担心我会跟她离婚，就瞒着我，在欣兰待产的时候，用被子将她活活的闷死了，一尸二命”
<br/>　　说到这里，赵老爷子充满了悔恨，不知道是恨自己的老婆太恶毒，还是恨自己没保护好欣兰跟她的孩子。
<br/>　　霍...
<br/>　　故事讲到这里，所有人撑着惊骇的眼神望向轮椅上的老者，整个空间变得寂静无声，以为是风流事，转眼变成了凶杀案？
<br/>　　他的情妇跟孩子被自己的老婆给谋杀了？
<br/>　　这信息量也太大了点吧。
<br/>　　赵家的孙子孙女相视的看了一眼，瞠目结舌：“爷爷说我们的奶奶杀人了，是杀人犯？”
<br/>　　他们其实根本没有见过自己的奶奶，只知道自己的奶奶很久以前就过世了，至于怎么死了，也从没听过。
<br/>　　此时的赵老爷子赵昌民说着说着，反而轻松了，这压抑在心中五十年的秘密，就如一座大山压在他心底，怎么多年一直守着这个秘密度日，也不是那么好过。
<br/>　　如今一说出来，他顿时觉得浑身都轻松了，甚至感觉自己年轻了十几岁。
<br/>　　只不过接下来那段故事，就有点让他难以启齿，可谓人神共愤。
<br/>　　他隐晦的抬了抬头，村民的最上空，一条金银蛇诡异的横陈虚空，用一双冰冷的蛇眸望着他。
<br/>　　他知道，自己只能继续的说下去。
<br/>　　“欣兰跟我未出世的孩子死后，我赵家就怪事不断，没多久我那个老婆也死了，可谓报应”
<br/>　　赵昌民唏嘘的叹了一口气：“家中怪事连连，我请了一个道士，道士说那是欣兰死不瞑目，要想赵家安康，就要将欣兰的尸体挖出来，用她的尸骨做成红心木，不但可以驱邪，还可以让她的鬼魂化为家蛇，帮我赵家看宅积福”
<br/>　　“我照做了，将欣兰的尸骨做成了红心木，高高悬挂在我赵家的房梁上”
<br/>　　然而，现场的所有人顿时目瞪口呆，只觉得脊梁上流下一股股的冷汗。
<br/>　　尸骨做成红心木，鬼魂化为家蛇？
<br/>　　对白镇人民来说，家蛇是善良的蛇，会守护，会帮你看宅镇福，更是蛇王的化身。
<br/>　　但其实还有一层背后的意义，就是人们认为家蛇是自己死去的亲人所化，所以一定不能杀死，否则会有报应。
<br/>　　这是一段传说，说捕蛇人要是被毒蛇咬死，不想离开人世，可以化为家蛇守护。
<br/>　　但是其中的代价就是从此失去六道轮回的资格，永世不得超生。
<br/>　　这个代价太大，没有人会愿意。
<br/>　　赵昌民听了道士的话，将自己惨死的女人尸骨做成红心木，逼着化为家蛇，守护赵家。
<br/>　　这太残忍了。
<br/>　　赵欣兰惨死在赵家，还要被逼着化为家蛇，为赵家看宅积福，永世不得超生。
<br/>　　阴毒，无比的阴毒，好狠的心呀。
<br/>　　须根之间，村民们好像终于知道了赵家的家蛇为什么会回来杀赵家的人。
<br/>　　原来是那位可怜女子的报复。
<br/>　　谜题瞬间得到了完美的解释。
<br/>　　“老爷子，你怎么能如此对待自己心爱的女人，她跟自己的孩子都被你老婆杀死了，你还将她的尸骨做成红心木，简直禽兽不如”
<br/>　　“赵昌民，你是不是人，如此阴邪的手段，你也能做的出来，你有没有良心”
<br/>　　“家蛇来杀你们赵家人，杀的好”
<br/>　　“如若换成我，我也会来报仇”
<br/>　　现场一时之间，一个个变的义愤填膺，满是怒火。
<br/>　　那位可怜的女人赵欣兰，太惨了，人间悲剧。
<br/>　　村民的理解跟事情的真相是有点误差，但这无关紧要，他们也不需要知道事情的全部真相，他们只要知道赵欣兰死的冤，死的惨，是一位可悲的女人，需要为她伸张正义。
<br/>　　德高望重的赵老爷子，顿时间变成过街老鼠，人人喊骂，人人喊打的地步。
<br/>　　就算是一个普通人，你也不能将她的尸骨做成红心木，让别人死了都不能超生，更不要说是为了你而死的女人。
<br/>　　别人怀了你的孩子，被你的妻子杀了，你如此待她?
<br/>　　禽兽不如。
<br/>　　“赵昌民，你个畜生，那女人为你生孩子，但被你老婆杀了，一尸二命，你还要将她尸骨做成红心木，你简直就是一个天大的畜生，最该死的那个人是你，你有老婆还去招惹别人，最后让她落得如此凄惨的下场”
<br/>　　一个老妇女暴怒，一颗鸡蛋狠狠的砸到了赵老爷子的脸上，蛋壳破裂，蛋黄四溅。
<br/>　　“我是罪人，我有错，这五十年的秘密我现在说出来了，请求大家的原谅”
<br/>　　赵老爷子激动的站了起来，顿时间，他自己都楞了，紧接着所有人都愣了。
<br/>　　中风瘫痪的赵老爷子，双腿早就失去知觉，做了两年的轮椅，现在突然站了起来。
<br/>　　这一幕可谓石破天惊，如一枚重磅炸弹投入了现场，将所有人都炸懵了。
<br/>　　旋即大家皆露出古怪的神色，都说好人不长命，祸害遗千年。
<br/>　　赵老爷子如此对待曾经心爱的女人，不但活到了八十二岁，现在连中风的双腿还神奇的恢复了。
<br/>　　老天无眼呀。
<br/>　　赵老爷子露出了狂喜，自己说出了五十年的秘密，整个人都放松了，血液循环流通，双腿恢复了知觉。
<br/>　　“哈哈...”
<br/>　　他笑了，在众目睽睽之下哈哈大笑，他都觉得自己该死，可老天偏偏帮他。
