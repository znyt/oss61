　　“我有我的打算，你相信我就好了。”叶少阳想了一下，道，“你跟梧桐说，让她做好嫁人的准备，原定的的计划不变。”
<br/>　　卢晓清定定地望着他，道：“我也不问你计划是什么了，我只希望，梧桐没有看错人。”
<br/>　　叶少阳咧嘴笑了笑。“
<br/>　　卢晓清因为是偷偷出来，不敢逗留，说完这件事，立刻也就走了。叶少阳把目光移到妙心脸上，笑道：“你又想说什么？”
<br/>　　“他都说完了。”妙心摊了摊手，“不过我觉得，你这个作法很蠢，争一时义气，但你想过没有，明天如果你败了，会受到什么样的羞辱？”
<br/>　　叶少阳沉吟道：“这个，我自有打算，正好你来了，我也找你帮个忙，明天，梧桐走后，我希望你能陪着她，你们关系好，这个忙你一定会帮。”
<br/>　　妙心怔住，随即哭笑不得的说道：“叶少阳，你不是真的想在众目睽睽之下，从张晓寒身边把梧桐带走吧？”
<br/>　　“有什么不可以？”
<br/>　　“你做梦呢？”
<br/>　　“梦想还是要有的，万一实现了呢。”叶少阳贱贱地挑了挑眉毛。
<br/>　　妙心气的苦笑。“你简直就是个疯子！”
<br/>　　“别的你不用管，你只要答应我这件事就行。”
<br/>　　“好好好，我答应你。我话也说了，你坚持那么做，我也没有办法劝你。我只当梧桐是看错人了吧。”妙心说完，甩给叶少阳一个大大的白眼，赌气冲出房门。
<br/>　　叶少阳知道她回去之后，一定会去劝梧桐丢下自己，逃婚离开，不过他算定，梧桐应该不会走。
<br/>　　当晚，一夜无话，叶少阳吐纳了四个周天，又睡了一觉，早上被道渊等人叫起来吃早饭。道渊已经出去转了一圈，回来告诉叶少阳，他出去打听了，法术公会派出了很多人筹备婚宴，已经忙起来了。
<br/>　　吃完早饭，叶少阳跟大伙一起过去看，来到桃花山的山门前，能看到很多人在搬运桌椅板凳，在往山上运，还有各种食材，都是素的，这是法术界的传统：藏传佛教，还有部分道教的门派，是不吃肉的，在酒宴上于是干脆就都不用肉食，这样也是对所有人的尊重。
<br/>　　叶少阳一行人，跟在队伍后面，一路来到山顶，见桃花山正殿前面的广场上，已经被布置出了很多桌椅板凳，再看昨天发生战斗的那座独峰上面，尸体已经被收拾干净，一个人都没有了，蓝衣人也不在了。
<br/>　　道渊真人已经打听过，桃花山的山河道人，昨天已经带着剩余的弟子们，将尸体收敛，然后跟着蓝衣人一起去了桃花山的后山，再也没出来过。
<br/>　　“蓝衣人和碧清还在山上，鬼婆就敢在大殿前摆酒，不怕她过来闹事？”毛小方纳闷道。
<br/>　　美华道：“我想，既然她敢这桃花山上摆酒，在她背后，肯定是有星月奴的支持，而且已经做好了完全的准备。”
<br/>　　叶少阳点头道：“你这话说的在理。”
<br/>　　随即转头看着大伙，说道：“小方，道渊，两位老兄，酒席之上，如果我跟别人打起来，你们千万不要帮忙，毕竟你们都是有宗门的，到时候免得给你们的宗门带去麻烦。”
<br/>　　毛小方和道渊真人互相看了一眼，就道：“你要干什么？”
<br/>　　叶少阳抿着嘴，笑了一下，吐出两个字：“抢婚！”
<br/>　　大伙目瞪口呆。
<br/>　　宴席在傍晚举行。
<br/>　　叶少阳一行人来到山门前，立刻看到两个负责迎宾的子弟在山门外招待，一左一右，俨然把桃花山当成了自己的地盘。叶少阳出示了喜帖，带着一干人走上了山路，一路之上，两边都挂着大红灯笼，张灯结彩，气氛倒也充满了喜庆
<br/>　　来参加喜宴的，都是昨天一起上山来的那些法师，其中大部分是依附法术公会的门派弟子，一路上谈笑风生，叶少阳在中间找到了茅山的师徒几人，跟几个年老的和尚和道士走在一起。其中就有龙虎山的掌教等人。
<br/>　　道渊真人走过去，跟自己的师父和几个师叔打招呼，交谈几句，大家一起回头来看叶少阳，面色中带着狐疑，一个长辈还斥责了道渊真人几句，道渊真人也不吭声，后来又回到叶少阳这边，叶少阳问他说了什么，道渊真人无奈表示，他的师父和师叔们很不爽自己跟他这个来历不明的家伙搅在一起。
<br/>　　一路上到山顶大殿前，放眼看去，大殿前方，已经搭起了几十张桌子，有不少法术公会的弟子正在伺候，将菜品从不远处的一间房屋里不断送出来，摆到桌子上去。
<br/>　　叶少阳放眼看去，看那个小房间门外堆积着成堆的木柴，想着这里本来应该是柴房之类的地方，被法术公会征用了，当成厨房，倒也是离广场挺近。
<br/>　　“叶少阳，你可真来了啊。”
<br/>　　在广场前面，陈晓宇和卢晓清等人充当傧相，恭迎参加宴席的众人。见到叶少阳过来，陈晓宇立刻跳出来，眉飞色舞，并且夸张地做了个请的手势。“我还担心你不来呢，你来了就好，你先去坐吧。”
<br/>　　叶少阳一言不发，带着一行人走到广场上，一眼看去，最前面的餐桌都有名字，上面写着一些人名，叶少阳在最前面一排找到了云春生等人的名字，想来最前面一排，都是几大门派的掌教之类的，法术公会想要笼络他们，自然要对他们恭恭敬敬的。
<br/>　　后面的桌子都是没名字的，可以随便坐，叶少阳带他们找了个最不起眼的角落，坐下之后，有法术公会的人过来倒茶，叶少阳打量前排座位的那些名字，有一些都是自己听过的——在自己那个时代，这些名字代表的都是一个个的声名显赫的前辈。
<br/>　　“叶少阳，你还真来了。”妙心走过来，挨着叶少阳身边坐下，冷冷说道。
<br/>　　叶少阳抓了一把南瓜子在吃着，冲妙心笑了笑，道：“等着看好戏就是了。”
<br/>　　妙心哼了一声道：“看着你怎么被人打死吗？我觉得我还是离你远点，免得到时候溅我一身血哦。”
