　　风云突变，神秘人刚刚融合元神，恢复实力，便有一道神秘的声音传来，让他兴奋的心情骤然为之一滞，紧接着，他便看到一身穿白玉道袍，脚踏星斗的道人向他走来。
<br/>　　那人一步走出，时空轮转，站在虚空，淡漠的望着他，就好似一座高峰，巍峨峻拔，高大伟岸，永恒屹立于此，镇压诸天，不可撼动。
<br/>　　“来者何人？”不过，神秘人却也并不害怕，他看着来人，淡淡问道。
<br/>　　那人双眼一眯，淡淡打量着神秘人，在神秘人就要恼怒之际，突然开口道：
<br/>　　“混沌未判吾先行，大道因果得其主。
<br/>　　阴阳道一无极生，四象五行开天印。
<br/>　　诸神灭世一念间，元心不周三生聚。
<br/>　　万象更新天罚圣，一元复始道天尊。”
<br/>　　神秘人一懵，但紧接着，便明白了，这人歌中所言，已经告知了他的出生及其来历，他开口问道：“元始天尊？”
<br/>　　来者正是从洪荒赶来的元始本尊，他颔首道：“不错，正是本座，汝为何人？”
<br/>　　神秘人一听，顿时恼怒，冷声道：“大言不惭，区区混元圆满就敢称天尊，汝可知，天尊可不是谁都能称的，只有达到天道境第一境才可称一声天尊。”
<br/>　　“哦？”元始一听，眉头一挑，似乎是第一次听到此事，他淡声道：“本座就称天尊，又有何不可？汝既说天道第一境称之为天尊，那本座且为你，天道又分几境，各为何名？”
<br/>　　神秘人一听，顿时知晓，元始不是从上界而来。原来是一个下界之人，他顿时放心不少，不过，下界之人，能够修炼到混元圆满，真是少见，说不得可为他以后返回上界寻得一帮手。
<br/>　　神秘人一阵沉思，回答道：“天道境又分三境界，即天位、鸿蒙、虚无，到达三境界之人又分别称为天尊，天君，天主。”
<br/>　　“嗯。”元始一阵沉思，接着问道：“那天道境之下又如何称呼？”
<br/>　　“天道之下，诸天万界大致相似，可分为三境界，即凡人境，仙神境，混元境，诸天虽然称呼不同，但其原理大致相似。其中，混元境又有四种称呼，一至三重天为初期，可称之为圣王，四至六重天为中期，可称之为圣皇，七至九重天为后期，可称之为圣尊，十至十二重天为圆满，称为圣主，就如你我这个境界。”
<br/>　　“哦？”
<br/>　　元始心念一动，这个境界划分与他所知的一致，他所在的世界，证道要分四步走，第一步蜕凡证仙，此步中人即为凡境。第二步寻仙证圣，为仙神境，第三步混元证天，为混元境，此天即为天道境。之前他只知道有天道境，却不知其细致划分，第四步合道证不朽，不朽即大道境，天道之上，永恒长存，亘古不朽，具有一切道性，掌握一切真理。
<br/>　　元始又问道：“那天道境之上呢？”
<br/>　　“天道之上？”那神秘人冷笑一声，道：“天道之上，只存在于传说之中，又有谁知道呢？至少吾还未曾听说过。”
<br/>　　“嗯。”元始点点头，至少他知道了现在还没有人达到大道境，或者不朽境，他便放心不少。接着他看向神秘人，道：
<br/>　　“汝为何人？从何处来？欲往何处去？”
<br/>　　从哪里来，要到哪里去？这是一个值得深思的问题，透露着过去，现在，未来，每个人思考的方向不同，回答亦不同。大致是从来处来，到去处去。
<br/>　　“有意思。”神秘人微微一笑，并未隐瞒，也许他有求于元始，说道：“吾为玄羽圣主，太玄宫九弟子，吾从永恒一界来，欲寻回去路。”
<br/>　　“永恒一界？”元始抓住了这四个关键的字眼。“可是上界？”
<br/>　　“不错，想必你也隐隐感觉到了，在诸天万界之上，还有一个大界，那是诸天万界的起点，亦为万界之终点。永恒一界，无上之地，亘古长存，轮回难覆，诸天灭而它独存，万界毁而它独生。亦称三千道州，上苍之上，诸天万界唯一的永恒之地。”玄羽回答道。
<br/>　　“上苍之上？”元始咀嚼着这几个字，他突然就想到，他的分身混沌道君好像就在遮天大世界之中，那里也是一个很有意思的地方。沉思了一会儿，元始便问道：
<br/>　　“你可知道如何去永恒一界？”
<br/>　　“永恒一界，它无处不在，但却又无处可寻，欲前往此地，只有三个方法。修为达到混元圆满，便可隐隐感受到那苍茫大界，若是想要上去，只有修为突破天道境，到时自有办法上去。
<br/>　　另外一个办法则是，有修为达到天道境的修士将你带上去，除此之外，若是还想要上去，只有寻得一些神秘的接引古殿，被接引而去。不过，若是被接引古殿接引上去，那可就要看运气了，运气好，去到一个好地方，没事，若是运气不好，那可就……”说道此处，玄羽嘿嘿一笑，却并没有多说。
<br/>　　元始自然也明白他的话中意思，运气不好，自然是送命去了。不过，他也不管这些，他继续问道：“你可知何处有接引古殿？”
<br/>　　“你是想去永恒一界，吾劝你想都别想。接引古殿一般都是许多纪元之前上界大教挑选下界弟子所用，但自从上界大战爆发，接引古殿便一直都没有用过，如此多纪元过去了，世界都毁灭创造了不知多少次，现在寻找它，哪里还有？”玄羽冷笑道。
<br/>　　“哦？”元始听后，也不气馁，他暂时又不想去上界，接引古殿没有，对他也没有什么坏处，若是他以后想要上去，突破天道境即可，他自信自己以后可以突破。不过，他还是从此人的话中，听到了一点信息，那就是上界此时并不太平，正在爆发战争，而且还持续了几个纪元。
<br/>　　“说了如此多，想必你也不是没有目的。”元始淡淡的看着玄羽，开口道。
<br/>　　被人识破，玄羽也不恼，他说道：“不错，我亦有自己的目的。我在一次偶然机会下来到下界，我欲再次回到上界，希望得到你的帮助。”
<br/>　　“哦？偶然机会？”
<br/>　　元始似笑非笑地看着玄羽，看看他如今的凄惨样子，再看看那破败的太玄宫殿宇，元始便知道那偶然的机会恐怕不是什么好事了。不过，元始也不揭穿，而是说道：“本座看你如今的样子，好像撑不了多久了，你确定你还能回去？”
<br/>　　“本圣主之事，不需你多管，你只要回答帮不帮这个忙即可？”玄羽好似一点也不在乎身上的黑暗物质，淡淡道。
<br/>　　其实他内心还是多少有点紧张的，他如今实力不如巅峰的十之三四，而眼前此人亦是混元圆满，虽然不似他一样是十二重天，但大战起来他多半要吃亏，这也是他一开始便不想与此人大战的原因，如若不然，换个实力低点的人来，不要太低，只要混元后期，他都不会与对方废话，直接将对方灭杀，吞噬对方一身精气，也能够让他恢复一些。
<br/>　　“帮忙？”
<br/>　　元始淡淡微笑，停顿一下，接着道：“可以，不过，本座要看看你有没有那个资格让吾帮你这个忙。”
<br/>　　话音刚落，元始一指点出，天地一片寂静，时间停止，空间冻结。天地之间在这一瞬失去颜色，唯有一指从天而降，寂灭万物。
<br/>　　寂灭指！
<br/>　　……
