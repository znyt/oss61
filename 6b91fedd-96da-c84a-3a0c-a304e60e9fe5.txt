　　上空压抑的气息，令人毛骨悚然的杀意，街上一道道目光顺着声音望去，旋即一阵惊呼声在街上响彻起来。
<br/>　　“果然，是罗家老祖来了。”
<br/>　　“哈哈……罗小夭被杀了，罗家老祖果然是气急败坏了，杀了小的来了老的。”
<br/>　　“有好戏看了，都站远点，免得待会被波及到。”
<br/>　　在围观者哗然间，先前两名老者见来者也是一惊，急忙恭敬道：“恭迎老祖。”
<br/>　　行礼的同时，两人心中也是极其的忐忑。
<br/>　　他们的职责是保护罗小姐，可现在罗小姐就死在他们身旁，让他们如何不担忧。
<br/>　　完全可以说，罗小姐的死是他们失职所至，若是老祖怪罪下来，他们两人的下场肯定很凄惨。
<br/>　　蔚蓝的天空上，蓝袍老者凌空而立，脸色阴沉如水，双眼中透着森冷的杀意，目光看向倒在血泊中的红裙少女，浮现一抹悲痛。
<br/>　　旋即，视线落在慕清霄与小医仙的身上，杀意四射，道：“你们……想怎么死？”
<br/>　　闻言，慕清霄嘴角牵起一抹不屑，轻轻拂过中指上的雪白色纳戒，语气平静道：“前辈，将此人处理了吧。”
<br/>　　“区区六星斗宗也敢放肆，交给老夫吧。”
<br/>　　话语刚落，天火尊者的身影伴随着威压凭空而现，白袍无风自动。
<br/>　　旋即，空间凝固犹如一片泥沼，所有顶着威压，目瞪口呆的望着视线中的老者双手结印，一道赤红色掌印拍出。
<br/>　　“玄火印。”
<br/>　　“斗尊……”
<br/>　　回过神来，罗家老祖瞳孔一缩，想要逃走，但空间已被封锁，赤红色掌印瞬间映入眼帘，面若死灰。
<br/>　　“嘭！”
<br/>　　顷刻间，罗家老祖连求饶的机会都没有，便被横空而过的赤红色掌印拍成了一团血雾。
<br/>　　天涯城一霸，六星斗宗罗家老祖如蝼蚁一样，死的不能在死。
<br/>　　天火尊者收掌，捋了捋白花花的胡须，不屑道：“区区蝼蚁，也犯不着你出手，实在有失身份。”
<br/>　　天火尊者虽是魂体，但创伤的灵魂早已经恢复，平日跟正常人没有任何区别，对付一个六星斗宗如杀鸡一样。
<br/>　　况且，天涯城这种小地方，鲜有人能看出他只是魂体状态。
<br/>　　望着天空上洒落的血水，围观群众暗自吞咽起唾沫，目光呆泄的望着白袍老者，心脏不受控制的跳动起来。
<br/>　　能够轻描淡写的一掌拍死六星斗宗，绝对是斗尊级别的恐怖存在！
<br/>　　斗尊级别的强者不应该都在中州，为何会出现在天涯城中？
<br/>　　最重要的是，听闻斗尊的话语，他似乎只是一个下属，专门在保护眼前的青年……
<br/>　　能让斗尊贴身保护，身份与背景是何等可怕？
<br/>　　罗家恐怕要完蛋了，这是围观者心中唯一的念想，有的人看向倒在血泊中的红裙女子，眼中满是鄙夷。
<br/>　　惹祸精也有这一天，终于踢到钢板上了吧，整个罗家估计都会被她给牵连到死为止。
<br/>　　雪白色纳戒一亮，天火尊者凭空消失，慕清霄替小医仙捋了捋银丝，道：“我们走吧，去中央的空间虫洞。”
<br/>　　“嗯。”
<br/>　　小医仙嫣然一笑，温柔的搂着他的胳膊，旁若无人的行走在街道上。
<br/>　　待两人离开，围观群众才从呆泄中回过神来，后者似乎对罗家不感兴趣。
<br/>　　于是，有的势力聚集在一起，并且商量起如何对付罗家。
<br/>　　毕竟空间虫洞可是一块超级大蛋糕。
<br/>　　若是把控的好，让自己势力成为千里范围内的一流实力也并非不可能。
<br/>　　同时，慕清霄的信息如同病毒一样，飞速在各各家族中传播起来，被各家族列为绝对不能招惹的人物。
<br/>　　……
<br/>　　……
<br/>　　刚才的事情对于慕清霄来说只是一个小插曲。
<br/>　　若红裙女子只是嚣张跋扈，他根本没有兴趣理会这种人，但她却对自己的女人出手，简直是不知死活。
<br/>　　若是那一鞭子打中，脸部非得开花不可。
<br/>　　虽然以后者的实力根本不能击中小医仙，可就冲她敢狠辣出手，慕清霄就没有放过她的理由。
<br/>　　两人穿梭在偌大的天涯城，行走的速度并不算快，但不时就有敬畏的目光投射过来。
<br/>　　毫不在意行人的目光，慕清霄知道，肯定是刚才的事情已经传了出去。
<br/>　　约莫走了半个时辰左右，横穿小半个天涯城，随后一座占地极为庞大的黑石广场就出现在了两人的视线中。
<br/>　　广场的面积极为宽阔，地面与建筑都是用特殊的黑石材料建造，冰凉中给人一种极其坚固的感觉。
<br/>　　此刻，广场上早已被两名斗宗与六名斗皇给封锁，中央高耸的台阶上，隐隐有一股浑厚的空间力量弥漫而出。
<br/>　　陌上人如玉，公子世无双。
<br/>　　由于慕清霄与小医仙的容貌都极其出众，来到广场上就引起了两名斗宗的注意。
<br/>　　不过，两名发色皆白的斗宗却不敢摆任何架子，满面笑容的迎了上来，拱手道：“我等代表邢家与魏家，恭迎各位前来。”
<br/>　　慕清霄瞥了一眼面前的老者，两人皆是三星斗宗。
<br/>　　邢家，魏家与罗家，并列天涯城三大家族，虽说是三大家族，但罗家老祖实力远高于两人，隐隐有一家独大的趋势。
<br/>　　并且常年霸占空间虫洞，底蕴早已超越邢家与魏家。
<br/>　　半个时辰前，邢家与魏家的老祖听闻罗家老祖与孙女皆被击杀，原本不信，但城中已经传的沸沸扬扬。
<br/>　　听闻出手的是斗尊，两位老祖也是吓的心惊肉跳，但听说此事皆是罗家孙女引起，皆是松了口气。
<br/>　　并且警告族中晚辈，若是谁敢不长眼，他们绝对亲手大义灭亲。
<br/>　　罗家倒台，对于邢家与魏家来说，自然是喜事一件，并取吩咐家族高层，打压罗家，并且瓜分资源。
<br/>　　仅仅半个时辰，两家便合力将罗家拿下，并且以迅雷不及掩耳封锁空间虫洞，等待慕清霄与小医仙前来。
<br/>　　弱肉强食，慕清霄也不在意两家的做法，最起码比罗家要低调的多，没有招惹自己。
<br/>　　“空间虫洞何时可以使用？”
<br/>　　“回公子的话，老夫两人合力开启空间虫洞，立刻就能使用。”
