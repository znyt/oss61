　　“画像？”萧逸微微皱眉，随后恍然。
<br/>　　难怪刚才那工作人员能认出他。
<br/>　　按理说，他戴着面具，没有准确面容下，外人根本难以认出他。
<br/>　　而且偌大个中域，戴着面具行走的猎妖师或者武者，多了去了。
<br/>　　他萧逸置身人群中，戴着面具，也不过是个不起眼的普通武者。
<br/>　　不过，如果这些工作人员早有准备，且有画像观摩的话，细细观察之，倒也能从眉宇间大概认出。
<br/>　　这些工作人员，可都曾是在外行走过的猎妖师，经验丰富。
<br/>　　而且，一般也会有个普通执事的头衔，自然眼里不俗。
<br/>　　再者，衡元地域作为有名的大地域之一，主殿这里的工作人员，能力自然更是出色。
<br/>　　“你们殿主寻我何事？”萧逸疑惑问道。
<br/>　　这位殿主，既然猜错他会来此，而且分发下画像，让工作人员认出他，想来是有事找他。
<br/>　　“这就不是属下知道的了。”工作人员尴尬一笑，道，“我只知，殿主吩咐，若您真来了，便带您来觐见。”
<br/>　　说罢，二人刚好来到后堂书房处。
<br/>　　工作人员道，“我已经通知过殿主了，易霄分殿主自己进去即可。”
<br/>　　说罢，工作人员恭敬地行了一礼，缓缓退去。
<br/>　　萧逸皱了皱眉，推开房门进入。
<br/>　　里头，迎面可见一桌案，桌案之上，一老者在批阅卷宗。
<br/>　　猎妖殿的负责人，向来极忙。
<br/>　　萧逸每次看到这些殿主们，都是在批阅卷宗。
<br/>　　老者，显然早已注意到萧逸的到来，放下卷宗，面露微笑。
<br/>　　老者示意萧逸做到一旁。
<br/>　　萧逸走到桌案一旁的椅子，坐下。
<br/>　　老者看了眼萧逸，点了点头，面露满意笑容，“果真是后生可畏，易霄分殿主这般年轻，就已有这等实力和功绩，着实了不起。”
<br/>　　萧逸微微行了一礼，问道，“不知殿主寻我来何事？”
<br/>　　萧逸不想浪费时间，直接问道。
<br/>　　按理说，他在风刹殿和修罗殿，乃是总殿副殿主头衔。
<br/>　　按级别，和这位殿主同级，并不需要行礼。
<br/>　　但他现今是易霄身份，而且这位殿主显然是位前辈，便当行个普通的前辈之礼了。
<br/>　　“倒没什么大事。”老者摆摆手，道，“只是老夫猜测到你可能会来衡元主殿。”
<br/>　　“而老夫又素闻易霄分殿主的名声，特地想见一面罢了。”
<br/>　　萧逸点了点头。
<br/>　　按他的行走路线，一路向东面行走。
<br/>　　这位殿主，乃是主殿殿主，自然是本事不俗，也能猜出他可能来衡元主殿的。
<br/>　　萧逸起身，拱了拱手，道，“若无要事，在下还要去处理任务，便告辞了。”
<br/>　　“呵呵。”老者闻言，笑笑，示意萧逸坐下，道，“不急。”
<br/>　　“素闻易霄分殿主行事雷厉风行，此番一见，果真名不虚传。”
<br/>　　“不过，也不差这些许与老夫闲聊的时间了。”
<br/>　　萧逸皱了皱眉，点了点头，再次坐下。
<br/>　　“前些天，总殿那边，曾经给易霄分殿主下了一道召令。”老者说道，“易霄分殿主你可知晓？”
<br/>　　“召令？”萧逸摇了摇头，“不知。”
<br/>　　“果然如此。”老者苦笑一声，道，“大概大半年前吧，总殿那边传承之地的考核开启，需得10位猎妖殿天骄前往参加。”
<br/>　　“早在这之前，易霄分殿主在方寸地域、四方域等地方，就已扬名。”
<br/>　　“虽然听说那时功绩还未够，但总殿也勉强判定你入围，便将第十个名额给了你。”
<br/>　　“只是，后来易霄分殿主你销声匿迹，总殿也寻你不得，这才放弃，将第十个名额给了别人。”
<br/>　　“不曾想，易霄分殿主你压根没理会此事，也不知晓。”老者苦笑一声。
<br/>　　萧逸也淡淡一笑。
<br/>　　大半年前，猎妖殿总殿给了他名额，他倒不太清楚。
<br/>　　但那时他去了风刹殿和修罗殿参加传承考核。
<br/>　　所以，即便猎妖殿总殿给他名额也无用，他也去不了。
<br/>　　不过也当然了，他那时的功绩，在猎妖殿总殿中，也不过是堪堪够资格入围。
<br/>　　找不着他，总殿直接把名额给了别人。
<br/>　　自然的，此事后来也不了了之，萧逸没去特别了解的话，也不太清楚。
<br/>　　这时，老者继续道，“易霄分殿主一心猎妖，不在乎这些虚名，毫无疑问乃是个有担当而出色的猎妖师。”
<br/>　　“但，易霄分殿主不是那些刚出茅庐的菜鸟猎妖师，应该知道头衔能给你带来更大的权利和好处。”
<br/>　　“易霄分殿主不在乎这些虚名虚利，但猎妖殿也总不能愧对你。”
<br/>　　“故老夫今日特地提起此事，便是想问问易霄分殿主是否要晋升副殿主的头衔。”
<br/>　　“晋升头衔？”萧逸恍然。
<br/>　　原来这位殿主寻他来，是为此事。
<br/>　　老者点点头，道，“你有两个选择，要么，任职总殿的副殿主，老夫会帮你上报。”
<br/>　　“只不过，这上报时间比较久。”
<br/>　　“要么，你可以在老夫这座主殿任副殿主头衔，老夫同样会帮你上报。”
<br/>　　“但这处理时间会快许多。”
<br/>　　萧逸闻言，思索了一下。
<br/>　　衡元主殿，只是普通的主殿，并非十大主殿之一。
<br/>　　故要上报的话，需得先经十大主殿，再上报总殿。
<br/>　　这样的话，处理时间，起码十天半月，乃至更久；之后，他也还得重新回来这里接令，完成晋升手续。
<br/>　　倒是麻烦得很。
<br/>　　而如果在这座普通主殿任职副殿主的话，则只需上报十大主殿即可。
<br/>　　时间上，快很多。
<br/>　　那些麻烦的手续，老者也可以帮他处理。
<br/>　　“在这里任职的话，需要多久处理时间？”萧逸问道。
<br/>　　说是在这座主殿任职，但不过是在这里挂个名罢了。
<br/>　　猎妖师的行踪、去留等等，猎妖殿也不会干涉。
<br/>　　只是升个头衔，多些权利罢了。
<br/>　　老者眼睛一亮，道，“若是在我这里任副殿主之职的话，顶多十天，老夫就能帮你处理好。”
<br/>　　“十天？”萧逸皱了皱眉，摇摇头，“太久了，我可能不在这片地域待太久。”
<br/>　　“这…”老者面露失望之色。
<br/>　　“算了，还是谢过殿主好意。”萧逸拱拱手，“晋升头衔之事，他日在下自己处理吧。”
<br/>　　“可是…”老者皱了皱眉。
<br/>　　萧逸沉声道，“如今邪修猖獗不堪，各地域时常爆发邪修祸乱之事。”
<br/>　　“在下还是要以斩杀为要务，耽搁不得。”
<br/>　　“那好吧。”老者也知道孰轻孰重，点了点头。
<br/>　　“告辞。”萧逸拱拱手，转身离去。
