　　总之在仗还没有开打这前，钦陵和黑齿常之已经经过了无数次的推演，按照增灶的数据来推演倭人的反应。
<br/>　　当灶火增加到以前的一点五倍时，这个时候正是倭人最难受的时候，不管是进还是退，都是一个艰难的选择。而这个时候也正是倭人最容易做出判断的时候，如果他们再不下定决定，之后他们只能更加被动。
<br/>　　所以按照这个数据，钦陵一边增灶一边撤兵，时机赶的刚刚好，兵撤完了，倭人的进攻也到了，可见老天还是站在大唐的一边的。
<br/>　　……
<br/>　　书归正传，却说那些倭军精锐被埋在地下的炸药炸的凄凄惨惨戚戚，而另一边黑齿常之则带着属于他的三万大军自两军驻扎那处十里空地的斜刺里杀出。
<br/>　　快马加鞭之下不到片刻三万大军已经到了倭人的那些老弱残兵的面前。
<br/>　　大唐靺鞨铁骑，本就是刚刚归附不久，野性正是难驯的时候，上了战场哪里还管什么老弱不老弱，手中长刀舞的如同风车一般直接杀进了倭人大营。
<br/>　　话说这些小鬼子的装备也是真差，除了那些装备了大唐武器的，其他人手里拿着的一般都是些连破破烂烂的弓箭，用来攻击的箭甚至都是弯的，连箭头都没有。
<br/>　　当然，如果你想叫的话，也可以把那些箭叫成小木棍。
<br/>　　这样简陋的装备别说是杀人，在大唐靺鞨铁骑精良的装备下，边伤人都难。
<br/>　　而且有鉴于李承乾下意识的对大唐炼铁工艺的保密，横刀的制作工艺并没有被倭人学到手，所以他们的铁制武器也是极度简陋，相对于唐横刀来说不是太软就是太脆。
<br/>　　不过这些都不是最重要的，重要的是三万大唐靺鞨铁骑在杀进倭人大营之后几乎如同蝗虫过境一般，所过之处齐刷刷的倒下一地尸体，根本不会有任何的活口存在。
<br/>　　秦吉胜和山背大兄在出兵之前根本没有考虑过自己大营的安全情况，在他们看来主战场一直就是在唐军大营一方，自己这边应该并没有那么主要。
<br/>　　所以倭人的大营根本没有设备任何一点防御措施，靺鞨骑兵进入之后几乎如入无人之境。
<br/>　　而事实上，倭人的大营之中，大部分的倭人还在做着大胜的美梦，他们相信山背大兄和秦吉胜一定能够战胜唐军，将那些恶魔赶下大海，或者将他们的人头垒成京观。
<br/>　　但是残酷的现实在告诉他们，梦中的一切是多么可笑，如恶狼般冲是营地的大唐靺鞨铁骑用鲜血告诉他们，大唐就是一个绝对不可战胜的巨人。
<br/>　　……
<br/>　　逃，必须逃，逃的越快越好，营地中的倭人使几乎使出吃奶的力气，发了疯一样在逃命。
<br/>　　而黑齿常之所带领的靺鞨骑兵则是像割韭菜一样追在他们的身后，用手里的长刀收割着一条又一条鲜活的生命。
<br/>　　秦吉胜同样在逃命，他的儿子服部晴川同样在逃命，看守他的那些守卫早就已经跑的没了影子，面对死亡的时候，谁还管特么命令不命令，反正留下来抵抗也是死，不如留待有用之身，回头再重整齐鼓好了。
<br/>　　所以服部晴川就这样逃了出来，乱军之中也顾不得去找自己的老子，抢了一匹不知是哪里来的‘哈士奇’，拼了命的跑。
<br/>　　在这个时候，所有的倭人只有一个信念，那就是跑的要比同伴快一点，只快一点就可以了，到时候有了同伴在后面给自己垫背争取时间，自己就可以逃得一命。
<br/>　　而正是因为这样的想法，那些逃跑的倭人竟也开始自相残杀起来，打伤同伴，让同伴垫后，让同伴拖延时间成了所有倭人的一致想法。
<br/>　　不得不说，这一次秦吉胜和山背大兄真是上了一次恶当。
<br/>　　当然，这也有他们轻敌的因素在里面，如果他们稳扎稳打，二十几万大军一点点推过去的话，未必不能发现大唐军营中的陷阱。可是求胜心切的他们完全忽视了这一点，一个猛子直接扎进了乱葬岗子里面，把自己给埋了。
<br/>　　……
<br/>　　敦贺一战，十万靺鞨骑兵共计杀敌四十余万，鲜血浸透地面达数尺之深。
<br/>　　山背大兄王在爆炸中被炸死，物部氏族长断了一条腿，秦吉胜也没有得到任何好处，在逃了一半的时候被黑齿常之的骑兵追上做了俘虏。
<br/>　　至于说秦吉胜的儿子服部晴川，战后一直没有找到他的影子，也不知道是死在乱军之中了，还是跑掉了。
<br/>　　不过相到于这次大胜来说，一个小小的服部晴川并不值得人掂记，反正就算是他能够逃走，估计也活不了多少时间。
<br/>　　在没有后勤补给的情况下，他根本没有能力招到足够的部下，况且倭人经此一战早就被吓破了胆子，全都逃进了深山里面，每日只知道往大山深处钻，谁还会理会一个二世祖呢。
<br/>　　……
<br/>　　李承乾收到来自倭岛的战报之后显的很开心，郑重表态，对倭人的惩罚可以结束了，余下的倭人只要肯出来做苦工，那就留他们一条小命。
<br/>　　倭国已经没有多少人了，满打满算不足三百万，再杀虽然可以但却需要花费极大的精力。
<br/>　　这种事情就好像我们在地上捡东西一样，当数量越来越少的时候，我们捡的就越费力气，等少到一定程度的时候，就只能一个一个的捡拾。
<br/>　　屠尽倭岛也是一样，开始的时候倭人有很多，可以放开手脚的去杀，可是到了现在岛上已经没有多少人了，就算是杀一次也就只能逮到百来个人，根本不值得再让十万大军留在岛上。
<br/>　　毕竟大唐的粮食也不是天上掉下来的，为了养活这十万远征大军，需要付出的远比他们正常消耗的要多上数倍，再打下去大唐也有些支撑不住了。
<br/>　　当然，更重要的一点就是，一些番邦使团已经到了，新一轮的洽谈已经展开，倭国的事情必须要放一下，再怎么说小鬼子也不如赚钱重要。
<br/>　　李承乾这位大唐新皇帝此时想的是怎么从其他地方把这一次战争的钱补回来，血洗倭岛就暂时告以段落好了。
