　　　　他田不易当然不傻，现在让张小凡去面壁思过，岂不是将七脉会武的桂冠拱手让给龙首峰和小竹峰。
<br/>　　不，绝不可能！
<br/>　　苏茹嫣然一下，风姿动人，极其清楚他心中的想法，走上去牵起丈夫的手，笑道：“我就知道，你这人刀子嘴豆腐心。”
<br/>　　...
<br/>　　...
<br/>　　竖日清晨，旭日东升，大竹峰众人早早的来到云海广场上，此时，原本围绕中心搭建的八座擂台，拆的只剩下四座，剩下的分做东南西北四个方位排列。
<br/>　　田不易与苏茹走在最前面，张小凡低着头，依旧有些后怕，却没想到师父并不惩罚他，反而像忘了昨晚的事情，让他照常参加比试。
<br/>　　走在众人中央，从未受到如此重视的他颇有些受宠若惊的样子，回头看了看，低声对身旁的杜必书道：“六师兄，大师兄伤的很重吗，怎会到了走不动的地步？”
<br/>　　杜必书摇了摇头，道：“师父早上给大师兄看过了，说是昨日那场比试中他与长门的常箭师兄比试太过激烈，斗的两败俱伤，伤了经络，只怕于修行受损不轻......”
<br/>　　“原来如此，那怎么不见慕师弟？”
<br/>　　“慕师弟一大早就不见了人影，也不知道去了何处。”
<br/>　　此时，北方最大的擂台下，人山人海，看这排场，一眼便能看出是陆雪琪在擂台上比试。
<br/>　　田不易瞥了一眼张小凡，又瞥了眼北方擂台上的陆雪琪，也不知道这小子到底有多少斤两，夺得桂冠又有几成把握。
<br/>　　......
<br/>　　......
<br/>　　此时，通天峰西北方，一处密林中，慕清霄盘坐于参天巨树那粗壮的枝干上，心神守一，掌心向天，昨日他心有感触，体内灵气动荡，清楚这是修为即将突破的征兆，乘夜来到这片密林中。
<br/>　　慕清霄正处于一个特殊的状态，仿佛自身多出了什么，这种感觉很特别，但又有些说不上来，只好运转心法，全速吸收四周浓郁的天地灵气。
<br/>　　他的修为早就已经是元婴二重巅峰，瓶颈犹如一层薄薄的纸片，一捅就破。
<br/>　　随着时间的推移，四肢百骸中的灵气越来越多，心法运转第八个周天，直至第九个周天，慕清霄身躯一颤，一股无形的波动向四周扩散而出。
<br/>　　双眼睁开，星辰一般的眼眸闪烁着精芒，神圣的气息犹然而生，待亮光消散，气息收敛，慕清霄站起身来，再次变得平淡无奇。
<br/>　　望着四周茂密的森林，慕清霄手一挥，一股气势扩散而出，将周身百米笼罩在内。
<br/>　　眼中闪过一丝困惑，这并非神识，也不是灵气，在周身百米内的情况却看的非常清楚，而且慕清霄能察觉到，气势凌厉至极，这明明就是他的剑意！
<br/>　　在他剑意的笼罩内，世间万物，唯我独尊！
<br/>　　慕清霄愣了愣神，眼中闪过一丝喜悦，心念一动，在他的百米范围内，思维所到之处，剑芒一闪，一颗巨树悄无声息的被截断，甚至他连手指都不需要动一下！
<br/>　　然后，他有尝试了一次，果然，百米范围内，剑气纵横，但百米外却没有丝毫异样，剑意也无法抵达。
<br/>　　这是何等恐怖的神通，他虽然能够将无上剑意收放自如，却不能随心所欲，哪怕用剑意，都必须由自身凝聚，或者用媒介，比如魂剑，又或者自己的双手。
<br/>　　但现在不同，只要在这百米范围内，他只要一丝心念，便能够瞬间斩杀其中的敌人，手指都不必动一下。
<br/>　　“这到底是什？”慕清霄喃喃自语，思索了半天也没想出个所以然，忍不住道：“系统，这到底是什么情况？”
<br/>　　“叮，本次回答的问题需扣除1000掠夺积分。”
<br/>　　慕清霄嘴角一抽，咬牙道：“扣吧，扣吧！”
<br/>　　“现在所施展的神通，名为‘领域’，唯有境界高深的修士才会诞生领域，宿主则比较特殊，提前诞生出了‘领域’，此神通也分类型，修士本身实力越强，领域范围也就越广，其中很多信息都需要宿主自行摸索......”
<br/>　　系统提示声落下，慕清霄剑眉一挑，呢喃道：“‘领域’，难道自己还不算是大修士吗？”
<br/>　　压下心中复杂的情绪，慕清霄微不可查的摇头，修行本是循序渐进，不可能一朝而成，这么多年都过来，他不可能因此而动摇，转身离开森林，向云海广场奔去。
<br/>　　......
<br/>　　......
<br/>　　当慕清霄来到云海广场，望着北方的擂台上，宛如九天仙姿的陆雪琪，微微一笑。

<br/>　　斩龙剑，剑身有龙形，通体碧绿闪耀，冷艳非常，乃是取自南疆极苦之地万载绿晶，费时六年所铸，剑成之日天有雷鸣，落雨似龙血，故名之曰：斩龙。

<br/>　　“铮！”
<br/>　　伴随着清脆的剑鸣，‘天琊’神剑出鞘，电光火石间，两人速度快若闪电，剑气纵横，整个擂台都变得满目疮痍。
