<br/>　　这陆信的嚣张，比起那陆莫，都是丝毫不差。

<br/>　　这陆家的人，如此嚣张，楚风眠也不会怕了他们。

<br/>　　“轰隆！轰隆！”
<br/>　　一声声轰鸣声，随着陆信的一拳打出爆发而出。
<br/>　　这一拳的力量，直接便是冲着楚风眠的胸口打了过来。
<br/>　　要将楚风眠的灵脉，一拳打碎。
<br/>　　“这事陆家的十重拳！威力极为霸道，就算是挡下了这一拳，其中还蕴含十重暗劲。”

<br/>　　这一股灵力轰击过来，顿时是将陆信的十重拳的力量，完全化解。
<br/>　　同时也是一股力量，将陆信的身躯，都给直接逼退。

<br/>　　一尊身影，突然现身。

<br/>　　看到这尊身影，周围众人不由都松了口气。
