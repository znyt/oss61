　　“差别很大，等级越高的武魂，晋升战魂之后威力越大！”
<br/>　　木璃淡淡的说道，虽然他不清楚苏莫为何问出如此问题，但作为南苑的导师，对于南苑弟子的提问，她还是会回答的。

<br/>　　最高的等级，也就是天级九阶了！
<br/>　　“多谢木导师指点！”苏莫向木璃道谢，他本还想询问一些关于神武学府的情况，但想想还是算了。
<br/>　　神武学府的情况，他在学府中自然会了解到。
<br/>　　“南苑广场，每月会有南苑导师公开授课，后天便是授课日期，你可以去听讲！”木璃道。
<br/>　　“嗯，我一定会去的！”苏莫点了点头，心中却是颇为好奇，居然每月都有导师授课，这神武学院对于弟子的培养，真是不遗余力啊。
<br/>　　木璃微微颌首，随即不再逗留，转身飘然而去。
<br/>　　而后，苏莫便返回了阁楼中，继续安静修炼。
<br/>　　同时，他心神沟通自己的武魂，仔细的感受。
<br/>　　战魂，是要武魂与自己的神魂结合，然后再有奥义之力的辅助，才能真正的晋升为战魂。
<br/>　　武魂和自己的神魂结合，这应该没有多大的问题，只要努力，然后耗费一定的时间，就一定能成功。
<br/>　　但这奥义之力，却是虚无缥缈，也不知道需要达到什么地步，才能用于武魂进化！
<br/>　　思索了良久，最后苏莫摇了摇头，多想无益，车到山前必有路，走一步算一步吧！
<br/>　　在安静的修炼之中，两日时间转瞬而过。
<br/>　　清晨。
<br/>　　南苑中热闹了起来，大量的南苑弟子纷纷走出自己的居所，向南苑广场赶去。
<br/>　　“今日，有两位导师授课，分别是孙仲闻导师和木璃导师！”
<br/>　　“每次木璃导师的授课，都是人满为患！”
<br/>　　“哈哈，确实，虽然南苑弟子中炼器师不多，但美人导师，谁不想一睹风采！
<br/>　　“对了，你听说了吗？最近南苑新来了一名弟子，据说火元圣师要收其为徒，那名弟子却拒绝了！”
<br/>　　“听说了，估计就是个傻叉吧，真是不知所谓！
<br/>　　“……”
<br/>　　阁楼中。
<br/>　　苏莫睁开了眼眸，耳中听到阁楼外路过弟子的谈话，不由得眉头微皱。
<br/>　　这件事情，居然闹得人尽皆知了？
<br/>　　少顷，苏莫叹了口气，打开了阁楼的大门，随着其他弟子离开的方向，向南苑广场而去。
<br/>　　南苑占地极广，如同一座城池一般，苏莫穿过无数的建筑，终于来到了南苑广场。
<br/>　　南苑广场非常大，方圆十里，坐落在南苑的中央位置。
<br/>　　在苏莫到达南苑广场之时，一眼望去，整个广场之上已经人满为患，聚集了不下四五千人。
<br/>　　嘈杂的声音此起彼伏，杂乱无章。
<br/>　　感受到这数千米弟子身上的气息，苏莫心中暗暗震惊，这些南苑的弟子，修为在武帝境之下的人，少之又少，十不足一。
<br/>　　几乎九成以上的人，都是武帝境的修为，非常的强盛。
<br/>　　苏莫心中不由得感慨，此等武道学院正是恐怖，普通的外府弟子都是武帝境的修为，那内府的弟子，绝对都是武尊境的修为了。
<br/>　　此时，广场的一端，已经搭建了一个高约两丈的木台，木台面积不大，只有方圆三丈左右，显然便是倒是授课的讲台。
<br/>　　苏莫在神武学府没有任何熟人，来到广场之后，便伫立在角落中，静静的等待了起来。

<br/>　　中年男子面色威严，身上气息浩大深邃，实力深不可测。
<br/>　　“拜见孙导师！”
<br/>　　“见过孙导师！”
<br/>　　随着中年男子到来，广场上的众弟子纷纷抱拳，向中年男子行礼。
<br/>　　“诸位弟子不必客气，安心听课吧！”中年男子摆了摆手，朗声说道。
<br/>　　随即，众人立刻安静了下来，几乎所有弟子全部都盘膝坐了下来。
<br/>　　苏莫见此，便也盘膝而坐，等待这位孙导师讲课。
<br/>　　“今日，为你们讲解关于战魂的知识！”孙导师目视全场，眸光如电，朗声说道。
<br/>　　苏莫闻言心中一震，顿时面露喜色，他正愁不了解战魂，今日的授课，居然就是讲解战魂的知识，真是及时雨啊！
<br/>　　随即，他立刻打起来精神，准备仔细的听讲。
<br/>　　少顷之后，孙导师心中组织了一番语言，便朗声开口了。
<br/>　　“众所周知，战魂乃是武魂进化而来，武魂的妙用，除了帮助武者吸纳天地灵气之外，还有很多特殊的妙用！”
<br/>　　“比如剑武魂，可以让武者的攻击更加锋锐，杀伤力更大，比如磐石武魂，可以让武者的攻击更加厚重、防御力大增！”
<br/>　　“当然，还有很多特殊的武魂，比如太古武魂榜上的那些超级武魂，都有匪夷所思的特殊能力……！”
<br/>　　孙导师口中滔滔不绝，从战魂的前身，武魂开始讲解了起来。

<br/>　　太古武魂榜！
<br/>　　这是什么榜单？难道是宇宙中各种武魂的强弱排名？
<br/>　　看来日后有时间的话，他要搞到这太古武魂榜研究研究，看看有没有自己这种吞噬武魂！

<br/>　　木璃导师到了！
<br/>　　“见过木璃导师！”
<br/>　　“见过木璃导师！”

<br/>　　众弟子纷纷起身，向木璃抱拳行礼，各个神情激动，神情明显比对待孙导师更殷切。
