　　天海三少入座之后，整个主桌上的人，明显感觉到了压力，一个个全都正襟危坐，同时心中对杨之凡的能量，也有了一个更为深刻的认识。
<br/>　　之前，他们以为，杨之凡虽然是东江省的顶级阔少，但毕竟如今是在天南省，杨家的势力只怕也鞭长莫及，虽然表面对他客客气气，心里却并没有太把他当回事。
<br/>　　可是现在看到，他竟然可以将天海市的三位顶级阔少，邀请到这里。
<br/>　　这就让他们，不得不重新审视，杨之凡的分量了。
<br/>　　杨之凡将众人的表情看样眼中，知道已经将众人镇住，嘴角不由得微微一笑，站起身来，道：
<br/>　　“诸位，今天是刘老师的生日，我们大家先来敬刘老师一杯，如何？”
<br/>　　众人轰然叫好，纷纷举起手中的酒杯。
<br/>　　刘传明顿时受宠若惊，慌忙站起身来，连连道谢，
<br/>　　“谢谢大家！谢谢之凡同学！老师我先干为敬！”
<br/>　　说完，刘传明头一仰，已经将大半杯红酒一饮而尽。
<br/>　　接下来，
<br/>　　宴会也正式开始了。
<br/>　　杨之凡长袖善舞，和天海三少谈笑风生，将整个宴会的气氛调动了起来，俨然一副主角姿态。
<br/>　　就连刘传明这个名义上的主角，也只能在一旁唯唯称是，却又不敢又任何不悦之色。
<br/>　　忽然，
<br/>　　酒至酣处，天海三少中的那位江少开口道：
<br/>　　“杨少，听闻你们生物学院这一届新生之中，有一个叫叶尘的，把广南首富萧天佐的千金追到了手，不知道是哪位，能否让我见识一下？”
<br/>　　杨之凡此时已经喝了不少酒，在众人的吹捧中，有些飘飘然，当即指着角落里的叶尘，道：
<br/>　　“就是他，他就是叶尘！”
<br/>　　天海三少不由得对视了一眼，眼眸中纷纷闪过一抹轻视之色。
<br/>　　从叶尘坐的位置来看，显然在生物学院的地位不高，而且他自己似乎也默认了这一点，否则若换成他们三个，被人如此轻视，早就直接闪人了，又岂会坐在那里，稳如泰山？
<br/>　　“原来就是他啊！看上去也没什么过人之处嘛！”
<br/>　　“看来那位萧大校花的眼光，实在不怎么好！”
<br/>　　“可惜啊！一朵鲜花插在了牛粪上！”
<br/>　　……
<br/>　　三人肆无忌惮的嘲讽起来，而且还故意将声音提高了许多。
<br/>　　叶尘仿佛没有听到一般，依旧自顾自的坐在角落，慢慢的饮着红酒。
<br/>　　倒是一旁的李虎听不下去了，霍然站起身来，
<br/>　　“你们胡说八道什么？我看你们分明就是嫉妒！”
<br/>　　天海三少听到这话，顿时也全都霍然站了起来，
<br/>　　“我们堂堂的天海三少，想要什么的女人没有，我们会嫉妒你们这些穷屌丝？”
<br/>　　“死胖子！你特么谁呀？竟然敢和我们这么说话？”
<br/>　　李虎一时冲动，站了起来，原本还有些后悔，可是听到对方这话，顿时便再也忍不住了。
<br/>　　他平生最恨别人叫他死胖子，不管谁这么喊他，他就是豁出性命，也要和那人干上一架！
<br/>　　“老子是你李虎爷爷，你想到怎的？”
<br/>　　李虎直接一拍桌子，威风凛凛道。
<br/>　　天海三少先是被李虎身上的王八之气镇住，继而纷纷大怒，
<br/>　　“好小子！你很狂啊！”
<br/>　　“我看你是不知道死字怎么写！”
<br/>　　“死胖子！你竟然敢和我们作对，信不信本少，今天让你走不出这酒店！”
<br/>　　……
<br/>　　眼看双方越闹越凶，整个宴厅顿时也躁动了起来。
<br/>　　众人显然也都没有想到，平时一向笑容可掬的李虎，竟然会突然发飙，敢公然和三大少叫板！
<br/>　　刘传明虽然是现场唯一的老师，可是面对赫赫有名的天海三少，他根本不敢管。
<br/>　　而杨之凡，却两眼微微一眯，也没有开口阻止，反而在暗暗观察着叶尘的举动。
<br/>　　他的直觉告诉自己，自己的这个同学，绝对不简单，可是他之前派人调查过，却一无所获，所以心里十分好奇，面对眼前的场景，他会如何处理？
<br/>　　然而很快，他却失望了，
<br/>　　叶尘从始至终，连眼皮都没有抬一下，仿佛就像是没有听到双方的争吵。
<br/>　　这时，
<br/>　　倒是陶婉清，忽然站了起来，
<br/>　　“几位，今天是刘老师的生日宴，还请你们都少说两句吧！不要因为一点小事，搅了大家的兴致！”
<br/>　　她本想劝双方罢手，可惜三大少此时正在气头上，如何能听的进劝？
<br/>　　“你算什么东西？也配教我们做事？”
<br/>　　那王少瞪了一眼陶婉清，直接训斥道。
<br/>　　众人顿时一片哗然，
<br/>　　显然没有想到，这三大少竟然一点都不给陶婉清面子！
<br/>　　陶婉清顿时美眸一瞪，气得肺都要炸了，饱满的胸脯，一起一伏，似乎要随时把晚礼服撑爆一般。
<br/>　　眼看陶婉清也要发火，一旁的杨之凡，自然无法再继续坐视不理，当即站起身来，一脸不悦道：
<br/>　　“王少！你过分了啊！婉清的意思是，你们之前的恩怨，完全可以出去解决，在这里，可不方便动手！”
<br/>　　“我不是……”
<br/>　　陶婉清想要开口解释，却被杨之凡强行按回座位之中。
<br/>　　天海三人听到杨之凡这话，不由得对视了一眼，
<br/>　　“之凡老弟言之有理，我们兄弟就给你这个面子！”
<br/>　　说着，三人又转而望向李虎，
<br/>　　“死胖子，咱们到酒店外面却解决，你敢吗？”
<br/>　　李虎被对方一口一个死胖子，叫的怒火冲天，可是他却也知道，对方都是不好惹的主，一时间大为踌躇。
<br/>　　就在这时，
<br/>　　叶尘忽然站了起来，拍了拍他的肩头，道：
<br/>　　“人家既然都已经划出道来了，咱们也不能怂，走！出去会会他们！”
<br/>　　说着，叶尘率先朝门外走去。
<br/>　　李虎听到叶尘这话，顿时精神一震，连他自己也说不上为何，就是隐隐觉得，只要叶尘在，他就一点也不怕了，于是也快步跟上。
<br/>　　唐威和陈东也没有犹豫，同样跟上上去。
<br/>　　天海三少见状，先是一愣，继而脸上纷纷露出一抹嘲弄之色，
<br/>　　“好好好！没想到，还真有人胆敢挑衅我们天海三少的威严！”
<br/>　　“不废了这个死胖子，本少的名字以后就倒着写！”
<br/>　　“走！出去弄他！”
<br/>　　三人也气势冲冲的跟了过去。
<br/>　　哗啦啦！
<br/>　　在场的众人，很多也十分好奇，于是也纷纷跟了出去，就连杨之凡略微犹豫了片刻，也追了上去。
<br/>　　剩下的众人见状，顿时也待不住了，于是全员出动，都纷纷离席，唯独剩下刘传明一个人，满脸的郁闷之色。
