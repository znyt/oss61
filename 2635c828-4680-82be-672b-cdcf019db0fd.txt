　　苏莫飞驰在天穹之上，下方是辽阔无垠的大地，苍茫无际。
<br/>　　他已经飞行了五日，还要几日才能到达神武学府。
<br/>　　这五日时间，他飞跃了无数的雄伟的城池，无数的高山大岳。
<br/>　　这些城池，无一不是人口过亿，无一不是壮阔雄伟。
<br/>　　不仅如此，他更是在很多灵气浓郁的山川大岳之上，看到了那些强盛的宗门势力。
<br/>　　远远的感感受到那些宗门内传出的强大气息，他都及早避开了。
<br/>　　他现在的实力，在古灵星河，在古周星，实在是普通至极，不得不小心翼翼。
<br/>　　苏莫一路急行，约莫又过了三日之后，他来到一片一望无际到荒原之上。
<br/>　　“快到了！”苏莫的面上露出了笑容，据闻神武学府的所在，便是在这片荒原之上。
<br/>　　苏莫继续飞行，又飞了约莫两个时辰之后，他的前方视线被挡住了。
<br/>　　他的前方出现了一道巨大的影子，长约三千里，高达数百里，横亘在大地之上，遮天蔽日，仿佛隔绝了一切。
<br/>　　“这是什么？”苏莫见此好奇不已，立刻稍稍放缓了速度。
<br/>　　而后，他将身形拔高，向上方飞去，顿时便看清了前方之物。
<br/>　　只见前方那巨大的物体，居然是一个庞大的球体，方圆千里，深陷在大地之中。
<br/>　　“这是……陨星？”苏莫见此微微愕然，因为这硕大无比的球体，散发着惊人的星辰之力，分明便是一颗小型的星辰，一颗陨落的星辰！
<br/>　　只见这颗星辰，有一大半都没入了大地之中，裸.露在外的部分，仿佛是一片高耸连绵的山脉。
<br/>　　随着苏莫继续飞行，他更是看到了星辰之上，坐落着无数的建筑，琼楼玉宇，白玉宫殿，数之不尽，连绵成片，高耸入云。
<br/>　　这一幕极为壮观，由于星辰很大，上方早已插入云层之中，目光望去，这些连绵的建筑群，仿佛是建造在云端之上的仙宫。
<br/>　　磅礴大气，云雾缭绕，光华璀璨，壮观无比。
<br/>　　苏莫不禁倒吸了一口凉气，难道这就是神武学府吗？果然不简单！
<br/>　　很快，苏莫便飞进了陨星，也将整个陨星之上的建筑尽收眼底。
<br/>　　这陨星之上的建筑，组成了一座圆形的城池，方圆两千里的巨城，城墙高达千丈，有东南西北四方城门。
<br/>　　在整个巨城之中，苏莫能感受到无数强大的气息，如同一颗颗耀眼的星辰，照耀八方，有的甚至比巫梵还要强大十倍，非常的惊人。
<br/>　　苏莫身形位于南方，清楚的看到南方城门之上，‘神武学府’四个光芒四射的大字。
<br/>　　“果然是神武学府！”苏莫眼眸一亮，这神武学府比他想象的还要不凡。
<br/>　　一座武道学院，居然建造在陨落的星辰之上，这真是不错的想法。
<br/>　　就在此时，苏莫感觉到惊人的吸扯之力，从下方星辰上传出。
<br/>　　这股吸扯之力极为强大，仿佛是万倍重力加身一般，将苏莫向下拉扯。
<br/>　　“这是重力！”苏莫心中惊讶不已，虽然这重力对他没有多少的影响，但这股重力明显是这颗陨落星辰自带的大地引力。
<br/>　　一颗小小的星辰，大地引力居然如此之强，比他踏足过的任何星辰都强大万倍。
<br/>　　在如此重力之下，像他这种实力的武者，实力会被削弱一大截。
<br/>　　唰！
<br/>　　苏莫没有停留，顺着大地引力的拉扯，径直飞到了陨落星辰之上，向神武学府的南门飞去，片刻间便来到了南门前。
<br/>　　南门高大，长宽均有数十丈，装有两扇巨大的黄金大门，气派非凡。
<br/>　　南门前，有两名年青的武者在把守。
<br/>　　不过，这两名守门之人，并不像其它地方的守卫那般凝神守护大门，而是盘膝坐在大门两边，闭目而坐，仿佛是两具雕塑一般。
<br/>　　即便是苏莫到来，这两人都没有睁开眼眸，完全不管不问。
<br/>　　这让苏莫心中惊奇，他打量了一番这两人，见两人的年纪和他相仿，却都是武帝境修为，相当不凡。
<br/>　　他暗暗猜测，这两人如此天赋，应该是神武学府的弟子吧！
<br/>　　此时，南门紧闭，苏莫又不好直接飞进城中，便走向那两名青年。
<br/>　　“在下苏莫，见过两位兄台！”苏莫走到两人身边，抱拳说道，言语客气至极。
<br/>　　但是，这两名青年没有任何反应，连眼眸都没有睁开。
<br/>　　若不是两人生命气息浓郁，苏莫都认为对方两人是死人了。
<br/>　　苏莫无奈，不过他也没有恼火，心中沉吟一番，再次开口道：“在下是奉一位前辈之命，来找火元圣师！”
<br/>　　苏莫抬出了火元圣师的名头，如此的话，对方应该不会作视不理了。
<br/>　　毕竟，能称为圣师的人绝对不是普通人物，在这神武学府应该也有一定的地位。
<br/>　　果然，他刚刚说出火元圣师之名，这两名青年顿时睁开了眼眸。
<br/>　　其中一名脸颊消瘦的青年，打量了苏莫一番，沉声问道：“你是奉谁之命？”
<br/>　　“苍云宗柴大师！”苏莫道，他并不清楚柴管事的具体名讳，便只能这么回答了。
<br/>　　柴管事是苍云宗的长老，又是炼器师，如此对方应该知道。
<br/>　　“柴大师？”青年闻言皱眉，他并没有听说过什么柴大师。
<br/>　　“非神武学府弟子，严禁进入学府内，你可有信物？我前去为你通传！”青年说道。
<br/>　　“有！”苏莫点了点头，随即便拿出了那枚柴管事给他的玉简，其内有柴管事所留的信息。
<br/>　　这枚玉简，苏莫也没有私自查看，但他也能大致猜出来，其内定是柴管事向那位火元圣师在推荐自他。
<br/>　　“在此等候吧！”青年接过玉简之后，便打开城门，向神武学府中走去。
<br/>　　而后，苏莫便静静的等待了起来，他一边等待一边打量四周。
<br/>　　这神武学府非常不简单，那高达千丈的城墙不知道是用什么金属打造而成，闪烁着森冷的光芒，给人一种厚重如山，坚不可摧的感觉。
<br/>　　脚下的大地，不仅引力强大，更是坚硬如铁。
<br/>　　这颗陨落的星辰，非常的而不简单，难怪神武学府会在此星辰之上建造学院！
<br/>　　苏莫没有等待太久，仅仅是半刻钟的功夫，那名前去通传的青年便去而复返。
<br/>　　“跟我来吧！”
<br/>　　青年向苏莫喊了一声，便为其带路，再次向府中走去。
<br/>　　苏莫立刻跟上，大步走进了神武学府之中。
