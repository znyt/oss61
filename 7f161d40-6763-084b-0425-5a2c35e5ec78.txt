　　“跟我来吧！”未天神王沉声道，随后，便带着苏莫离开了宫殿。
<br/>　　宫殿之外，围拢的近千人，看到跟着未天神王走出的苏莫，纷纷恍然，原来是苏莫在搞鬼。
<br/>　　一瞬间，很多人都怒了，这苏莫简直是可恶，如此糟蹋他们的紫灵之气。
<br/>　　“好了，大家都散了吧！”未天神王来到殿门之前，向围拢的众人挥了挥手。
<br/>　　听闻未天长老之言，众人纷纷准备离开。
<br/>　　不过，一名身穿紫衣的青年却是立刻指着苏莫，大声质问：“苏莫，你知道自己在做什么吗？”
<br/>　　“什么意思？”苏莫见此，顿时眉头一皱，他知道自己吞噬紫灵之气不妥，但是，他吞噬的紫灵之气，还不到这混沌古山上紫灵之气的千分之一，对方值得动这么大的火气？
<br/>　　对于这人，苏莫有些眼熟，对方好像也参加过千选战，但是，他并不知道对方的名讳。
<br/>　　“谁给你的权利，肆意糟蹋紫灵之气？”紫衣青年怒喝道。
<br/>　　“何为糟蹋？”苏莫冷冷的问道，对方若是说他不能吞噬，或者是混沌族要他赔偿，他都没有怨言，但是，居然说他糟蹋紫灵之气？吞噬了就是糟蹋？
<br/>　　“你不是我朝之人，更不是我族之人，难道不是在糟蹋？”紫衣青年怒声说道，他心中很是气愤，这紫灵之气，乃是鸿蒙紫气元神产生的灵雾，极其有限。
<br/>　　现在看似很多，但若是有神王级别的长老突破，或者是七八品以上的人冲击境界瓶颈，会大量的消耗紫灵之气。
<br/>　　可以这么说，紫灵之气产生的量，供应所有混沌古山上的人，都有些捉襟见肘。
<br/>　　所以，紫衣青年才会如此愤怒，而且，他过段时间便准备冲击八品的修为，正是需要紫灵之气的时候。
<br/>　　苏莫闻听对方之言，微微的叹了口气，他的确不是混沌神朝之人，对方如此愤怒，也无可厚非。
<br/>　　“恒宸，苏莫修炼的混沌道统，也算半个我族之人，你少数两句吧！”未天神王摆手说道，此事主要责任在他，是他一时大意了。
<br/>　　“未天长老，我也是一时怒急，才会如此，请长老见谅！”紫衣今年恒宸向未天神王抱拳道，既然对方开口了，他也不能再说什么，免得得罪对方。
<br/>　　“嗯！”
<br/>　　未天长老微微点头，随即示意苏莫，便带着他向山腰之处走出。
<br/>　　望着未天神王和苏莫离去的背影，众人面色复杂，苏莫的身份这么敏感，未天长老还让其留在混沌古山，这是何意？
<br/>　　对于混沌族高层的想法，一般的弟子无法知晓，只能暗暗的猜测。
<br/>　　未天神王将苏莫带到了山腰之处，找了一个无人居住的小型宫殿，给予苏莫作为了暂时的居所。
<br/>　　“前辈，这令牌你要收回去吗？”宫殿中，苏莫取出了未天神王之前给予他的令牌，沉声问道。
<br/>　　对方既然不让他吞噬紫灵之气，想必，也不会再让他出入混沌古山的各种地方了。
<br/>　　“这令牌你暂时留着吧，需要资源，凭此令牌，你就能获取！”未天神王摇头说道，除了紫灵之气外，族中资源倒是不忌讳苏莫享用。
<br/>　　至于紫灵之气，此事要请教元王，看看能不能给予苏莫一部分。
<br/>　　“多谢前辈了！”苏莫抱拳道谢，此举，已经算是很大的恩德了。
<br/>　　“这座宫殿也有修炼室，不过，你无法吞噬到紫灵之气，你就暂时住在此地吧！”未天神王叹了口气，随即便转身离去。
<br/>　　苏莫目送对方离开之后，便在密室中找到了修炼室，继续开始了修炼。
<br/>　　他本来想试试，仅凭自己的吞噬元神，能否吞噬到上空的紫灵之气，但是想到这无异于不给混沌族颜面，所以立刻打消了这个念头。
<br/>　　随后，他便只能自己修炼了起来，之前吞噬了不少紫灵之气，现在还没有炼化完。
<br/>　　……
<br/>　　未天神王又返回了山巅之上，那座古朴的院落之中。
<br/>　　此事此刻，院落之中，不仅有混沌元王，还有十几位神王级别的强者。
<br/>　　苏莫来到混沌古山，这不是小事，所以，只要不是在外的长老，或者是闭关的长老，基本上都来了。
<br/>　　“元王，苏莫来到了我混沌古山，消息一定会被帝一魂知晓！”
<br/>　　“不错，我担心，帝一魂和亘古魔国、幽冥神朝、不死神朝之人很可能会来要人！”
<br/>　　“我们不得不防！”
<br/>　　一众神王级别的长老，纷纷开口说道，他们混沌神朝虽然是除了天池之外实力最强的神朝，但也不可能是三大神朝的对手。
<br/>　　石桌前，一身灰衣的混沌元王，面色淡然，眸光深邃浩瀚。
<br/>　　“诸位，既然本座决定要帮助苏莫，这一切都是不可避免之事！”
<br/>　　混沌元王环视众人，继续道：“不过，你们放心，三大神朝知晓轻重，现如今，神域的空间越发的孱弱，一旦遭遇大战，整个神域都有可能崩溃，这一点三大神朝一清二楚！”
<br/>　　众人闻言，暗暗点头，此言倒是不假，但是，就担心三大神朝不顾全大局。
<br/>　　“元王，帝一魂现在有三大神朝之助，估计实力会快速增长，苏莫的时间真的不多了！”未天神王道，亘古魔国和幽冥神朝、不死神朝，底蕴不比他们低，全力相助帝一魂的情况下，帝一魂实力的提升绝对非常快。
<br/>　　“你有什么想法？”混沌元王问道。
<br/>　　“这紫灵之气，是不是让苏莫吞噬一部分？”未天神王试探着问道。
<br/>　　“这绝对不行！”
<br/>　　“紫灵如此稀少，我们都不够用，如何能给予苏莫？”
<br/>　　“我不赞成！”
<br/>　　未天神王的话音一落，立刻有几名长老开口了，提出了反对的意见。
<br/>　　这紫灵之气对于他们来说，实在是太珍贵了，怎么能给予苏莫？
<br/>　　当然，他们也知道，若是苏莫大量的吞噬紫灵之气，修为必然会飞速的增长，但是，若是苏莫最后成功了还好，若是身死道消，那紫灵之气不是就白白浪费了。
<br/>　　混沌元王闻言沉默，他心中是想要给予苏莫大量的紫灵之气，但是，他知道，必然所有长老都会反对。
<br/>　　“这样吧，这混沌古山上的紫灵之气，让苏莫吞噬一成！”沉默了片刻，混沌元王开口说道。
<br/>　　“什么？”
<br/>　　“一成？太多了！”
<br/>　　“元王，还请三思，这紫灵之气乃是我混沌神朝立足之本！”
<br/>　　霎时之间，便有几名神朝提出了反对意见，给予苏莫一成的紫灵之气，这简直是太冒险了。
<br/>　　“诸位，有付出才有回报！”
<br/>　　混沌元王面色不变，继续道：“苏莫有最大的希望，成为至强者，并且超脱纪元；我等能否永生不朽，还要仰仗苏莫，一点紫灵之气而已，千年之内，鸿蒙之气元神便能补充这些消耗！”
<br/>　　“这……！”众人闻言，纷纷迟疑，混沌元王的话他们自然明白，不过，他们却是看不到多大的希望。
<br/>　　首先，苏莫能不能成为至强者，这不确定，再者，苏莫成为了至强者，能不能超脱纪元，这也不确定；最主要的是，苏莫很可能连神王都没达到，便会陨落在帝一魂的手中。
<br/>　　“就这么定了，本座后续会布置一个阵法，只让他吞噬一成的紫灵之气！”混沌元王不待众人同意，便一言敲定了此事。
