
<br/>　　<br/>　　    是人都有生辰八字，皇帝要登基的时候得由大师看看，他的八字跟哪一天贴合然后才能登基为皇，普通人婚丧嫁娶也会让先生断一下看日子合适不合适，生辰八字在咱们国家的民俗中占有很重的地位，可以这么说吧你要是能把自己的八字给搞明白了，那你这一辈子可能就无往而不利了。<br/>　　<br/>　　    断八字的话一般的先生最多只能看看皮毛，还得是道家大师能评得准一点，至于大部分的人是完全领会不了的，但要是想知道自己的八字好不好其实也简单，就是看八字的轻重，也叫称骨算命（这个在上本书里提过，这本就不详细介绍了）就是看你的八字是几两几钱的。<br/>　　<br/>　　    八字轻者最易见鬼，字重则命硬，这是最典型的八字轻重的利弊，但除此以外还有不少因素会伴随着一个人的八字轻重。<br/>　　<br/>　　    就比如，2两1的命，短命非业谓大凶，平生灾难事重重，凶祸频临陷逆境，终世困苦事不顺，2两4的命，此命推来福禄无，门庭困苦总难荣，六亲骨肉皆无靠，流到他乡作老人。反之命重的必然富贵，就像七两的命，是此命推来福不轻，何须愁虑苦劳心，荣华富贵已天定，正笏垂绅拜紫宸。<br/>　　<br/>　　    七两一钱的就更厉害了，此命生成大不同，公侯卿相在其中，一生自有逍遥福，富贵荣华极品隆。<br/>　　<br/>　　    这种命贵的很，妖魔鬼怪哪里敢近身？<br/>　　<br/>　　    再说王惊蛰跟陈放，他只看了对方一眼，虽然不知道他的生辰八字，但稍微品了下后，就断出他的八字肯定很轻，属于愿意招鬼的那一类，这一点则是从他脸上的骨骼看出来的。<br/>　　<br/>　　    “把你的八字报给我……”王惊蛰问了陈放一句，对方把八字就讲给了他听。<br/>　　<br/>　　    王惊蛰听完后，略微掐指一算，说道：“此命推来骨格轻，鬼怪见了不绕行，若不过房改名姓，也当移徒二三通，你的八字果然不太好”<br/>　　<br/>　　    陈放虽然没听得太明白，但也感觉这不是什么好话，就急头白脸地问道：“这是怎么回事呢？”<br/>　　<br/>　　    “简单地说，就是命轻容易招鬼，再一个就是，你在哪上班？医院，火葬场还是陵园，哪种地方？”王惊蛰蹲下收拾着菜刀装进了包里，他知道自己可能得要和陈放走一趟了。<br/>　　<br/>　　    “你怎么知道我在殡仪馆的？”<br/>　　<br/>　　    王惊蛰背上包，说道：“你走路脚步发飘，身上阴气太重，印堂阴晦，你本身又没被鬼给上了，除了这个原因的话，肯定是跟你的生活环境有关了，唉？你是什么时候出事的？”<br/>　　<br/>　　    “断断续续地有一个月了吧，最开始的时候我开车从一个养老院里带了一个老人的尸体回来，那天晚上我怎么开都没开回到殡仪馆，后来找了一辆车给我带路，才发现殡仪馆离我就一个路口，之后那几天我总觉得自己睡觉的时候会被鬼压床，不管在哪里睡觉都是的……”陈放回忆着跟王惊蛰讲述着自己这一月来的经历，最后说道：“今天出了个更大的麻烦，就是我差点被人给掐死了，我自己都看见了脖子上的手印，但其他人却什么都看不见，到现在我都崩溃了，不知道自己是真撞邪了，还是人格分裂了”<br/>　　<br/>　　    “我也能看见”王惊蛰瞥了一眼他脖子说道。<br/>　　<br/>　　    “啊？你也能看见？”陈放的脸“唰”地一下就白了。<br/>　　<br/>　　    王惊蛰也没跟他解释，两人从城隍庙出来，到了路边拦了辆车，他就跟陈放说去他拉老太太尸体的那个养老院一趟。<br/>　　<br/>　　    路上，陈放按耐不住担忧和疑惑，就问道：“按你说的，我命轻容易招鬼，那早就应该招了吧？可是我在殡仪馆上班都快三年了，怎么就一个月前才出的事呢？”<br/>　　<br/>　　    “这种事吧，也是看机缘的，你以为鬼是遍地走的，说碰就能碰见啊？再一个，你们殡仪馆肯定有人看过风水做过法事，你在那里上班自然没那么容易招上，说来也算你便宜了，你要是换个工作场所的话，没准还早就碰上了”<br/>　　<br/>　　    陈放点头说道：“还真是这么回事”<br/>　　<br/>　　    “还有一点就是，你在沪海，这座城市的风水能镇邪，鬼怪比其他地方也少了很多……”<br/>　　<br/>　　    这座直辖市得算是国内的经济重城了，GDP百分比占据非常的高，创造的经济价值相当惊人，通俗点说就是沪海到处都充满了金钱的味道，特别是在陆家嘴开发区，保险公司，银行等各种金融机构到处都是，开发区郊区工厂林立。<br/>　　<br/>　　    金钱是能压邪的，就拿铜钱来讲，历史越悠久的辟邪的能力自然就越大，戴在身上的话是能起到一些驱邪避灾的效果，同样也是一个道理，一座城市经济太发达那必然是和它的风水有关系，自然沪海这座城市就能镇得住那些魑魅魍魉了。<br/>　　<br/>　　    当然了，这也是相对而言的，只能说相比其他的地方要少，而不是绝对没有的。<br/>　　<br/>　　    三十多分钟后，车开到了陈放曾经拉老太太尸体的那个养老院，两人下车后站在门口也没进去，王惊蛰问他老人是死在哪个房间的。<br/>　　<br/>　　    “三楼，左边第二间吧”<br/>　　<br/>　　    王惊蛰眼睛瞄了过去，顿时就望见三楼的窗户里站着一道黑影，头上顶着花白的头发弯着肩膀，看起来老态龙钟的。<br/>　　<br/>　　    “一个月前死的，那连三七都过去了，魂怎么还没有进阴曹地府呢？”<br/>　　<br/>　　    老太太的阴魂看着挺可怜的，就站在窗户里张望着远方，就好像有种望眼欲穿的意思。<br/>　　<br/>　　    “她的尸体呢？”王惊蛰收回目光问道。<br/>　　<br/>　　    “火化了啊，骨灰盒就放在殡仪馆里呢”<br/>　　<br/>　　    “烧了？”<br/>　　<br/>　　    “对啊，她无儿无女的没人管，都是由民政来安排的，死了之后也没人守灵，没操办什么后世，所以死后第二天就火化了……”<br/>　　<br/>　　    王惊蛰顿时“哦”了一声，说道：“她这是死的不甘心了啊，头七没人烧，又没人给送终，自然心有不甘了，难怪她缠上了你呢，得亏是这老太太生前还算心善，不然你可能早就被他给祸祸了”
