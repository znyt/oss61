　　“不！！！”
<br/>　　放弃的念头，才刚刚闪过，便被叶尘强大的执念压倒，拼尽了最后一丝力气，又迈出了一小步……
<br/>　　嗡~~
<br/>　　就在叶尘完全丧失意识之前，他似乎看到了一团光，紧接着便什么也不知道了……
<br/>　　也不知道过去了多久，
<br/>　　等到叶尘的意识再次恢复过来，发现自己，已经来到了一个玄而又玄的奇妙空间之中。
<br/>　　“这是哪儿？我身上的伤势，居然完全恢复了？”
<br/>　　叶尘又惊又喜，他之前的身体连同神魂，几乎都已经完全崩溃，没想到现在居然完全好了！
<br/>　　而最让叶尘感到震惊的是，他体内的四十九条法则符文，竟然出现了相互融合的趋势！
<br/>　　“灭世者！你终于来了！”
<br/>　　就在叶尘，正百思不得其解的时候，忽然一道缥缈的声音，陡然在这个空间里响起。
<br/>　　下一刻，
<br/>　　叶尘的面前，出现了一道虚幻缥缈的身影。
<br/>　　这是一个一身白袍，而且白发白须的神秘老者，身上透着一股难以形容的神秘气息，就好像整个人，被一道层薄纱笼罩似得，以叶尘如今的修为和实力，竟然都看不清他的面容。
<br/>　　“你是谁？”
<br/>　　叶尘其实已经隐隐猜到，眼前老者的身份。
<br/>　　能够出现在千劫神塔之中，而且如此神秘的老者，多半就是这千劫神塔的主人了！
<br/>　　果不其然，那老者声音无比缓和，道：
<br/>　　“吾名元始，乃是这一方世界的开创者！”
<br/>　　“灭世者，我在这里，等了无数纪元，终于等到你来了！”
<br/>　　叶尘再次听到老者对自己的称呼，顿时大为不解，
<br/>　　“您刚刚称我为，灭世者？”
<br/>　　老者点了点头，
<br/>　　“是的！我所开创的世界，注定要在你的手里终结，所以你便是灭世者！”
<br/>　　叶尘奇道：
<br/>　　“我要毁掉你一手开创的世界，难道你就一点也不生气？”
<br/>　　不料，老者哈哈大笑，
<br/>　　“万物有生有灭，宇宙同样也是如此！我的使命是开创世界，而你的使命则是毁灭世界，我们都是在行使自己的使命罢了！”
<br/>　　“当然，灭世者同样也是创世者，这个世界在你的手里终结，新的世界也将在你的手里诞生！”
<br/>　　“所以，你便是下一任元始大帝！”
<br/>　　叶尘低头沉默了一会，方才又抬起头来，
<br/>　　“我懂了！不过，我还有最后一个问题！”
<br/>　　“混沌法则？”
<br/>　　元始大帝似乎早就料到叶尘会问什么，直接开口道。
<br/>　　叶尘点了点头，
<br/>　　“据说，昊天界没有混沌法则，就永远无法成为真正的主世界，所以我想知道，如何才能领悟五十条大道中的最后一条，混沌法则？”
<br/>　　元始大帝笑了，
<br/>　　虽然叶尘无法看清楚他的脸庞，但是却能清晰的感觉到。
<br/>　　元始大帝缓缓道：
<br/>　　“混沌法则，并不存在！”
<br/>　　“不存在？”
<br/>　　叶尘顿时懵了，
<br/>　　“怎么会不存在？这怎么可能！”
<br/>　　从昊天大帝到昊天界的世界之心，乃至其他大帝级主宰，都在苦苦追寻的混沌法则，竟然不存在？
<br/>　　“难道宇宙之中，根本就没有第五十条法则？”
<br/>　　叶尘忍不住继续追问。
<br/>　　不料，
<br/>　　元始大帝却又再次摇头，
<br/>　　“可以说有，也可以说没有！”
<br/>　　叶尘彻底懵了，
<br/>　　如果不是知道对方，乃是修真界第一位大帝级的强者，估计非得把他当做神经病不可！
<br/>　　元始大帝接着又缓缓道：
<br/>　　“有无相生，生死相成，高下相顷，前后相随……”
<br/>　　“所谓混沌，便是无有、无无、无生、无死、无高、无下、无前、无后……”
<br/>　　叶尘听完之后，顿时一头雾水，完全不知所云。
<br/>　　还想继续再问，
<br/>　　元始大帝忽然又道：
<br/>　　“本帝传你一道法诀，你从此间出去之后，可将那混世魔猿放出，它会助你灭世！”
<br/>　　说完这话，
<br/>　　元始大帝屈指一弹，
<br/>　　嗖！
<br/>　　一道白芒没入叶尘的眉心之中，
<br/>　　嗡~~
<br/>　　叶尘脑袋一阵眩晕，脑海之中，已经多了一道极为玄妙的法诀。
<br/>　　等到叶尘回过神来，元始大帝早已经不见了踪影。
<br/>　　……
<br/>　　与此同时，
<br/>　　千劫神塔外面的众人，并不知道神塔内发生了什么，不过此时整个大会场，已经彻底沸腾了！
<br/>　　因为大屏幕上，叶尘的闯关层数，已经定格在1000层！
<br/>　　而千劫神塔的最顶端，亮起了一道无比璀璨的光芒，直冲苍穹星河之上！
<br/>　　这一刻，
<br/>　　除了合道仙君级的强者，以及几大天骄还能勉强保持镇定以外，其他众人几乎全都跪倒在地。
<br/>　　尤其是元始门上上下下，包括元始门的门主在内，全都跪在地上，磕头朝拜起来，
<br/>　　“神塔亮，圣人出！我元始门守护神塔无数纪元，没想到竟然在我这一代，等到了圣人出世啊！”
<br/>　　元始门门主元道一，跪俯在地上，老泪纵横。
<br/>　　而曦月，也早已经喜极而泣，不停的喃喃道：
<br/>　　“他做到了！他真的做到了！”
<br/>　　叶尘在考核之前，曾经说过，他的目标，是1000层！
<br/>　　当时，所有人都觉得，他是一个狂妄无知的狂徒，就连曦月也不信。
<br/>　　没办法，在此之前，无论是谁，也都不可能相信的啊！
<br/>　　可是现在，他竟然真的说到做到了！

<br/>　　琼碧落张了张嘴，哑口无言。
<br/>　　轰隆隆！
<br/>　　就在叶尘登顶不久，千劫碑忽然一阵剧烈颤动，

<br/>　　千劫碑，最顶端榜首的位置，
<br/>　　第一名昊天的名字，开始缓缓消失，然后变成了一个全新的名字：
<br/>　　叶尘！
<br/>　　紧接着，

<br/>　　因为叶尘两个字的出现，下面所有人的名字，全都往下挪了一个位置！

<br/>　　绝无敌见状，顿时忍不住大吼了一声，险些一头栽倒在地。

<br/>　　他恐怕是有史以来，在千劫碑上待的时间最短的一位了……
