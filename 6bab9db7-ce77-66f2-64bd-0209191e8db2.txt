　　落下来的正是那只十公分的银白大蚊子，它悄然落到捕蛇人的后脖颈处，空气中，寒光点点，只见它长长尖尖的口器抬起，宛如银针锋利，对准男子的脖子骤然的刺了进去，刺下去的时候，这只大蚊子的微小眼睛中冒气幽然的光，让人不禁的毛骨悚然。
<br/>　　长长尖尖的口器轻松无压力的刺进了男子的脖子皮肤之中，过程一气呵成。
<br/>　　霎时间，满嘴喷吐沫的捕蛇人微微蹙眉了下，脖子处微微一股刺痛，刚想抬手去摸摸，但一位客人询问了他几句话，他也顾不得别的，先做生意要紧。
<br/>　　十公分的银白蚊子将口器刺入捕蛇人的皮肤，那双微小的眼睛就一直冒着幽光，幽光越来越甚，它在吸血了，那是兴奋，振奋，愉悦的表现，周围全是人，但没有人发现这只大蚊子。
<br/>　　唯有捕蛇人手中的五步蛇，开始嘶嘶作响起来，像是预警什么。
<br/>　　“别叫，可别吓跑了我的客人”
<br/>　　捕蛇人不轻不重的拍了一下五步蛇的头，继续跟眼前的一众客人卖力推销这条蛇。
<br/>　　“大家看，这条蛇好像凶起来了，看起来都想咬人了”
<br/>　　捕蛇人连忙解释道：“这蛇早被我驯服了，不会咬人的，要是咬人，早就先咬我了”
<br/>　　他说着，忽然浑身颤栗了一下，脸色迅速的苍白起来，双脚变的无力，整个身子摇晃了几下。
<br/>　　怎么回事，有点头晕？
<br/>　　捕蛇人心中疑惑了下，今天太累了吗。
<br/>　　他到现在还没发现自己的后脖位置有一只十公分大的蚊子在吸噬他的血液。
<br/>　　最令人望而生畏的是，这只银白蚊子它吸的并不是捕蛇人体内普通的血液，而是精血。
<br/>　　精血，血本源于先天之精，一个人体内的血量可以占据人身体的百分之八的重量，身体血液应该有个四千毫升左右，根据每个人的情况而定，普通的小蚊子就算吸血，吸走一点点血液，也不会对人有什么太大的影响。
<br/>　　但精血就不一样了，精血的盈亏决定人体的健康与否，人体内血液可以有几千毫升，但精血乃生命精华所在，也就那么十几滴左右，少一滴，生命寿命就等于少了几年。
<br/>　　这只银白蚊子长的独特异常，吸血的手法更为致命，它吸走的却是人的精血，匪夷所思，杀人于无形。
<br/>　　“嘶嘶...”
<br/>　　捕蛇人手中的五步蛇不断的在惊叫，开始不安分的扭动起来。
<br/>　　此时，这条蛇的样子在外人的眼中，就分明是凶光乍现，准备要咬人。
<br/>　　“这蛇越看越凶，大家还是不要买了，五步蛇的蛇毒可是致命的，我可不想走五步就死翘翘了”
<br/>　　不少人纷纷的摇头，捕蛇人闻言大惊，：“各位老板，是人太多，估计蛇受到惊吓了，但它真的不会咬人，要是你们觉得价格高，我们好商量”
<br/>　　话说着，捕蛇人越发的蹙眉，总感觉身体哪里有点不对劲，脸色越发的苍白起来，呼吸都开始急促费劲了。
<br/>　　那是他的生命力在迅速的流逝。
<br/>　　短短的时间，那只银白色的大蚊子就吸走了男子五滴的精血，使得男子的寿命锐减，而且甚至有了生命的危险。
<br/>　　反观蚊子，它腹部微微鼓涨了起来，体重明显的增加了，体表外隐隐还泛着一股微亮的光泽，灿灿生辉。
<br/>　　“嘶嘶...”
<br/>　　突然，捕蛇人手中的五步蛇凶光大亮，再也按捺不住，蛇躯一弓一展，朝着他的脖子处骤然撕咬了过去。
<br/>　　可电光火石间，捕蛇人脖子处一抹光影闪了出去。
<br/>　　“啊...这蛇真的咬人了”
<br/>　　瞬息间，人群中的尖叫声猛然迸发出来，只瞧捕蛇人摇摇欲坠，眼神变的暗淡无光，高大的身躯突然轰然的倒地不起，他的脸色苍白的宛如一张白纸，整个人陷入了一阵晕迷之中，奄奄一息。
<br/>　　“不好了，蛇咬死人了，蛇咬死人了”
<br/>　　这群外地人看到此一幕，吓的三魂不见了七魄，不知如何是好。
<br/>　　这片地带立刻混乱起来。
<br/>　　“怎么回事，都退开点，让我们进去看看”
<br/>　　附近刚好有两位专业的捕蛇人经过，连忙上前而来，今天是蛇王节，而且有蛇王大人坐镇，怎会有蛇咬死人的一幕发生？
<br/>　　他们向前一看，大惊失色，还真有一位同行倒地不起，一张脸白的吓人，而且他的身上也果真趴着一条毒蛇。
<br/>　　“五步蛇？”
<br/>　　五步蛇虽然是常见的毒蛇，但它的蛇毒可是致命的，今天要是在白镇发生蛇咬死人的事件，那么负面影响力之大，将不可想象。
<br/>　　甚至会直接影响蛇王大人的声誉。
<br/>　　一人出手如电的连忙将这条五步蛇给先抓了起来，另一人赶紧检查这位晕迷着的伤口。
<br/>　　很快，他在晕迷者的后脖颈处发现异常，一大快皮肤既然红肿了起来，伤口不但化脓，还溃烂了。
<br/>　　这是明显的中毒征兆啊，而且看样子分明是中毒不浅。
<br/>　　“奇怪，五步蛇的毒应该不是这样子的啊”
<br/>　　这人疑惑起来，但凡中了五步蛇的毒，伤口附近会出现较多淤斑，伤口出血难止，甚至还会有血疱跟水疱。
<br/>　　但眼前的晕迷者，他的伤口只是红肿跟溃散，连血都没有一丝流出来，另外他们连毒牙印也都没发现。
<br/>　　“不管了，先将这人抬到蛇王庙中，慢一点，真的要出人命了”
<br/>　　两位捕蛇人相互看了一眼，当机立断。
<br/>　　与此同时，街道人海中，不少人觉得身上某地方刺痛一下，应该是被蚊子叮咬了，一手拍过去，什么都没有，很快，刺痛的部位就起了一个大红包。
<br/>　　他们也没在意，继续闲逛，被蚊子叮咬了一下很正常，况且现在是夏季，白镇后背靠山，蚊子多也是普遍现象。
<br/>　　但慢慢的，他们身上的红包变的奇痒无比，手抓不止，抓着抓着既然开始化脓跟溃烂，整个过程是非常快的，然后走着走着，他们头一重，直接栽倒在地，瞬间引发一阵恐慌。
<br/>　　渐渐的，白镇街道中，不断有晕迷者，这些人如果是被十公分大的蚊子叮咬到，不到一分钟就会晕迷，奄奄一息，要是被那些一公分的白蚊叮咬，还能坚持一会儿。
<br/>　　但是十几分钟后，脸色越发苍白，伤口奇溃烂，然后不醒与世。
