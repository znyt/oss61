        &lt;h3&gt;第三卷第二百一十四章  调侃&lt;/h3&gt;
<br/>
<br/>        凌辰一行人踏上了回山西的飞机，此行多了一个宁玥。
<br/>
<br/>        对于凌辰来说，他满心满肺都是想着宁雅，别人的心思他哪里知道，况且按照他冷清的性子也不会去猜测的，带着宁玥，无非宁玥也姓宁，是宁雅的妹妹而已。
<br/>
<br/>        董亚子火辣辣的性子对于刘诚之来说就是克星，那个憨憨的刘诚之被凌辰那一番发错了的甜言蜜语惊醒了，伊人在侧何必远寻？
<br/>
<br/>        虽然还是和董亚子斗嘴，可是两个人感觉很快乐，一行四人在飞机上低声谈笑，气氛融洽，凌辰因为见了宁雅心情也很好，脸上是温润如玉，看起来温暖了很多。
<br/>
<br/>        “爱情的力量真是伟大，千年冰山已经化为春水，冬天终于是远离了，刘诚之，你可以不用再冻得瑟瑟发抖了。”董亚子的心已经不再凌辰的身上，自然而然的就没有那种委曲求全的感觉，对凌辰多了一丝调侃。
<br/>
<br/>        说者是调侃，听者觉得心里也很舒坦，但是旁人可不知道是怎么想的。
<br/>
<br/>        “说不定哪一天寒暴来临，避之不及，我还是需要一点距离，亚子，最温暖的地方是你的身边，如火般燃烧的心总是让人热血沸腾。”刘诚之大概是受凌辰的影响久了，情话也是信手拈来。
<br/>
<br/>        “噗嗤！烧不死你的小命？到时候别让我出手相救！”凌辰忍不住笑了，董亚子有时候是那种说翻脸就翻脸的人，别看现在你好他好，真生气起来，也是够刘诚之喝一壶的。
<br/>
<br/>        “这个你就别操心了，烧死也是我心甘情愿，总比爱而不得强。”好兄弟就是要狠狠的捅你一刀，疼不疼我不管，只是想让你明白道理。
<br/>
<br/>        刘诚之已经给凌辰帮着将路铺好了，宁雅的父亲见了，宁雅也见了，后面他要是再怂了，他也是没有办法的，看着他们俩个人拖拖拉拉这么多年，真的是让人又头疼又着急。
<br/>
<br/>        真可谓：皇帝不急太监急，将他这个内当家的已经急得着急上火了。
<br/>
<br/>        “辰哥哥，你们说话就像在打哑谜一样，我怎么听不懂？”宁玥不知道他们三个人说的是什么意思。
<br/>
<br/>        “你姐姐有男朋友吗？”刘诚之问宁玥。
<br/>
<br/>        “没有啊，这么多年我妈妈也不知道给她介绍了多少个优秀的男生，她就是不愿意，今年过年都没有回家，也不知道去了哪里。”宁玥的语气多少有点责怪的意思。
<br/>
<br/>        “哦，过年都不回家啊？你姐姐确实不应该，为什么不回家呢？”凌辰的语气有些冷。
<br/>
<br/>        “年龄大了，有她自己的生活了呗，再说，她那个性子谁能管得住？我妈妈都不敢说她，每次回家大家都将她当小姐一样供着。”宁玥的语气里已经是十分的不满了。
<br/>
<br/>        “她不就是宁家的嫡长女吗？难道嫡长女就应该当受气包？”凌辰对宁雅的这个妹妹已经没有好感了，几句话已经消磨了他的耐心。
<br/>
<br/>        “咳咳咳。”刘诚之咳嗽了几声：“我今天的嗓子不知道怎么回事，老是痒痒的，凌总，你也算是半个医生了，快给我说说咋回事。”
<br/>
<br/>        “吃饱了撑的。”凌辰没好气的说。
<br/>
<br/>        “我吃你了还是喝你的了？好歹我陪了你几天，你竟然这样对我？”刘诚之其实是怕凌辰再怼宁玥，宁玥在宁家可是比宁雅要吃香很多。
<br/>
<br/>        “这样，下午我让你吃个够，全部补回来，怎么样？”凌辰当然也明白刘诚之的意思，奈何现在他就是听不得谁说宁雅的坏话。
<br/>
<br/>        “俩个大胖子就知道吃吃吃。”董亚子嫌弃的说。
<br/>
<br/>        “你说我们俩个是大胖子？”凌辰和刘诚之同时指着自己问董亚子。
<br/>
<br/>        “胖不胖自己照照镜子不就得了。”董亚子没好气的说。
<br/>
<br/>        “看来你有恶趣味，喜欢大胖子！哈哈哈！”凌辰心情变好了。
<br/>
<br/>        “亚子，他们俩个人哪里胖了，这身材刚刚好。”宁玥笑眯眯的插话。
<br/>
<br/>        一行四人就在调侃中飞到了山西，下了飞机凌辰直接将宁玥安排给了家人，让他们带着宁玥逛逛山西的风景，对于他们的家人来说，凌辰突然带回来了一个小美女，虽然没有挑明是不是女朋友，但是他们一定要照顾好宁玥给宁玥留一个好印象，毕竟凌辰到了娶妻生子的时候了。
<br/>
<br/>        他、刘诚之和董亚子即刻投入到工作中，三一五说到就到，有专门的部门管理，但是他们也需要提供相应的数据和证据。
<br/>
<br/>        在刘诚之去甸城的事情上他们发生了分歧，分歧的原因是董亚子，董亚子坚决不让刘诚之去甸城。
<br/>
<br/>        “刘诚之，每年的打假活动都是在斗智斗勇，有可能会发生一些不好的事情，甸城比较远，你不能一个人过去，那样会孤立无援。”
<br/>
<br/>        “亚子分析的很对，诚之我们还是安排别人过去吧。”凌辰也不同意刘诚之过去，他宁愿他自己亲自去。
<br/>
<br/>        “凌总，董总，我在年前就说过了，我不想在累述太多遍，从道义上来讲，我应该担起责任，从私人的感情来讲，我不希望董亚子受到任何威胁！”刘诚之斩钉截铁，没有任何回旋的余地。
<br/>
<br/>        “我的身份没有人知道，就你们俩个知道，在年前见过何少华，何少华从来都不将我放在眼里，我过去他是不会将我当回事情的，相反我会利用我的身份做更多的事情。”
<br/>
<br/>        “可是......”
<br/>
<br/>        “你们在这边加大力度，我在那边游山玩水，就这样决定了。”刘诚之直接就拍板了，刘诚之此时表现的强势让凌辰和董亚子无言以对。
<br/>
<br/>        “诚之哥哥，你要去甸城？”宁玥从外面推门进来。
<br/>
<br/>        “你知道甸城？”刘诚之没有想到宁玥会进来。
<br/>
<br/>        “之前我还去过，爸爸认识那边的一个人，好像他们之间有生意来往。我能不能和你一起过去玩几天？”宁玥问刘诚之。
<br/>
<br/>        “你咋哪里都想去，在山西让你凌辰哥哥陪着不香吗？”刘诚之简直是无语了，他能带着宁玥过去？
<br/>
<br/>        “凌辰哥哥不是忙嘛，哪有时间陪我。”他们几个一回来就开始讨论工作，都将宁玥忘记了，虽然有人陪着宁玥玩，但是宁玥还是觉得她受到了冷落。
<br/>
<br/>        “宁玥，这几天让你亚子姐姐和凌辰哥哥陪着你好好玩几天，然后让你凌辰哥哥送你回去，你可别到处乱跑了，我没有办法给宁伯伯交代。”刘诚之交代着。
<br/>
<br/>        “对了，宁玥，你认识甸城的谁？”
<br/>
<br/>        “秦少卿和陆玉，去年我和爸爸一起过去的。爸爸想过去参加他们的项目。”宁玥大大咧咧的说。
<br/>
<br/>        “你已经去过了甸城，就不要再去了，听话啊，这几天就让你凌辰哥哥陪着你，你想去哪里只管吩咐他。”
<br/>
<br/>        送走了宁玥，刘诚之对凌辰说：“这几天好好招呼宁玥，让她回去给你美言几句，你的追妻路就更加轻松了。另外那天我好像听陈子昂也说过和宁伯伯要合作甸城的项目，你将这个情况告诉宁雅，让宁雅告诉陈子昂。”
<br/>
<br/>        “为什么要告诉陈子昂？”凌辰不解。
<br/>
<br/>        “你没有发现你未来的老婆现在依仗着陈子昂这颗新秀吗？”刘诚之反问凌辰。
<br/>
<br/>        凌辰突然恍然大悟也深感愧疚，他需要更加努力，他才是宁雅的依仗！
