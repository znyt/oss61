　　摇晃酒杯的动作一僵，月山习轻笑了一声，向叶使了一个眼神。
<br/>　　随后，叶转身拿出一张信封，恭敬的递了上来。
<br/>　　“这是拍卖会的邀请函，时间是下周五，刘先生有兴趣的话，可以凭这张邀请函参加。”月山习放下酒杯，和刘哔对视着。
<br/>　　心中微微一动，刘哔结果信封，从里面拿出一张金色的邀请函来，上面没有署名，只是简单写着拍卖会的地址，最下方有一个略显诡异的印章。
<br/>　　这么轻易的就能将邀请函搞到手，让他有些意外，一时间，他也吃不准月山习在想什么。
<br/>　　“刘先生啊，我们这种家族，和您这样的高阶喰种是没法比的，而且CCG这段时间监管的这么严格，如果您长时间在我们这里逗留，恐怕……”
<br/>　　脸上的笑容有些苦涩，月山习叹了口气，十分小心的开口，他怕自己的话语会引起刘哔的不满。
<br/>　　这样高等级的喰种，不是他可以得罪的，而月山家的地位日益稳固，他们不想和刘哔扯上什么关系。
<br/>　　一旦暴露，这可是牵扯到整个家族的事情，容不得他有半点疏忽。
<br/>　　“恩，放心，参加完拍卖会，我就会离开。”猜到了月山习心中的想法，刘哔嘴角微微一扯，毫不犹豫的答应了下来。
<br/>　　反正月山家还知不知道能坚持到拍卖会开始呢，依照他留下来的线索，估计很快CCG就会找上门来。
<br/>　　将拍卖会的事情决定好后，月山习也是松了一口气，接下来的晚宴，在一片愉悦的气氛中进行了下去。
<br/>　　酒足饭饱，刘哔带着什造离开了餐厅，看着两人离去的背影，月山习脸色微微沉了下来。
<br/>　　他总觉得有些不对，事情进展的，实在是太过于轻松了。
<br/>　　像刘哔这样的高阶喰种，每一个都会有自己的组织，而且找上他，也绝对不只是想要摆脱CCG的追踪。
<br/>　　本来他以为刘哔背后站着的是青铜树，但根据金木告诉他的，这个猜测立马被他否决了。
<br/>　　也不可能是小丑，他们家族和小丑关系如此紧密，不可能会不知道这人的消息。
<br/>　　至于其他的一些组织，月山习都大概了解一些，都一一否定了。
<br/>　　但可以确认的是，这个组织针对月山家，一定有什么阴谋，只是一时半会，他也找不到什么头绪。
<br/>　　坐在餐桌面前，月山习苦思冥想着，片刻，站起了身来。
<br/>　　他需要自己父亲的意见，至少家族也应该警惕起来了。
<br/>　　……
<br/>　　“喂，你居然是女的！”
<br/>　　“哈？你怎么会认为我是男的？”
<br/>　　“可你明明这么平！”
<br/>　　“……”
<br/>　　“而且你还这么虎！”
<br/>　　“……”
<br/>　　“再说你……”
<br/>　　“闭嘴，闭嘴闭嘴闭嘴！”
<br/>　　月山家的一处客房里，刘哔站在门口，带着不可置信的神色看着什造。
<br/>　　他的对面，什造躲在浴室门后，气呼呼的看着刘哔。
<br/>　　这真是一个毁三观的事情，在此之前，他一直都将什造当成男生看待，毕竟对方身上没有一点女性特征。
<br/>　　但细致观察之后，还是发现了一些不同，在某些地方，什造确实是偏女性化多一些。
<br/>　　“问够了没有，我要洗澡，洗澡！”气呼呼的抓着大门，想要将门口关闭，什造明显真的是有些气愤。
<br/>　　拍了拍额头，刘哔带着不能理解的表情，转身回到了床上，口中碎碎念的嘀咕着什么。
<br/>　　他还是很难接受，什造是女生的事实。
<br/>　　浴室传来哗啦啦的流水声，刘哔趴在床上，随手拿过了手机。
<br/>　　虽然这个世界无比危险，但终归给了他一些安宁的感觉，也让他想起了末世还没爆发之前的场景。
<br/>　　车水，马龙，高楼大厦，形形色色的人群，还有各种娱乐设施，他和刘莽坐着，在街边的小摊上吃着烧烤。
<br/>　　那是一段多么美好的时光，可惜，都已经逝去了好些年了。
<br/>　　可他也没想改变什么，虽然也什么改变不了。
<br/>　　之所以这么努力的提升自己战力，他的目的也不是为了什么拯救末世，而是想要让自己和弟弟能够过得更好。
<br/>　　至于那些和他没什么关系的人，刘哔也没有放在心上。
<br/>　　反正人有很多，即使死掉一些也没关系，他没有义务，也不想去做那些英雄应该做的事情。
<br/>　　或许曾经想过，但十年的打磨，让他看清了很多东西。
<br/>　　有的时候，他总是再想，像个禽兽一样混沌的活着，或是清醒的死去，哪一个更为糟糕。
<br/>　　每一次，他都不想面对自己心中的答案，可最终还是选择了顺其自然。
<br/>　　他只是想活，至于活下去的手段光不光彩，真的不太重要。
<br/>　　死人是没办法感受世界的，死了就是死了，什么都消失了。
<br/>　　他不想消失。
<br/>　　十年饮冰，难凉热血，最终，他妥协了。
<br/>　　算了，
<br/>　　只要能活下去，
<br/>　　期望这种东西，能带来的，不也只有失望吗。
<br/>　　伸手拿过枕边的控制器，按下了关灯的按钮，顿时，房间陷入了一片黑暗。
<br/>　　置身处于这种环境之下，刘哔轻松了很多，或许是因为习惯，他并不喜欢光亮。
<br/>　　有灯罩着，会影响到他的睡眠。
<br/>　　大概过了十多分钟，浴室中水流的声音安静了下来。
<br/>　　有人在走动，是什造，这傻妮子，没有开灯。
<br/>　　一阵悉悉索索的声音响起，下意识的，刘哔睁开眼睛，看到床边的黑影。
<br/>　　什造站在床边，将浴巾叠起，放在了床头，随后躺在了他的身边。
<br/>　　似乎是在躲避什么，什造背对着刘哔，和他隔开了一些距离。
<br/>　　借助着朦胧的月光，什造的肩膀如同羊脂玉一般，裸露在外面，增添了一些妩媚。
<br/>　　目光中带着温柔，刘哔转了个身，将美人拥入了怀中，两人紧紧贴合在了一起。
<br/>　　感受到身后的动作，什造身体微颤，抓住了扣在她小腹间的大手。
<br/>　　并不是拒绝，而是她现在有点慌乱，这种情况，妈妈和大叔都没教过她应该如何处理。
<br/>　　“阿嚏！”
<br/>　　将头侧到一边，刘哔嗅到一股清香的味道，忍不住鼻腔的瘙痒，不合时宜的打了一个喷嚏。
