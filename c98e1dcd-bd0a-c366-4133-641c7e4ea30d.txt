　　林峰左手拿着AK47点射，右手拉着斩灵刀斩杀那些突进来的丧尸。

<br/>　　林峰一路横扫，所向无敌。

<br/>　　然后就是剧烈的爆炸声音和人的惨叫。

<br/>　　此时此刻，有个干事已经是下半身被轻而易举的撕碎，死不瞑目的躺在了地上，内脏大肠更是流了一地。

<br/>　　“看来这个就是黑龙镇内的B级怪物了，还真是够强大的啊，隔着这么远我都可以察觉到对方身上的一股力量感。”

<br/>　　而在另外一边。

<br/>　　在他们身后有大概四百个战士会成全。
<br/>　　此时此刻，他们每个人都面色难看。
<br/>　　在他们面前，A级怪物变异响尾蛇正吐着蛇信子盯着他们，在这个变异响尾蛇身边还有足足三头八阶的变异蛇。

<br/>　　寒鸦道：“变异响尾蛇我来对付，其余的八阶变异蛇你们来解决！”
<br/>　　寒鸦说完就是冲了过去。
<br/>　　“上！”
<br/>　　罗敏等人紧随其后。
<br/>　　话说八阶的变异熊被林峰火箭弹来了一下后，身上都着火了，但是却没有受到多大的伤害，对方身上的一层厚实的皮毛，简直也是无敌的存在。
<br/>　　“这皮毛肯定很值钱了。”
<br/>　　林峰二话不说就再次一发射了过去直接把变异熊打的趔趄，但是却无伤大雅，对方彻底的暴怒了又是一阵怒吼，砸碎了身边的一根电线杆子，然后当起棍子就是朝林峰砸来。
<br/>　　林峰立即闪躲，砰，水泥的电线杆子直接断裂。
<br/>　　林峰捡起一段碎石就是朝变异熊丢了过去。
<br/>　　砰！
<br/>　　变异熊一拳就是砸爆了，然后整个人就怒冲而来。
<br/>　　林峰不敢召唤出99坦克，他怕自己的99坦克都要被这个残暴的玩意给砸爆了。
<br/>　　二话不说掏出了98K，瞄准了暴怒之中的变异熊眼珠子。这个变异熊浑身都是绝对防御，唯有眼珠子是弱点。
<br/>　　砰！
<br/>　　噗嗤！
<br/>　　子弹直接射穿了变异熊的眼睛，顿时对方疼的暴怒连连。
<br/>　　“一个！”
<br/>　　林峰拉大栓，子弹上膛后，再次瞄准了另外一个眼睛。
<br/>　　砰！
<br/>　　噗嗤！
<br/>　　又是一个眼睛直接被打爆。
<br/>　　林峰把98K放入物品栏，然后取出了汽油弹朝旁边一些企图过来偷袭的变异狗丢了过去，瞬间七八头变异狗被烧的皮开肉绽。
<br/>　　林峰拿出斩灵刀就是朝失去了双眼的变异熊斩杀而去。
<br/>　　刷！
<br/>　　一刀下去！对方身上出现一个大血口。
<br/>　　变异熊暴怒的一巴掌拍了下来，但是他没了眼睛，只能纯粹瞎拍，林峰脚下一动直接躲开了。

<br/>　　变异熊一巴掌把水泥路都给怕出巴掌印。
<br/>　　林峰趁此又是一刀切了上去，顿时变异熊后背再次出现几道血痕。

<br/>　　林峰看了眼这变异熊尸体，直接挖出了他的晶石后，整个尸体收入物品栏，这个变异熊虽然身体上有几个刀痕，没有之前的变异鳄鱼那么完整，但至少也没缺胳膊少腿还是有价值的。
