　　从九叔家中看过一些资料，凭借一些特征，刘哔也大致能区分出这些僵尸的等级。
<br/>　　这些僵尸，身上大致都长着黑色或者白色的毛发，都是一些很低级的白僵和走僵，但也有一小部分，身上的毛发已经变成了浅绿色，应该是刚刚进化不久的毛僵。
<br/>　　这些毛僵，大概有三五只的样子，彼此距离都很相近，给刘哔带来了一些威胁。
<br/>　　以他目前的实力，寻常僵尸对他已经造成不了什么威胁了，唯一能和他交手的，也只有修炼百年以上的僵尸。
<br/>　　而这些僵尸，在这个世界中也不是十分常见，这个村子有这么多，也是鬼鹰强行催化的缘故。
<br/>　　就如同之前在义庄击杀的那头毛僵一样，虽然等级都差不多，但实力上却相差甚远。
<br/>　　认真的说，这些僵尸只能算伪毛僵罢了，不过他们体内的尸丹倒和真正的毛僵没有多大差别，这也是让刘哔动心的原因。
<br/>　　他还有不到一个月就要离开这个未免了，但帮紫女解除封印的任务还没有多大进展，他可不想接受任务失败的惩罚。
<br/>　　“这些僵尸，就是货物，但我送过来的时候，它们还不是这样。”鼓了鼓胆，任山凑到刘哔耳边，声如蚊呐般的说道。
<br/>　　听到任山的提示，刘哔心中一动，有些懊恼。
<br/>　　他倒是没有想到，既然鬼鹰修炼需要尸丹，那直接养在尸体里，用的时候在取就好了。
<br/>　　这样存放的话，不但尸丹内的尸气不会流失，反倒是随着时间会逐渐增强，对于鬼修来说，这也是最好的存放方式。
<br/>　　但这样一来的话，刘哔想要取到尸丹，就只能自己动手从这些僵尸身上收取了。
<br/>　　可是这么多僵尸，再加上实力不明的鬼修少年，他应对起来也要承担不小的风险。
<br/>　　“刘先生，要不还是算了吧，咱不能为了几颗尸丹，不要命了吧。”看刘哔表情有些犹豫，任山小声劝道。
<br/>　　之所以开口相劝，是他担心自身的安危，如果刘哔真肯听他的，不和尸**手，那自然是再好不过的了。
<br/>　　没有理会任山，刘哔看着眼前的尸群，心中像是下了什么决定，从空间戒指中掏出了一根龙头拐杖。
<br/>　　这是鬼鹰的武器，被他击杀后就顺手收走了，但还不知道有什么实际用处，但却比一般的铜铁武器都坚硬许多。
<br/>　　他还是决定出手了，任务已经完成到这种地步，现在放弃，未免太过于可惜。
<br/>　　夜色朦胧，一片漆黑之下，任山分辨不出刘哔手中是什么武器，只是看到一根粗长的东西，像是棍棒。
<br/>　　他有些好奇，不知道这根棍棒是从什么地方掏出来的，这么大的东西，不太可能藏在衣服里。
<br/>　　但这并不是他所关心的东西，他只知道，刘哔没有听他的劝告，准备动手了。
<br/>　　而他也不准备留下来帮忙，反而已经计划逃走了。
<br/>　　面临这样的情况，他并不认为刘哔能讨到什么便宜，这些僵尸的实力，他曾经可是见过。
<br/>　　和之前他所说的不同，那些或许全部变成了僵尸，这是他早就知道的事实，甚至当时尸变的时候，他就在现场。
<br/>　　之所以没有早告诉刘哔实情，是他想要借助这些僵尸缠住刘哔，从而达到让他逃走的目的。
<br/>　　虽然刘哔已经答应放过他，但与其奢望敌人放过，他还是习惯将性命掌握在自己手中。
<br/>　　并不清楚任山的想法，既然已经决定动手，刘哔便也没再犹豫，挑选了距离他最近的一个毛僵作为目标，直接飞身冲了上去。
<br/>　　趁着尸群没有反应过来，他想要先解决一头对方的顶级战力，那样接下来就会轻松许多。
<br/>　　一道亮丽的红色在半空中划过，刘哔速度很来，几乎瞬间就来到了那头毛僵面前，当头一拐杖劈了下去。
<br/>　　被这突然的袭击搞得一愣，那头毛僵怒吼一声，仓促将双臂举到了头顶，一阵金戈交错的声音响起，刘哔手中传来了一股强大的反震力，像是敲到了钢铁上一般。
<br/>　　而那头毛僵也没有占到什么便宜，此时它的双臂如同折断了一般，向下耷拉了下来，身体更是像断线的风筝一般向后倒飞出去。
<br/>　　毕竟太过于仓促，没有任何准备的情况下硬挡刘哔的全力一击，即使它是毛僵，也根本占不到一点便宜。
<br/>　　一击得手，刘哔还想追击，但四周的尸群已经反映了过来，吼叫着将刘哔围在场地中间。
<br/>　　见状，他也只能叹了口气，放弃了追击的想法，转而和周遭的僵尸交战了起来。
<br/>　　没能秒掉这头毛僵，让他感觉有些可惜，但好在将它的双臂给折断了，也算给他创造了不小的优势。
<br/>　　耳边充斥着僵尸的吼声，刘哔不闪不避，任凭这些攻击落在他的身上，反而借助这些僵尸攻击的机会，出手解决了几头低级的走僵。
<br/>　　实际上，这些僵尸最难对付的地方就是其身上携带的这些尸毒，寻常人和僵尸交手，为了不感染尸毒，总会畏手畏脚，发挥不出全部的实力。
<br/>　　但刘哔不同，他本身就是丧尸，不管是任何病毒对他都产生不了什么作用，而‘不死’技能的特性，也让他根本不必顾忌除头部以外的任何攻击。
<br/>　　如此一来，战斗自然是他占据了上风，场中的僵尸的数量，也以一个惊人的速度减少着。
<br/>　　没了尸毒的特性，这些僵尸就是一些皮厚一点的普通怪物罢了，况且行动还那么僵硬，他应对起来无比自如。
<br/>　　但身在围墙上的任山就不是这样想的了，在他看来，现在刘哔身陷尸群，浴血奋战，此时简直是他逃走的最佳时机。
<br/>　　故此，他心中没有任何犹豫，转身就要离开，至于刘哔的死活，他压根不会去在意。
<br/>　　反正就算刘哔侥幸活下来，那个时候他也早就离开，世界之大，他如果刻意躲藏起来，刘哔也根本找不到他。
<br/>　　事实上，他还是希望刘哔会死在这里，然后他借此机会回去将任婷婷杀掉，这样他所做的事情，就不会被别人知道了。
