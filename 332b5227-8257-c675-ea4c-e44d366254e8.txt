　　轰！
<br/>　　待那些白色光芒涌入到神王路西法的身上，后者的气息顿时开始节节攀升，身上的光芒也越发璀璨耀眼，远远望去，简直犹如太阳一般！
<br/>　　“这是神界赫赫有名的光明祭祀术，说白了就是邪术，靠牺牲手下的生命元气，来提升他路西法一个人的实力！”
<br/>　　远处观战的无涯老人忍不住开口道。
<br/>　　一旁的魔祖冷冷一笑，
<br/>　　“什么邪术正术？这些人与其被那姓叶的小子各个击破，倒还不如将实力全部献祭给路西法，说不定还有一拼之力！嘿嘿，这路西法估计也想不到，区区一个世俗小子，竟然可以将他逼迫到如此程度吧！桀桀桀！”
<br/>　　无涯老人见魔祖发笑，不由得眉头微微一皱，
<br/>　　“魔祖，咱们是不是也该出手了帮他一下？”
<br/>　　魔祖桀桀一笑，
<br/>　　“无涯，你未免也太高看那小子了吧！路西法连光明祭祀这样的狠招都使了出来，即便杀不死那小子，也足以将他的全部底牌逼出来，我们倒不如先见观其变的好！”
<br/>　　“更何况，这四周还有数十万的世俗军队，同样不容轻视啊！”
<br/>　　无涯老人听到这话，顿时眉头皱的更紧了，
<br/>　　“正所谓唇亡齿寒！倘若路西法真的落败，接下来可就轮到我们了，还是不要托大的好！”
<br/>　　魔祖顿时有些不耐，正要开口反驳，那华山之巅，传来一道巨响，却是叶尘和神王路西法，已经再次交上了手。
<br/>　　在神界众强者类似于献祭的支撑下，路西法的气息陡然攀升了数倍，已经无限接近于化神强者！
<br/>　　“末日审判！”
<br/>　　路西法高举手中权杖，璀璨的光芒，将整座华山都笼罩其中，仿佛要将世上的一切吞没，犹如末日降临。
<br/>　　轰隆！
<br/>　　权杖落下，以迅雷不及掩耳之势，狠狠砸落在叶尘的头顶上空。
<br/>　　“死吧！”
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，只来得及扬起手中紫琼剑，奋力挡了上去。
<br/>　　砰！
<br/>　　一道巨响过后，叶尘直接连人带剑倒飞而出，被狠狠砸落到下方的山脉之中，顿时形成了一个巨大的深坑！
<br/>　　轰隆隆！
<br/>　　整座上千高的山峰，竟然瞬间崩塌！
<br/>　　周围掀起数百米高的蘑菇云，犹如刚刚被核弹攻击过一般。
<br/>　　“好恐怖！”
<br/>　　四周华夏军队看到这一幕，顿时全都纷纷倒吸了一口凉气。
<br/>　　一击之威，竟然毁掉了一座上千米高的大山，古代神话中的神仙，恐怕也不过如此吧？
<br/>　　继而，众人又开始担忧起来，
<br/>　　“叶帅不会有事吧？”
<br/>　　彭！
<br/>　　众人这道念头才刚刚升起，又是一道巨响，那座被路西法毁成一片废墟的山峰，骤然暴涨开来，继而一道紫色光芒冲天而起！
<br/>　　“想杀我，你还不够格！”
<br/>　　伴随着一道冷漠的声音响起，那紫色光芒之中，出现了一道瘦削少年的身影，只是身上血迹斑斑，显然方才那一击，已经令他受了伤！
<br/>　　“哼！我倒要看看，你能撑到几时！”
<br/>　　路西法看到叶尘这么快，就重新站了起来，顿时再次暴喝一声，继续朝叶尘攻了过来。
<br/>　　路西法是因为接受了其他神界强者的祭祀，所以修为才能突然暴涨，但这种力量毕竟不是自己的，所以也就无法持久，他必须速战速决。
<br/>　　轰轰轰！
<br/>　　又是一连十多次攻击，而且招招凶悍霸道，没有一点留情。
<br/>　　然而，
<br/>　　叶尘似乎是有些惧了，不再和他正面对抗，只是利用空间法则，不停的躲闪抵抗。
<br/>　　时间一久，路西法顿时有些急了，
<br/>　　对方明显是想和他打消耗战，等到光明祭祀的效果消失，他的优势也就荡然无存了。
<br/>　　“小子！你就只会躲吗？”
<br/>　　路西法愤怒的大吼大叫，而叶尘却根本不予理会，依旧不和他正面对抗。
<br/>　　眼看着自己的实力已经开始有减弱的趋势，路西法忽然大吼一声，反而朝华山脚下冲去。
<br/>　　“我杀不了你，那就先杀光山下的人！”
<br/>　　以路西法如今的速度，眨眼之间便已经冲到数十里外，直奔易山河和萧若曦等人扑了过去。
<br/>　　唰！
<br/>　　一道人影在空中一连闪烁几次，闪电般挡在了路西法的面前，赫然正是叶尘！
<br/>　　“你找死！”
<br/>　　叶尘两眼发红，扬起手中的紫琼剑，便迎头斩落。
<br/>　　路西法不惊反喜，扬起权杖迎了上去，
<br/>　　“来得好！”
<br/>　　轰隆！
<br/>　　紫琼剑与金色权杖相撞之后，顿时发出一道惊天巨响，在两人强大的能量波动之下，周围千米之内的山峰，瞬间全部倒塌！
<br/>　　甚至连空间，似乎也发生了扭曲，足足过了好一会，众人方才重新看清楚战场中的情况。
<br/>　　却见叶尘已经倒飞而出！
<br/>　　“死吧！”
<br/>　　路西法满脸兴奋之色，不等叶尘站稳脚步，已经再次冲杀了过来！
<br/>　　“啊！！”
<br/>　　周围围观的数十万军队，全都纷纷惊呼起来，有的人甚至已经不敢再看。
<br/>　　“叶帅……要败了吗？”
<br/>　　就在所有人一阵绝望之时，忽然一道淡漠的声音响彻天地，
<br/>　　“逆！生！百！转！”
<br/>　　轰！
<br/>　　原本已经陷入绝境叶尘，身上陡然爆发出一股强悍无比的霸道气息，紫色剑芒更是直冲云斗！
<br/>　　“什么！！”
<br/>　　路西法察觉到叶尘身上的气息变化，顿时两眼瞪得老大，惊疑中带着浓浓的不甘之色。
<br/>　　“这不可能！”
<br/>　　路西法心头狂吼的同时，也施展出了自己的最强一击。
<br/>　　轰隆隆！
<br/>　　两道犹如天神般的身影，再次狠狠撞击在一处，整座华山周围，已经完全沦为了一片废墟，甚至一些碎石屑，激飞到数十里外，竟下起了石雨！
<br/>　　“谁赢了？”
<br/>　　四周围观的众人，一个个全都死死盯着，那片已经狼藉不堪的战场。
<br/>　　只见那璀璨的光芒落下，空中两道身影，昂然悬立在空中，谁也没有倒下，似乎是旗鼓相当，
<br/>　　“咳咳咳！”
<br/>　　路西法一阵咳嗽过后，依旧一脸难以置信，
<br/>　　“你、你这是什么秘术？你的实力，怎么会变强这么多？”
<br/>　　叶尘随手拍了拍身上的尘埃，冷冷一笑，
<br/>　　“我的手段，又岂是你能揣测的？”
<br/>　　路西法先是一愣，继而桀桀一笑，
<br/>　　“即便是施展了秘术，你的实力也不过和我相当而已，你以为你杀得了我？”
<br/>　　“是吗？”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“逆生百转！”
<br/>　　“二转！”
<br/>　　“三转！”
<br/>　　随着叶尘淡漠的声音落下，他身上的气息陡然再次攀升！
