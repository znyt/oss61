宋晓的突然出现让夜滢和厉嫣然都一楞，要知道这场媒体发布会可是她们精心安排的，现在突然杀出一个人物就代表了不可知的变数，刚才夏夕绾已经强行扭着了局面，现在加上宋晓，绝对不是什么好事。
<br/>
<br/>        夜滢迅速道，“宋晓，你来的正好，范恬的孩子没有了，罪魁祸首就是夏夕绾！”
<br/>
<br/>        范恬接受到了暗示，当即跑到了宋晓的身边，拉住了他的胳膊，“宋晓，我流产了，我现在肚子好痛，你带我离开这里吧。”
<br/>
<br/>        宋晓冷眼看着范恬，直接将她给推开了，“范恬，孩子是怎么没有的，难道你心里没点数吗？”
<br/>
<br/>        范恬一僵。
<br/>
<br/>        不，他不应该知道啊。
<br/>
<br/>        这时记者们纷纷发话宋晓，范恬一直说她的孩子是被夏夕绾用流产药给害死的，难道另有隐情？
<br/>
<br/>        宋晓冷笑了一声，“她说的没错，孩子是用流产药害死的，但是，这个流产药可不是夏夕绾下的，而是范恬自己下的！”
<br/>
<br/>        什么？
<br/>
<br/>        全场震惊了，范恬肚子里的孩子竟然是…范恬自己下了流产药给害死的？
<br/>
<br/>        都说虎毒不食子，范恬亲手杀了自己的孩子，这怎么可能？
<br/>
<br/>        范恬脸色一白，面如死灰，她当即否认道，“宋晓，你在胡说些什么，这可是我的孩子，我为什么要害死他？”
<br/>
<br/>        “范恬，你看这是什么？”宋晓拿出了几张照片，“这是你去黑市买流产药的照片，当场被拍下了，如果你还不承认，我手上还有你汇款交易的银行记录！”
<br/>
<br/>        范恬看着那些照片，她浑身颤抖，怎么可能，宋晓是怎么知道这一切的？
<br/>
<br/>        没错，流产药是她自己下的，也是她去黑市亲自买的，这一次她想将夏夕绾置于死地，所以每一个细节都没有假手于人，她敢笃定这一切不会被发现的。
<br/>
<br/>        现在宋晓是怎么知道的，他竟然还掌握了证据！
<br/>
<br/>        铁证如山，她还如何辩解？
<br/>
<br/>        这些媒体记者们都兴奋起来了，原来事情的真相是这样的，他们将镜头聚焦在了范恬的脸上，
<br/>
<br/>        范恬，原来这一切都是你自导自演的，你竟然杀了自己的孩子，太残忍了。
<br/>
<br/>        范恬，你为什么要这样做，你做这些就是要栽赃陷害夏夕绾吗？
<br/>
<br/>        范恬，那你和夜老究竟是怎么回事，你是不是真的喜欢夜老？
<br/>
<br/>        这些问题一个接一个砸了下来，范恬知道自己完了，她双腿一软，直接狼狈的瘫坐在了地上。
<br/>
<br/>        宋晓看着范恬，冷哼了一声，“她和夜老究竟是怎么回事，你们好好看一看我这张脸不就清楚了？”
<br/>
<br/>        大家迅速看向宋晓，这么一看宋晓的脸跟年轻时的夜老竟然十分的相似！
<br/>
<br/>        宋晓继续道，“你们是不是看出我和夜老很相似了，其实我是去整容的！我那么爱范恬，和她如胶似漆的时候她向我提出了一个要求，她让我去整容，她给了我一张夜老年轻时的照片让我去整成一模一样的！”
<br/>
<br/>        宋晓的眼里露出了痛色，“范恬，我那么爱你，你让我去整容我也去了，你轻贱我就算了，但是你不该杀了我们的孩子，孩子没有的那一刻我跟你也就完了！”
