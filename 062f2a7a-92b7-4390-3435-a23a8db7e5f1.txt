　　第24章
<br/>　　林道，斜阳，几匹马，一行人。
<br/>　　“左……左，右……右。”高飞豹不停呵斥。
<br/>　　“你就不能下马牵着他们走。”高飞虎忍无可忍。
<br/>　　“不行，我最起码是有马一族，下了马人们会以为我是赶车的。”高飞豹。
<br/>　　高飞虎无言可对。
<br/>　　高飞豹骑在一匹瞎马上，拽着另外一匹，各自瞎了一只眼，走路时很容易转圈。几里路走得他汗流浃背。
<br/>　　他擦掉额头汗珠，眼望着将下山的夕阳发出一声长叹：“此情此景，真让我忍不住赋诗一首。”
<br/>　　“快闭嘴，就你那学问，还赋诗一首。”高飞虎有时候很受不了自己这个弟弟。
<br/>　　“大哥，你说这世上真有五雷法么？”高飞豹忽然问。
<br/>　　高飞虎心中一动，知道他和自己打的是一样主意。
<br/>　　“据说是有的，咱们亲眼见过，还能有假。”
<br/>　　“那天离得太远了，而且有雾，其实都没看清楚。”
<br/>　　“这个还是等军师回来，他见多识广，一定知道。”
<br/>　　“现在，蛤蟆寨可是有不少粮食啊。”高飞豹刚说完，又立刻道：“左……右，左……”
<br/>　　远远草丛中，趴着四个人。
<br/>　　李家庄的财主李壮窝在草丛中，已经换了穷人的麻衣，身后跟着自己三个儿子。
<br/>　　“爹，你说这世上真有神仙么？”大儿子问。
<br/>　　“怎么没有！否则李狗剩家里两匹马是怎么来的，只能是神仙给的。”
<br/>　　“可咱们在这里等这么久了，也没见到神仙。”二儿子道。
<br/>　　“等一等，这种事要心诚则灵。”
<br/>　　正说着话，前方传来一阵阵马蹄声。李壮眼睛一亮：“来了，你们三个听好，看我表演。”
<br/>　　“老二，别生气了，回去让咱娘烧几道菜，喝点酒。”
<br/>　　“大哥，你说这两匹马卖出去，有人会要吗？”
<br/>　　“估计，没人和你一样蠢。”
<br/>　　“山大王饶命啊，山大王饶命啊……”
<br/>　　突然四个人影冲出来，噗通跪倒在二人面前。
<br/>　　高飞豹吓了一跳，怒喝道：“你们是什么人？”
<br/>　　“大王，我们都是山下的穷人，你们有麦子要割没有，有马随便给我们十几二十匹就行……”
<br/>　　两句话，句句扎在高飞豹心窝。他用麦子换了一匹瞎马，不知青牛山多少山贼笑话自己呢。我才有两匹瞎马，你们竟然管我要马……还十几二十匹！
<br/>　　他翻身下马，一脚将李壮踹翻在地，刀压在对方喉咙上：“说，是谁派你来气我的。”
<br/>　　李壮心里一激灵，冷汗唰唰的往外冒，怎么事情和自己想的不一样。他只能求饶：“大王饶命，大王饶命……我们都是穷人，身上什么都没有……”
<br/>　　“等一等。”
<br/>　　高飞虎下马，捏了捏李壮的脸，又扯开他身上的麻衣，露出里面丝绸质地的小衣。
<br/>　　“那家穷人长的这么细皮嫩肉，还穿得起丝绸。”
<br/>　　有一个喽啰过来，盯着李壮的脸道：“大当家，这好像是李家庄的财主李壮，弟兄们踩过他的盘子，这老小子墙高人多，咱们不敢动手。哈哈，今天怎么自己跑到山里来……”
<br/>　　“这定是我平时好事做得太多，神仙可怜我们，这就是福报啊。”高飞豹一声长叹，挥手道：“弟兄们，绑了，让他家里拿赎金。”
<br/>　　……
<br/>　　扭蛋机在程大雷脑海中出现，里面的扭蛋不停翻滚。
<br/>　　程大雷紧张得心脏都要跳出来，上次刷BUG委实做得有些过分，他一直担心系统故意报复，说不定抽奖直接抽个『谢谢惠顾』，也不知道霉运离开自己没有。系统我也不要什么，BK火炮，坦克，直升机随便来一样就好。
<br/>　　一颗扭蛋被弹出来，于此同时，程大雷收到一道系统提示。
<br/>　　嘟，恭喜获得初级山寨防御图纸一份。
<br/>　　咦，这次的奖励和以前倒是不一样。程大雷最后从土炕旁边的柜子里，将这份系统奖励的图纸找了出来。
<br/>　　山寨建设防御攻势，已经是势在必行的一件事。以前山寨一共五个人，什么都没有，别人都不稀罕抢你，可现在不同，山寨已经有五万多斤粮食，而且自己又置身在青牛山这土匪窝中，现在都不知道有多少人盯着青牛山流口水呢。
<br/>　　靠掌心雷忽悠，估计也忽悠不了多久，山贼中虽然聪明人不多，但毕竟还是有的。
<br/>　　必须在他们觉悟之前，把山寨防御建立起来，不然的话，自己就只是一个库管，那五万斤粮食只能是替别人看着的。
<br/>　　既然系统奖励自己一份图纸，自己就正好将山寨防御建设起来。可这又难免迎来第二个问题：山寨人口不足。
<br/>　　算上黄三元，此刻的山寨也不过六个人，想要靠这几个人把山寨防御设施建立起来，根本就是痴人说梦。
<br/>　　而自己身上正好有一个任务，就是招募人口，将山寨人口发展到100，系统会再次奖励一次抽奖机会。
<br/>　　可是，自己去哪里招募人呢？
<br/>　　想到这件事，程大雷有些发愁，人口或者劳动力成为限制山寨发展的一个要素。想着，程大雷打开了系统界面，查看山寨信息。
<br/>　　山寨：蛤蟆寨（破败的初级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：6
<br/>　　技能：齐心，治愈
<br/>　　夜深人静，程大雷独自思考着该如何发展山寨，山寨又该往哪个方向发展。
<br/>　　其实无论何时，山贼都是最不入流的存在，程大雷倒是很希望能抽奖抽到诸葛亮这样的谋士，张飞这样的谋士，先不说系统会不会抽到，即便是有，如他们这样等级的人，会安心扶保一个山贼么？
<br/>　　呵呵，要知道周仓拼了命往关二爷面前凑，最后把命都给了二爷，临死也不过是在二爷后面扛刀。
<br/>　　他程大雷可不想给谁扛刀。
<br/>　　而这个世界……哪怕仅凭程大雷对这个世界的一丁点了解，也知道乱世将至。这倒不是说程大雷有多聪明，但至少他肚子里也装着五千年历史，虽然不能说精通，但也能看得懂历史大势。如今的帝国，外有戎族虎视眈眈，内有群阀割据，大武这栋破屋子早已经千疮百孔，急需要推到重建。
<br/>　　秦末，汉末，隋末，唐末，宋末……那个乱世，不是死人，死人，死人，不停的死人。
<br/>　　等到乱世来临，群雄逐鹿天下，自己一个小小山贼，该靠什么立足呢。
<br/>　　他如今对这座山寨已经有属于家的归属感，秦蛮，林少羽，徐神机，徐灵儿……这种类似亲人一样的存在。他希望守住这座山寨，不让任何人来染指。
<br/>　　而留给他的时间，可能真的没有多少。
