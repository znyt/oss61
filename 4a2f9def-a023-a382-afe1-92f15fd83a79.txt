<br/>				    这一看，深邃的黑瞳不由的微闪，眼底划过一抹意外，他低声对护在怀中的凤九说着：“阿九，我们好像到了别人的宗门里面了。”<br/>　　<br/>　　    “啊？”凤九从他的怀中探出头来往下一看，这一看，也不由一呆：“好热闹啊！我们该不会撞到他们一年一度的比试上去了吧？”<br/>　　<br/>　　    只见下方，人山人海围聚而起，似乎不止一个宗门的人，因为，每一处地方的人都穿着同一色的衣袍，而在中间的高台上，那在高台上比试的两人也因他们突然的出现而呆了呆，停了下来。<br/>　　<br/>　　    此时，那下方的众人都有些傻眼，他们好几个宗门的人正参加着一年一度的大比，突然间，天空中一阵刺眼的光芒闪过，便见漩涡一现，一抹黑色的身影突然出现在视线之中。<br/>　　<br/>　　    然，当他们定睛看去时，却见一抹红色的身影从那黑色身影的怀里冒出个脑袋来，仔细一看，竟发现是一名黑袍男子怀中搂着一名红衣少年。<br/>　　<br/>　　    光天白日之下，两名男子搂搂抱抱的出现在众人的视线里，这一幕，几乎看傻了底下的众人，他们只看过男抱女，还没见过男抱男的呢！<br/>　　<br/>　　    “这两人是从哪里出来的？”<br/>　　<br/>　　    “这该不会是从那登仙梯上下来的吧？”<br/>　　<br/>　　    “从那登仙梯上下，又怎么会搂抱在一起？而且还是两男的？”<br/>　　<br/>　　    “难道那边大陆的人都好这一口？”<br/>　　<br/>　　    “哎，你们看，那黑袍男子长得很是……很是……怎么形容呢？”<br/>　　<br/>　　    旁边一名男子接过话，道：“这黑袍男子，一身气势凌厉摄人，周身散发着强大的威压与上位者的尊贵气息，容颜俊美而不失刚毅冷峻气息，别的不提，单单这容貌就已经胜出四大仙门十大天骄之首的东方千绝了，倒是没想到，在那边那样的地方，居然也能孕育出这般堪比神衹的人物来。”<br/>　　<br/>　　    而此时，高台上方坐着的那些宗门尊者们，看着那从上面突然出现的两人却是眸光微闪，尤其是其中的几个看到那从黑袍男子怀中探出脑海来的那张绝美容颜时，更是眼睛一亮。<br/>　　<br/>　　    是她！那个回他们看中的小丫头！<br/>　　<br/>　　    也在这档时，轩辕墨泽带着凤九稳稳的落到那高台上，站稳后搂着凤九腰间的手才松开，将她从怀里放了出来。<br/>　　<br/>　　    “怎么？还好吧？”他无视着周围的众人，而是低头询问着怀中的凤九。<br/>　　<br/>　　    “嗯，没事。”凤九说着，眨着一双眼睛看着周围的众人，同时走出了轩辕墨泽的怀里。<br/>　　<br/>　　    “真不好意思，我们好像打扰到你们了。”凤九歉意的说着，朝台上的两人拱了拱手。<br/>　　<br/>　　    而这时，那些人才看到，原来那个躲在那黑袍男子怀里的少年，竟是一个长得极为俊美动人心魄的少年！<br/>　　<br/>　　    看着少年一袭耀眼红衣着身，墨发轻束着披散在身后，一双清眸带笑，唇角微勾，露出邪魅肆意的笑容，举止更是洒脱随意，活脱脱的一个富贵人家小公子的模样，看得他们不由呆了呆。<br/>　　<br/>　　    这少年，生得当真好看！<br/>
