　　鸿钧道人成圣后，众修士纷纷前往混沌之中，寻找鸿钧道场紫霄宫，只是，他们又哪知道这混沌之中的凶险呢？
<br/>　　进入混沌，四周灰蒙蒙一片，没有天，没有地，没有光明，没有黑暗，一切生命迹象尽皆无，到处都是无穷无尽的混沌气流，凶险无比！
<br/>　　混沌之中暗含罡风，暗含地水火风雷，看似平静，实则暗藏杀机。稍微一不留神，即便是那大罗金仙，也要陨落在那混沌之中。没有强大的肉身或者护身至宝保护自己，贸然进入混沌之中，绝对是找死的行为。
<br/>　　元始三人进入这混沌之中，看到无数道修士都往这混沌之中赶，有的刚进混沌便化为飞灰，有的还能坚持一时三刻，有的有灵宝的护身，还能坚持住，抵御混沌的侵蚀。
<br/>　　在看到元始三人进入混沌后，周边的一些人笑道：
<br/>　　“看看，这三人能坚持几刻，我赌他们连一刻都不能坚持。哈哈哈！”
<br/>　　“他们看似不简单，我赌坚持两刻钟。”
<br/>　　“我赌三刻！”
<br/>　　“我赌一刻不到。”
<br/>　　“哈哈哈！”
<br/>　　……
<br/>　　元始看都不看他们一眼，这些人注定是不能进入混沌之人，只能在这里通过嘲笑别人来换取一点安慰，这种人简直可怜到可恨。他与他们简直是不在一个维度上，有什么好计较的。
<br/>　　“哥哥，你说他们能坚持多久？”这时，不远处一个极为端庄漂亮女子对着他旁边的兄长问道。
<br/>　　“我亦不知，我看他们不简单呐。”
<br/>　　“有什么不简单的，哼！”女子娇哼道。
<br/>　　元始三人进入混沌，元始的肉体早已达到混元境界，故而这混沌之气对他没有什么伤害，反而，只要是混沌之气一近他身，他便全部吸收到身体空间之中，用来扩增空间宽度，这混沌之气乃最为原始的能量，不能浪费啊。
<br/>　　而老子与通天就没有那么强大的肉体了，他们进入混沌之后，感觉到混沌之气对他们不断的侵蚀，抵抗了一阵子后，就纷纷祭出法宝来抵挡。
<br/>　　老子祭出天地玄黄玲珑宝塔，宝塔悬浮他头顶之上，玄黄二气弥散下来，形成一道玄黄瀑布，将他笼罩，万法不侵，混沌之气不能入内。
<br/>　　而通天则头顶十一品净世白莲莲台，净世之光下垂而来，将他四周的混沌之气境界抵挡在外，不得入内。
<br/>　　“啊！这都可以？”混沌外的众多修士尽皆傻眼，没想到会出现这种情况。
<br/>　　“没想到啊，看走眼了，他们竟然有这么好的灵宝！”
<br/>　　“我们去将他们抢了，如何？”有人动起了别样的心思。
<br/>　　“你傻啊，你没看到，旁边还有一人在吸收混沌之气啊，那可是混沌之气啊，他吸收了这么多竟然没事？你惹得起吗？”此时亦有人看到了旁边的元始，觉得这人恐怖至极，不可招惹。
<br/>　　“没想到他们这么强，看他们的样子，难道就是昆仑三清？”不远处，那漂亮女子与他哥哥看了元始三人，也凝重道。
<br/>　　另一边不远处，还有一个极为漂亮端庄，身穿一身黄色衣裙的女子与一个身穿黑色衣裙的女子站在一起，看着元始三人，其中黄色衣裙的女子道：
<br/>　　“不知为何，我总感觉他们三人与我们冥冥之中存在某种联系？”
<br/>　　“啊？你也有这种感觉啊，我也有。”这时，黑色衣裙的女子惊声道。
<br/>　　“大兄，我们该往何处走？”通天性格直爽，发现自己不知道紫霄宫方向，便对着元始问道。
<br/>　　元始闭目不语，右手五指在快速运转，指间黑白两色环绕，玄之又玄，却是元始在用《元始金章》推算紫霄宫的方向。
<br/>　　不到片刻时间，元始便睁开双眼，对着身边二人淡淡道一声：“跟我来。”
<br/>　　之后，他转头看了一眼黄色衣裙女子与黑色衣裙女子，就带着两人往混沌深处而去。元始也感觉到了自身与两人之间的联系，他猜测，她们恐怕就是那十二祖巫之中的后土与玄冥了，因为只有十二祖巫与自己三人联系极为密切，他们三人是盘古元神清气所化，而十二祖巫乃盘古肉身精血所化。
<br/>　　不过，就算知道他们是十二祖巫，元始亦没有多管，直接带着两人往混沌深处而去。
<br/>　　“吹牛吧，哥哥你的推算能力在洪荒已经算得上是数一数二了，现在就连哥哥你都没有推算出来，他们怎么可能现在就算出来？”一旁，女子看着三人离去的声音，不信道。
<br/>　　“我看他们不像是作假，须知天外有天，人外有人。”男子对着身边的女子凝重道，他这个妹妹，什么都好，就是攀比心太重了，真不知道以后会吃什么亏。
<br/>　　“妹子，他们走了，我们怎么办？”另一边，黑衣女子对着黄衣女子问道。
<br/>　　“我们跟上去。”
<br/>　　黄色衣裙女子毫不犹豫道。他想起了之前那个男子回头看向自己两人的那一眼，也许是在传递什么信息。
<br/>　　“好。”
<br/>　　……
<br/>　　之后，那女子与他哥哥也踏入混沌之中，男子取出一架古琴，他十指纷飞，弹出琴音，一个个音符玄之又玄，汇集在一起形成浪潮一般，使得两人没有受伤。
<br/>　　再之后，帝俊与太一两人头顶先天灵宝，玄之又玄，散布星辰之光，沐浴星海，朝着一个方向而去。
<br/>　　之后，一个气息缥缈，仙风道骨的道人，脚踏地书，手持拂尘，寻了一个方向前行，而在那道人身边，还有一个一身红袍，面带和善微笑的道人。
<br/>　　之后，又有无数人施展各种手段，前往混沌深处，寻找那紫霄宫。
<br/>　　众修士进混沌，就如同八仙过海，各显神通。
<br/>　　……
<br/>　　而就在洪荒众强者寻找紫霄宫的路上，此刻，混沌深处，有两个道人正在对峙着，如果元始在此，就一定会发现，这两人他都认识，一个就是之前成圣的鸿钧道人，另一个却是三族大战之时元始在虚空乱流之中遇到的扬眉老祖。
<br/>　　“鸿钧道友，贫道成就混元之时，天道将贫道轰了出来，你为天道的代言人，此番因果，看来是要找你了结了。”
<br/>　　杨眉老祖看着鸿钧道人，淡淡道。
<br/>　　鸿钧看着眼前的道人，心中纠结，这人是混沌魔神转劫而生，与天道不在一方，是天道要清理的变数，如今证得第三步混元大罗金仙，天道将之轰出洪荒。这番因果，看来是逃不了了，也罢，那就做过一场吧。
<br/>　　“贫道亦想与道友做过一场，以此了结因果，今后此事就此罢休，如何？”
<br/>　　“大善！”
