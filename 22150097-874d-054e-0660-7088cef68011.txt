　　“行，那咱们改天再战，这段时间辛苦你们了。”叶帝打字回复着，相较于他们这些打入世界赛的队伍来说，皇族这个没有进入到世界赛的队伍确实是要更辛苦一些。
<br/>　　看着叶帝回复的话，楚白衣几人只是苦笑。
<br/>　　“要是在来几场这样的训练赛，我感觉要有心理阴影了，弄不好以后在面对白队的时候就不战而败了。”白鸽一脸苦笑的说着。
<br/>　　“大家振作一点，今天这场比赛也是天时地利人和在里面的因素，并不会经常发生的，封神之战哪里又有那么容易，大家好好恢复一下，明天继续再战，怎么着也不能让他们老赢啊，我们可是皇族。”皇族的教练在一旁对着众人打气道。
<br/>　　“对，我们是皇族，怎么可能轻易认输，在赛场上没有赢过蓝白两队，在训练赛上咱们一定要杀杀他们的威风。”听着教练的话，几人的情绪果然好转了很多，狂战开口狠声说道。
<br/>　　“为了我们华夏区的荣誉，这段时间大家就在辛苦辛苦，他们的夺冠之路上，是有着我们的功勋章的。”教练继续开口说道。
<br/>　　“教练，叶哥问我们这个视频能不能传到网上？”看着屏幕上方叶帝再次发过来的消息，楚白衣开口问道。
<br/>　　“这个不是他们的大招么？怎么要发到网上？”听着楚白衣的话，苹果有些不相信的问道，还站起来特意向着楚白衣的电脑上看了两眼。
<br/>　　等到他看到叶帝确确实实发过来了一条这样的消息之后，苹果的脸上不解的神色更加的浓了起来。
<br/>　　“刚刚那场比赛打的太激烈了，我们都没有提前退出去，已经留下战绩了。”一旁的司马贼跟着开口说道。
<br/>　　“那到底是发还是不发？”飞鸽看着教练问道。
<br/>　　这个时候他们已经明白了叶帝的意思了，这一场比赛对方毕竟是踩着皇族来封神的，如果发出去的话，对他们皇族肯定会有一些不好的影响。
<br/>　　甚至可能每每说道封神战的时候，都要提上一嘴皇族，到时候他们注定是一个承当绿叶的命。
<br/>　　“我去和老板商量商量。”教练感觉这个事情他也做不了主，连忙向着外面走去，一边走一边拨打着战队老板的电话。
<br/>　　他虽然是教练，管理着自家的选手，但是这种关系到皇族的声誉的事情上，他还是做不了主的，虽然他内心里是比较赞成对方将这个视频发到网上的。
<br/>　　在看到叶帝发过来这条消息的一刹那，他的脑海里就已经思考了很多东西，他知道这个视频如果被发到网上的话，那么其他战队肯定会更加的针对香蕉了。
<br/>　　但是不要忘了，白队这个战队从来不是以香蕉为主体的战队，他们战队的真正核心从始至终其实只有一个，那就是叶帝。
<br/>　　赢下比赛的时候，可能mvp是其他几名选手，但是在逆风局的时候，叶帝永远是场上最稳健的一个，甚至连夜雪狂歌都不如他。
<br/>　　如果那些顶尖的战队真的要去针对香蕉的话，那么叶帝这个点就要彻底解放出来了，说不定到时候他们还会见识到一场如春季赛开赛那样，叶帝卡萨丁的封神之战。
<br/>　　能够在国际赛场上让华夏人扬名，即使不是自己的战队，但是作为一个华夏人，他也是非常乐于看到的。
<br/>　　拨通电话后，这边教练便是将叶帝说的话跟那边说了一遍，又将自己的分析讲了一下，然后便是静静的等待着对方的答复。
<br/>　　几乎在他说完之后，电话那边连犹豫都没有犹豫直接给出了明确的回复。
<br/>　　“让他们发，只要他们能够拿下冠军，我们皇族愿意承担这些东西。”
<br/>　　听着老板那斩钉截铁的回话，皇族的教练刹那间便是感觉新潮澎湃了起来，一脸兴奋的开口回应着。
<br/>　　他想过老板会答应，但是没有想到自家的老板会这样的痛快。
<br/>　　过了一会后，叶帝他们这边便是得到了对方的回复，可以发，皇族希望白队夺冠。
<br/>　　“他们这样一来，我怎么感觉压力好大啊。”看着这个回复，萝卜深深的吸了一口气，开口说道。
<br/>　　“皇族真的太够意思了，无偿陪我们打训练赛，然后现在又甘愿当绿叶，他们付出的真的是太多了。”茄子跟着说道。
<br/>　　“所以无论如何，这届世界赛我们都要努力到最后，拿下那块冠军的奖杯，要不然可就辜负太多人的期望了。”叶帝同样沉声说道。
<br/>　　随后他这边便是将刚刚录制下来的视频发布到了火神的视频区当中。
<br/>　　就在叶帝发出这个视频的一瞬间，无数关注着叶帝的粉丝们便是接到了有一个新视频的提醒，那些有时间的粉丝们自然是第一时间打开了这个视频。
<br/>　　当看到标题上写的荣耀白VS皇族，香蕉封神之战的时候，还没有看到视频的正式内容，这些粉丝们便是已经激动了起来。
<br/>　　叶帝这边在将视频发出去之后，几人便是走出了训练室，打赢了这样的一场比赛后，他们现在也是要平复一下心情，没有继续打排位赛。
<br/>　　“怎么这么早就出来了？”看着叶帝几人出来，另一边还在客厅内看着其他战队比赛视频的蒋念几人有些诧异的问道。
<br/>　　“心态打崩了，他们不敢打了。”香蕉依旧是一如既往的在那里吹着牛逼，当然了他确实是很牛逼。
<br/>　　“你别说话。”飞刀压根就不信他说道话。
<br/>　　“真打崩了，视频都已经发到视频区了，你们有兴趣的话可以过去看看，香蕉的封神之战。”看着几人将目光放到自己这里，叶帝笑着点头说着。
<br/>　　“视频都发了？”秦霄有些讶异。
<br/>　　“打的太激烈了，刚刚没有机会提前退游戏，战绩都保留了下来，所以就发了。”叶帝点头。
<br/>　　“我去，是不是真的啊，景禹快点搜索一下，正好咱们看看。”蒋念对着前面的李景禹喊着。
<br/>　　他们这里的电视都是连接着电脑的，所以是可以从这里看到电脑上搜索出来的东西的。
<br/>　　听着蒋念的话，李景禹也是没有犹豫，直接搜索到了叶帝的id，随后点开了他刚刚发布的那个视频。
