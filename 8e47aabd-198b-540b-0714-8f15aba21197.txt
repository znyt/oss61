　　高漫秋回首望了一眼，脚步不停朝着出口大步而行，风扬起她的发丝，遮挡了双眸一闪而过的风情与坚定。
<br/>　　十二星辰，星月之轮。
<br/>　　很巧，她听说过这八个字，在高家的一本残卷上。
<br/>　　在阳谷的入口地带，从阴谷出来的高家八十人将楚云天围成了一个圈，但却小心的保留了一段安全距离，力保不会出现任何突发·情况和失误。
<br/>　　同时在楚云天的周围，还被放置了部分食物和水，只是他并没有碰这些食物哪怕一下，明眼人能都能出来这个老头已经长期未进食，而他身边的那些蚊虫残渣，更显得有些凄凉。
<br/>　　“怎么样，还是不肯说？”一名高家的精英成员小声开口。
<br/>　　旁边的人摇头回答：“一声不吭，我都怀疑他不是聋哑人。”
<br/>　　那名高家精英成员又仔细观察了楚云天一眼，紧接着目光集中到地上的那些昆虫残渣物上，道：“这个老头不简单，他没有能力去捕捉其他动物，这些天应该就是靠着这些蚊虫维持生命。看到没有？这些虫类并不是被一口吞下，而是被剔除了无用或有毒部位，他只吃了这些虫类身上有营养的那部分。”
<br/>　　听着这名高家精英成员的解释，周围一圈的高家人都不禁对楚云天刮目相看，在饥肠辘辘的情况下还能有这般理智，甚至有条不紊的安排生命中最后一程的时间，简直就像是一尊活体机械。
<br/>　　要知道如果不是他们恰好出现在这里，如果没有地上的这些食物，眼前这个老头要不了一两天就会被活活饿死。
<br/>　　但哪怕这样，他还是闭着眼睛一动不动，对四周散发着诱人香味的食物看都不看一眼。
<br/>　　这需要多高的自制力？
<br/>　　多强的定力？
<br/>　　很多人都心生感叹，颇为佩服。
<br/>　　“吃吧老头，没毒。”一名高家精英成员看不下去，开口劝道。
<br/>　　“我们不是坏人。”另一人也开口，紧接着又觉得哪里不对，补充道：“当然也不算什么好人，不过我们不杀人，从不。”
<br/>　　这是一句至理名言，神秘家族从不杀人，因为根本被制裁一门压制的死死的，不敢对人类世界出手。
<br/>　　楚云天依然对这些人置之不理，只是脑海中已经在逐步分析起来，将所见所闻以及眼前这些人的话整合，理出一个清晰脉络。
<br/>　　能生出楚涵那样堪称分析帝的儿子，楚云天的推理能力自然不在话下。
<br/>　　看到劝说无用，高家的这八十人也不再浪费口舌，但对这个令人印象深刻的老头更加尊重，虽然他们真的很想知道，这个毫无战力的普通人，究竟是如何来到这个地方的。
<br/>　　这时候一道微风拂过，高漫秋穿着一席复古繁华的裙装出现，作为神女她从阴谷走出至平台，用不了一息时间。
<br/>　　“大小姐。”
<br/>　　高家人纷纷行礼，他们并没有叫高漫秋神女大人，而是循着过去的习惯叫她大小姐，但是对高漫秋的尊敬却只增不减。
<br/>　　高漫秋点头，目光自然而然的看向旁边的楚云天，当下的场面万分古怪，高家的这批人可谓是精英中的精英，就这么将一个手无缚鸡之力的老人围住，很是耐人寻味。
<br/>　　只是高漫秋并没有问什么，只看了一眼便快速的将情况分析的差不多，无论是楚云天身边的蚊虫残渣，还是那堆高家人留下的食物，都给了高漫秋很好的线索，但她也并未逗留，直径走向了阳谷入口。
<br/>　　高家的精英成员快步跟上，不再将注意力放在楚云天身上，他们有更重要的事要做。
<br/>　　巨大的石台缓缓下潜，很快就将这批人带到了地底的巨大墓园。
<br/>　　这时候的楚云天才动了动已经发酸的身躯，思索了一会儿后，拿起眼前的食物和水小口的进食。
<br/>　　他并没有狼吞虎咽，而是循序渐进的将自己的胃填至七分饱，然后便立即拿起纸笔，将刚才那群人的特征都画了下来，尤其是他们口中的那位‘大小姐’。
<br/>　　他记住了这群人的样子，更将高漫秋的容貌深深印在脑中。
<br/>　　所有的一切都让楚云天弩定，这是又一批的神秘家族！
<br/>　　依靠着高家人留下的食物和水，楚云天又在这里停留了一天，这段时间内此处分外的安静，什么都没发生，更没有任何狂化动物靠近。
<br/>　　那些凶猛的狂化动物似乎对此处颇为忌惮，最近的一次也只敢靠近楚云天十米的地方便不再上前，这也是楚云天在此处辛苦活下来的关键。
<br/>　　只是在一天后，楚云天忽然起身动了动筋骨，把所有记录的东西藏在灌木丛的土地里，并在上方留下一个微不可查的标记，然后他将地上剩下的食物和水背起，一步步走向了那个石台。
<br/>　　他弩定，近期已经等不到人了，接下来的就需要进去，去那些人口中所说的墓园里看看。

<br/>　　远在东部海域的某个孤岛上，炎炎烈日从上方照下，刺目的光线打在楚涵的脸上，让他下意识抬手挡住眼睛。
<br/>　　双目睁开一条缝，映入眼帘的是一大·片蓝天白云，耳边有‘哗哗’的海浪声，旁边是令人心旷神怡的绿荫。
<br/>　　天气很好，好的让人好想再次闭眼，好好睡一觉。
<br/>　　只是楚涵的口鼻中传来一股臭汗味，让他一下子清醒！
<br/>　　刷！

<br/>　　楚涵难得有一瞬间的愣神，恍惚间不太记得清醒时的情况，只知道自己好像睡了很久，并且现在饥肠辘辘。
<br/>　　这时候的上空一片阴影一闪而过，楚涵抬眼望去，恰好看到天照3000从他头领飞过去，虽然离得近，声音却很小。
<br/>　　天照3000内，旺财趴在窗边惊吓的一声大叫：“握草别降落，他醒了！”
<br/>　　高少辉立马一个拉直将天照3000的高度强行抬高，同时吓的一背的冷汗：“我靠这么快就醒了，他伤势可不轻，高烧了一周昨天还处于无意识状态呢，今天怎么就能坐起来了？”
<br/>　　旺财关心的则是其他事：“他脑子有没有烧坏？还记不记得之前的事？算了反正不是我动的手，要倒霉也是你。”
