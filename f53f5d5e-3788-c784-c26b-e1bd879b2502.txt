　　林峰掉了一个团，足足一千多人，全副武装的就朝新华图书馆奔去。
<br/>　　新华图书馆是个很大的图书馆，足足有上下两层，之前夏天的时候有许多人来这里乘凉，对面就是肯德基必胜客之类的快餐店，非常方便，很适合泡在这里。
<br/>　　“首长，发现可疑目标。”
<br/>　　一个侦察兵带着望远镜回来报道。
<br/>　　“多少人。”林峰问道。
<br/>　　“大概二十人上下！”
<br/>　　侦察兵说道。
<br/>　　“二十人？那怕个毛，直接给我强攻，以摧枯拉朽之势，给他们突突了。”
<br/>　　林峰直接大手一挥就是下令。
<br/>　　“是！”
<br/>　　瞬间一个团的兵力就开始行动起来，一对对人马就是开始狂攻，催泪弹闪光灯各种手榴弹往那个图书馆内一丢。
<br/>　　然后一群人拿着步枪就冲了进去，很快就响起了密集的枪声。
<br/>　　林峰就在外面等着，一千人的规模队伍，不可能拿不下区区二十人，就算这二十人都是异能者，那也不可能。
<br/>　　现阶段的黑铁战士还是太弱了，根本不可能抵抗得了人类的武器枪械。
<br/>　　没多久。
<br/>　　枪声就消停了。
<br/>　　林峰知道战斗已经结束了。他就进入这新华图书馆内，发现这里的确是个据点，里面有不少资料，林峰在一个黑板上居然还看到了自己的照片。
<br/>　　“妈的，谁偷拍我的，拍的这么丑。”
<br/>　　林峰无语了，拍也不拍个帅点的照片，在这个照片旁边还写着几个字。
<br/>　　必杀。
<br/>　　除了林峰之外还有其余人的照片，都是些林峰不认识的，很显然他们也都是战门必杀的人物。
<br/>　　“首长，外面出现了大概一百多只丧尸！应该是被枪声吸引过来的！！！”
<br/>　　“歼灭！”
<br/>　　“是！”
<br/>　　林峰大手一挥顿时一群人开始行动起来。
<br/>　　哒哒哒哒哒。
<br/>　　一只只丧尸在密集子弹下直接倒下。
<br/>　　“这些丧尸晶石就归你们了。”
<br/>　　林峰摆了摆手，这区区百来颗晶石他还真的不怎么在意。
<br/>　　“多谢首长！”
<br/>　　林峰不在意，一群基层士兵可在意极了，别看林峰现在都九阶了，要不是因为天上地下唯我独尊八荒六合第三重死活捡不到，把他给卡主了，说不定他现在都已经十阶了，毕竟林峰物品栏里可还有两个九阶的晶石。
<br/>　　实际上，基地内大部分人都还是三阶上下，一些排长也就这个实力，连长可能要强一些有个五阶上下，陆勇这个基地大佬之一的存在也就是个七阶，还是最近才从六阶突破到七阶的。
<br/>　　林峰能够突破的这么快，也无异于他捡空投得来的道具实在是太犀利了，神乎其神的手段，各式各样的手段，坦克大炮无人机，飞镖符箓大宝剑，外加僵尸妲己孙悟空，简直就是所向睥睨。
<br/>　　据点清缴，林峰带着这群人回到了基地。
<br/>　　“陆将军，你的人我可都给你一根头发丝没少的全部带回来了。可到时候别找我碰瓷啊。”
<br/>　　“你小子。”
<br/>　　陆勇哭笑不得。他堂堂基地首脑之一，找一个十七岁的少年碰瓷？？
<br/>　　“行了，没事的话我走了。”
<br/>　　林峰目光落在了周华身上。对方在做引体向上。
<br/>　　“五百八十七……五百八十八……”
<br/>　　林峰忍不住开口道：“九阶到十阶，需要的是内力，你每天做这些肌肉训练，是不可能突破十阶的。你做十万个都不可能突破。”
<br/>　　周华愣了一下，旋即无奈的摇头：“我知道，不过内力哪有那么好修炼出来，尤其是我这种摸着石头过河的。我只能不断的锤炼肉体的力量，以此来刺激内力的生成。”
<br/>　　林峰暗道也是，要想修炼内力，那不是吃饭拉屎就可以修炼出来的，要有功法，没功法不可能修炼出来的。
<br/>　　林峰暗道自己倒也有个天上地下唯我独尊八荒六合，不过这个功法肯定能传授给别人的。
<br/>　　“这个林峰小兄弟，怎么忽然说这些了，他怎么知道九阶到十阶，要想突破必须要内力过关。这件事情还是你上次告诉我我才知道。”陆勇看着林峰离去的背影好奇道。
<br/>　　周华摇了摇头，“不知道。”
<br/>　　他的目光有些闪烁，按道理，知道这些的只有九阶的强者了，因为突破九阶的人都会有一种，距离十阶遥遥无期的干净。
<br/>　　林峰一个八阶的没道理知道，除非对方突破了九阶。可是林峰突破九阶的事情，除了林峰自己人知道，其余人根本不知道。而且周华也不愿意去相信林峰这么快时间内就突破九阶。
<br/>　　与此同时。
<br/>　　巨树之森。
<br/>　　战门总部。
<br/>　　“该死！最新消息，我们新书图书馆的据点被曙光基地的那群杂碎给摧毁了。”
<br/>　　“妈的，这群杂碎，上一次，没坑杀他们的先遣部队，这一次反倒是又被他们摧毁了一个据点。眼下我们在基地内的据点没了，在外面的据点也被摧毁了一个。”
<br/>　　“该死，这个陆勇还真是小瞧他了。”
<br/>　　此时此刻。
<br/>　　一个大殿内。
<br/>　　坐着几个老者。
<br/>　　他们都是外门长老。绝对的有话语权。
<br/>　　“事到如今，你们居然还可笑的认为这一切都是陆勇做的，陆勇他虽然有点本事，但还不足以做到这一切。”
<br/>　　一个人从黑暗之中走了出来，这个人穿着一个黑色的大衣，浑身就好像有一层雾气一样，看不清他的本来摸样。
<br/>　　但是他的出现，让几个高高在上的战门外门长老都是情不自禁的站起身。
<br/>　　“你怎么来了。”
<br/>　　几人都有些忌惮的看着这个黑衣人。
<br/>　　一位属于第一邪教的巡逻夜叉。
<br/>　　在第一邪教内，没有诸如战门这样的长老，也没有诸如战士会的干事
<br/>　　普通成员就叫做小鬼。队长就是牛头马面，干部就是阎王罗刹。高级干部就是夜游神。
<br/>　　至于邪教的教主就叫做冥神或者是教主。
<br/>　　眼前的这个，就是第一邪教位于江城的阎王。
<br/>　　眼前这几个外门长老虽然都是九阶的，但是看到眼前此人也不得不感到敬畏。
<br/>　　“来这里只是给你们一些情报罢了，免得你们一直被别人牵着鼻子走，这一次，灭掉你们一个据点的的确是曙光基地的，但是带队的人，甚至可以说发起人，是这个叫做林峰的小子。”
<br/>　　“林峰？？”
<br/>　　几个战门外门长老都是拿着林峰的照片。
<br/>　　“这小子，怎么看起来这么猥琐。”一个长老看到照片之中的林峰正一脸猥琐的笑容。皱了下眉头。这要是让林峰知道了定会无语，那个狗仔偷拍的，居然也不给他拍的帅一点的。
