　　要知道，哪怕只是最低级的一品丹药，想要将其炼制到完美级别，也绝对不是一件简单的事情。
<br/>　　即便是很多资深的高品炼丹师，也很难做到一点。
<br/>　　因为完美品级的丹药，并不是光靠炼丹技术就能炼制出来的，还需要一些运气成分，丹药之中暗含一丝天道之力，才算得上是完美无瑕。
<br/>　　而但凡完美之物，天必妒之！
<br/>　　就好比修真者，需要渡劫一样，一旦完美级别的丹药问世，便会引动天地之力，从而出现天地异象。
<br/>　　当然了，因为叶尘炼制的，仅仅只是一品丹药，所以引发的天地异象很弱，只是发出了一道声响，这才让台下的众人，误以为是炸炉了。
<br/>　　不过，众人也很快反应过来了，发现叶尘手中的丹炉完好无损，而且一枚透着淡淡玄光的丹药，自丹炉之中，缓缓悬浮起来。
<br/>　　“这……这是什么丹药？怎么还会发光啊？”
<br/>　　“这他娘的是洗髓丹？看上去好厉害的样子！”
<br/>　　“一群白痴！那是丹芒，只有高品灵丹，或者完美品阶的丹药，才会出现的丹芒！”
<br/>　　“什么！你的意思是说，这小子竟然炼制出了一枚完美品阶的丹药？这怎么可能！！”
<br/>　　……
<br/>　　人群之中，还是有不少有见识的人，很快便认出了，叶尘炼制的这枚洗髓丹，竟然是完美品阶的丹药！
<br/>　　一时间，整个大殿之中，顿时炸开了锅。
<br/>　　就连站在最前端的屠苏公子和蝉衣小姐，两人也全都露出浓浓的震惊之色，
<br/>　　“完美品阶的一品丹药！这怎么可能？”
<br/>　　屠苏公子不由得失声道。
<br/>　　要知道，即便是他，自从开始修炼炼丹术开始，都还从来没有炼出过一次完美品阶的丹药。
<br/>　　因为这对于炼丹师来说，是可遇而不可求的事情，即便是比他还要厉害的炼丹大师，也不敢保证一定可以炼制出完美品阶的丹药。
<br/>　　可是现在，这件事情，居然被眼前这个，曾经被他无比鄙视的废物做到了！
<br/>　　这让他如何能接受？
<br/>　　然而事实摆在眼前，却又由不得他不信，
<br/>　　“他一定是走了狗屎运，所以才碰巧炼制出了一枚完美品阶的丹药，否则以他这么烂的天赋，怎么可能具有那样的实力？”
<br/>　　屠苏公子不停的摇着头，不停的低声自我安慰着。
<br/>　　不料，一旁的蝉衣小姐，忽然插口道：
<br/>　　“屠苏师兄又何必自欺欺人呢？能够炼制出完美品阶的丹药，必然是对炼丹之道，有着深刻的了解，甚至已经触摸到了一丝天道！”
<br/>　　“即便此人是碰巧炼制出完美丹药，本身的炼丹功底，肯定也是不错的！”
<br/>　　屠苏公子听到这话，脸色顿时微微一变，旋即冷冷笑道：
<br/>　　“他若真如师妹你所说的那么厉害，那为何这么大年纪，才来参加一品炼丹师的考核？这你又作何解释？”
<br/>　　蝉衣小姐秀眉微微一蹙，摇了摇头道：
<br/>　　“这个我也想不明白，不过这高人行事，自然有独到之处，我们想不通也很正常！”
<br/>　　屠苏公子冷哼一声，道：
<br/>　　“就算是完美品阶的丹药又如何？一品丹药终究只是一品丹药，即便是再完美，也比不上一枚普通的二品丹药！”
<br/>　　蝉衣小姐张了张嘴，最终没有多说什么。
<br/>　　的确，叶尘虽然炼制出了一枚完美级的丹药，但毕竟只有一品，尽管造成了不小的轰动，那主要是因为，太出乎众人的意料之外了，但实际意义却十分有限。
<br/>　　纵然是一枚完美品阶的洗髓丹，依旧只是一枚洗髓丹，也就比普通的洗髓丹，效果要好一些，真实的用处并不大。
<br/>　　而考核大台上，丹师盟会的几位长老和那位云丹宗的火龙长老，同样也是惊愕过后，便并没有太放在心上，显然和屠苏公子一样的想法，也都觉得他，只是一时的运气好，所以才能炼制出完美级的一品丹药。
<br/>　　另外一边，
<br/>　　叶尘领取了丹师宝卷之后，在万众瞩目之中，缓缓走下了考核大台，随便找到一个没人的位置站定。
<br/>　　众人看到这一幕，顿时再次纷纷议论起来，
<br/>　　“看样子，他这是要继续参加考核啊！”
<br/>　　“也对，此人能够炼制出完美级的丹药，想来炼丹水平应该不低，看来之前他是扮猪吃虎！”
<br/>　　“不过，这二品炼丹师的考核，可要比一品炼丹师困难多了，他之前如果只是走了狗屎运，接下来，运气可就未必有那么好了！”
<br/>　　……
<br/>　　众人冲着叶尘指指点点，有人看好，有人却觉得他只是一时运气。
<br/>　　很快，
<br/>　　一品炼丹师的考核结束了。
<br/>　　随着丹师盟会的长老，宣布二品炼丹师考核开始，叶尘随着众人，再次登上考核大台。
<br/>　　参加二品炼丹师考核的人数，比之前上一场，明显要少了许多，仅仅只有一百来人。
<br/>　　当然了，考核的内容，自然也比第一场要困难了许多。
<br/>　　“你们眼前的这个箱子里面，一共是六十九种二品丹药的药方，你们每人可以五次抽取的机会，然后从五个药方之中，挑选其中一个，来作为你们考核的试题……”
<br/>　　随着丹师盟会的长老，将考核规则向众人讲解清楚之后，考核便正式开始了。
<br/>　　相比于第一场，这第二场的考核通过率，明显要低了很多，简直只能用惨不忍睹来形容。
<br/>　　前十人的考核，仅仅只有三人通过，其余七人全部遭到了无情淘汰。
<br/>　　通过的人，自然无比欢喜，而被淘汰者，则个个垂头丧气。
<br/>　　很快，
<br/>　　终于轮到叶尘，
<br/>　　这一刻，叶尘能无比清晰的感受到，所有人的目光，全都齐刷刷的朝他望来，就连台上那几位考核官，也全都露出一抹期待之色。
<br/>　　很显然，
<br/>　　由于第一场造成的轰动效应，叶尘已经成为众人密切关注的对象。
<br/>　　叶尘气定神闲的来的桌前，随手从箱子里，取出了一张金色玉简，只见上面刻着三个字：
<br/>　　“筑基丹！”
<br/>　　轰！
<br/>　　等到一旁的工作人员宣读出结果，台下众人顿时纷纷躁动起来。
<br/>　　当然，众人之所以这么激动，并不是因为这个题目有多难，恰恰相反，这恐怕是二品炼丹师考核中，最简单的一个题目了。
<br/>　　因为筑基丹，在二品丹药中，本就属于中等偏下级别，而且又是炼丹师最常炼制的丹药，可以说毫无难度可言。
<br/>　　“看来这家伙，今天还真是走了狗屎运！”
