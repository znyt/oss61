　　第735章
<br/>　　路在脚下。
<br/>　　如果可以的话，程大雷很想一巴掌抽过去，看对方能不能正常说话。所谓高僧么，总喜欢玄之又玄，云里雾里。但程大雷不是这样的人，同一个山贼打禅机也是一件很扯的事情。但现在走投无路，寄人篱下，程大雷也只能压抑住冲动。
<br/>　　程大雷深吸一口气：“请法师解惑。”
<br/>　　了然法师扶须微笑：“清风寺建寺已有百年，算起来比帝国的历史都长。诸多年来，都是我崔家在经营。寺中有一条密道，直通城外。”
<br/>　　程大雷看向崔白玉道：“你们崔家这么喜欢打地洞么？”
<br/>　　崔白玉有些无语，了然法师也明显愣了愣，到现在他还摸不准程大雷脾气。
<br/>　　“程当家，此事宜早不宜迟，还须早早决断。”
<br/>　　程大雷皱眉想了想：“能走人的话，我的座骑能不能过？”
<br/>　　了然难以置信的看着程大雷：“修一条暗道费时费力，又不可引得外人注意，是一件极不容易的事情。从清风寺到城外有三里地，修造时耗费了许多功夫，只能容一人弯腰前行，马匹是万万不能过的。”
<br/>　　程大雷叹了一声：“我的牛怎么办，总不能折在这里，那畜生落在魏牧冲手中，定会被他做成红烧牛肉。”
<br/>　　崔白玉开口：“现在也不是在意一头畜牲的时候。”
<br/>　　了然法师看着程大雷，心道这人恶贯满盈，人都不知杀了多少。怎么关键时刻，却在意一头畜牲的性命。如此因小失大，岂是做大事的人。
<br/>　　程大雷沉吟片刻，道：“你们几个走地道出去，我再想法子，咱们约好地方，在城外汇合。”
<br/>　　“大当家，你这样做太冒险了吧？”徐神机。
<br/>　　“好了，就这样说定了。”程大雷摆摆手：“你们先走，我留在汝南城，我一人一骑，想要脱身也容易些。”
<br/>　　面对汝南城的虎狼之兵，程大雷一个人面对的确有几分凶险。虽然都有些担心，但程大雷主意已决，大家也没有再说什么。何况，留下来也必然是成了他的累赘。
<br/>　　了然法师笑笑：“程当家稍安勿躁，如果想要走也不急在这一时半刻。凉那魏牧冲也未必会着急攻过来，大家想必也累累，权且休息一夜，再做打算不迟。”
<br/>　　程大雷点点头，这一夜拼杀对体力是极大的消耗。程大雷现在勉强和了然和尚说话，也无非是强撑着一口气。
<br/>　　了然和尚安排客房让几人休息，至少睡一觉，恢复一些气力。当几人都离开后，偏偏崔白玉留下了。
<br/>　　她郑重其事的行了一礼：
<br/>　　“叔父。”
<br/>　　了然法师表情也变得郑重，问道：“以你观之，程大雷此人如何？”
<br/>　　崔白玉道：“我与其相处也有些时日，却仍看不懂他的为人。”
<br/>　　了然和尚皱起眉头，崔白玉的才能他也知道，若说她都看不穿一个人，事情的确变得复杂起来。
<br/>　　“未来逐鹿，他有几分取天下的可能？”
<br/>　　崔白玉皱眉思索片刻，道：“他的为人我看不穿，总有奇怪不可解之处。但能短短时间折腾出一番天地，身上必有不凡之处。但，他似乎没有取天下的心，所作所为，明显在故步自封。”
<br/>　　“既然有争天下的力气，却为何不去争呢？”了然和尚呐呐叹了一声，表情明显有些困惑。
<br/>　　他沉默思索，道：“今天我与他见了面，仅仅一面之缘，心中也难以判断。不过，他或许是我们崔家的机会。”
<br/>　　了然和尚自顾自的说下去：“遭遇这些事，是崔家树大招风，也是崔家往日造的孽。无论如何，崔家这棵大树已经倒了。但你或许是崔家的一段新枝，新枝吐芽，日后未必不能长成一棵参天大树。”
<br/>　　崔白玉咀嚼着叔父的话，渐渐悟透了他话中的深意：借程大雷之力，东山再起，让崔家仍旧是崔家。
<br/>　　“此人性子很硬，他若没有争霸的心，怕我也未必能做什么。况且，世道纷乱，日后谁能问鼎天下，如今谁也说不好，未必便是程大雷。”
<br/>　　了然和尚摇摇头：“崔家并非一直是崔家，崔家先祖当年也不过是个铡草的马夫。但崔家可以撑三百年不败，总是有他的原因。这场难，是以前我们从未遇到过的，但哪有如何，崔家可以三百年不倒，但倒了也要随时能站起来，如此，崔家才是崔家。”
<br/>　　“可惜你是个女儿身，很多事没办法做。可你未必不能借助这一点，你的美貌天下无双，完全可以扎根在程大雷这棵大树上，借树开花。”
<br/>　　在谈论这些事的时候，了然和尚神情冷漠，偶尔眼神里的炙热也不是因为亲情，而是因为野心。
<br/>　　一个老和尚，黄土埋到脖子，但不是青灯古卷，而是踮着脚尖张望着天下大势。随时准备争霸逐鹿，让崔家东山再起。
<br/>　　如果野心这种东西永远流动在崔家的血脉中，那崔家能够传承三百年不倒的确是有原因的。
<br/>　　比如程大雷，就很缺乏这种东西。了然和尚话中的意思，便是让崔白玉如藤蔓一般缠绕在蛤蟆寨这棵大树上，用蛤蟆寨的养分开他崔家的花。
<br/>　　崔白玉表情冷漠，眼中却有某些东西被点燃。她自从陷在蛤蟆寨之后，日子便是浑浑噩噩，数次想要挣脱程大雷的束缚，却是有心无力。
<br/>　　但了然和尚的话却为她指明另外一条路。陷在程大雷手中也未必不能做事。崔家已经坍塌，自己身后没有靠山，那就自己长成一棵大树，自己做自己的靠山。
<br/>　　程大雷这个时候已经酣然入睡，在清风寺的客房中打着呼噜。他并不知道，此刻主持静室中正在谈论他。
<br/>　　次日天还没亮，几人已经准备妥当。了然和尚年岁已老，便是走几步的力气也无，难得这个年纪还野心如火。
<br/>　　这次带路的是香火僧慧禅，他带着大家来到清风寺后院，移开一口枯井的井口，既然沿着梯子下井。
<br/>　　程大雷担心路上发生什么意外，决定送他们一程，于是也下了井，打算将他们送出城。
<br/>(本章完)
