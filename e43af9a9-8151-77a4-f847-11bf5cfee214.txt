　　第543章
<br/>　　“我没钱，有钱也不给他们花。”程大雷道：“第三种？”
<br/>　　“第三种就是靠才华。”刘发财道：“你是天下第一诗人，天下第一棋手，哪怕是天下第一木匠，铁匠都能见到孟夫子。”
<br/>　　“这样啊。”程大雷叹了口气：“关键是这三样东西我一样都没有啊，天下第一山贼的名头，好像在这里不管什么用。”
<br/>　　刘发财叹了口气：“怪我没有提前说，我看我们还是找个地方住下去，从长计议。那红院就不错。”
<br/>　　“其实我倒是有个法子。”程大雷道：“这孟夫子总有亲人，我将他们的女儿孙女绑了，以此要挟，不愁得不到消息。依你刚才所说，万卷书院应该很有钱，趁机还能捞一笔。”
<br/>　　“……”刘发财一阵无语：“程当家，琅琊城是个有文化的地方。”
<br/>　　“是呀，你说的我知道，但我不是文化人呐。”程大雷用手摁住剑柄，胸膛挺起，看架势要闯万卷书院的山门。
<br/>　　“程当家，慎重，慎重。”刘发财忙拦住：“咱们再想想办法，总是有办法上山的。”
<br/>　　程大雷却也没有硬闯上山的心思，这次毕竟是求人办事，那能一上来就动刀动枪的。当然，他没有太多时间在这里浪费，如果一时半会想不到好办法，用孟夫子亲人的性命要挟他，也不是不能做的么。
<br/>　　反正我是个山贼。
<br/>　　正这时候，一麻衣书生从程大雷身边擦肩而过，踩着石阶上山。
<br/>　　“这不是也可以上去么。”程大雷拍拍刘发财的肩膀：“走上去看看。”
<br/>　　刘发财撇起嘴，嘴里嘀咕了一声：“那有那么容易啊。”
<br/>　　沿着石阶上山，程大雷看到山道直通封顶，眼观白云绕山峰，耳边听涛声阵阵。程大雷缓步上山，却也是个修身养性的好说在。
<br/>　　山道两侧，开辟出一方方石台，或有读书人坐而论道，或三五好友吟诗作赋，或安坐对弈，或在一起摆弄琴箫，或有人长袍临风，比武教技。
<br/>　　从来文无第一，武无第二，但也正因为没个第一，所以读书人才好争个第一。士子大多目高于顶并不是没有这方面原因。毕竟，武者间你要是不服我可以打得你服，但读书人之间我就是不服你你怎么办，你又不能揍我。
<br/>　　此时，程大雷来到半山腰，见眼前是一片平台。上山的路拦着一道红漆门，刚才超过他们的麻衣少年此刻跪在大门前。
<br/>　　一宽袍大袖的儒生立在大门前，道：“书院对弟子有严格要求，你若想入书院读书，就该在每年春三月参加书院的大考。”
<br/>　　“弟子已等了三年，只想书院能与我一个参加大考的机会。”麻衣少年道。
<br/>　　宽袍儒生摇摇头，将门掩上，上山去了。
<br/>　　程大雷一头雾水：“这是怎么会事？”
<br/>　　“程当家有所不知，想书院在帝国什么地位，从书院出来就能成为拉拢的对象。帝国读书人是多的，这些人肯定都想在书院读书，书院也不能什么人都收，总要考核从中选择优秀者。”
<br/>　　“言之有理。”程大雷点点头。
<br/>　　“所以书院有三收三不收。”刘发财。
<br/>　　“怎么讲？”
<br/>　　“每年春三月开大考，成绩优异者录取。此外，有名人推荐者收，书院子弟收，本人有出奇才学者收。”
<br/>　　“也不是没有半点道理了。”程大雷道：“那三不收呢？”
<br/>　　“没钱的不收，书院的花销是大的，出门在外也讲究行头，食不厌精脍不厌细么。毕竟，你连一套好行头都置办不起，不是给书院丢人么。第二，贱民子弟不收，书院子弟的父母总不能是什么奴隶劳役。第三，五体不全者不收。缺胳膊少腿的，面目丑陋的，也是给书院丢脸。”
<br/>　　“啧，蛮好蛮好。”程大雷这才想起来，刚才麻衣少年时走路一瘸一拐，当是占了五体不全，而看他一身麻布短衣，该也是没什么钱。
<br/>　　那麻衣少年跪在地上，求学之心十分诚挚。身前人来人往，也没人往他身上多看一眼。这种事在书院应该是经常遇见，不会有人觉得奇怪。
<br/>　　过不多久，来了两个短衣打扮的人，看他们模样应该是学院的仆人，也就是……校工。
<br/>　　二人架起麻衣少年就往山下走，举止不算粗鲁，但也绝不客气。该也不是第一次处理这种事。
<br/>　　程大雷拦下了麻衣少年，将两位校工驱散。
<br/>　　“男儿膝下有万金，这见谁就下跪的毛病，该不是圣人教诲？”程大雷道。
<br/>　　少年名叫楚云生，他垂着头，有气无力道：“为学当不耻下问。”
<br/>　　虽然很是垂头丧气，但看表情，也不太瞧得起程大雷和刘发财二人。得，还没上山呢，便染上了书院的臭毛病。
<br/>　　程大雷也不以为意，道：“想上山么？”
<br/>　　楚云生眼里这才亮了亮，随之摇摇头：“我已在琅琊城住了三年，家财耗尽，都没通过这道门，天下已没人能帮我。”
<br/>　　“啧，瞧你这话说得，只要你想上山，我就能帮你打开这道门。”
<br/>　　楚云生呵呵笑了一声，没有当场讥讽程大雷，也算他有些涵养。
<br/>　　刘发财也一头雾水，自己二人上山还没找到路呢，程大雷又为何敢夸口帮别人。
<br/>　　程大雷目光转了一圈，心中忽有了主意，道：“我告诉你一首诗，你写在大门上。到时候门自然会开。”
<br/>　　“咦！”
<br/>　　刘发财睁大眼睛，看模样程大雷和自己相似，气质其实也差不多，若说程大雷是个杀猪的屠户，不会有人怀疑，但他说自己会写诗，的确就有些匪夷所思了。
<br/>　　何况这是什么地方？琅琊万卷山，读书人的圣地，才子的窝子，你要靠一首打油诗当敲门砖，怕是不能。
<br/>　　楚云生垂着头，心里也没太在意，他此刻还处在颓废的情绪中。
<br/>　　随之，就看到程大雷以脚尖点地，潦草的在地上写字。一开始楚云生不觉得怎样，但看完第二句，他的瞳孔已炙热起来。
<br/>　　拱手冲程大雷拜道：“阁下高才，楚某五体投地。”
<br/>(本章完)
