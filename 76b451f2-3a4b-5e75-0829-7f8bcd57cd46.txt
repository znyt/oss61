<br/>　　三个月过去，妇女命大，在一帮家属的层层保护下，顺利回到了家。

<br/>　　这次没有咬，因为妇女一声尖叫后，直接被吓死了。
<br/>　　属于恐吓谋杀。
<br/>　　棕色长毛狗不止一次如此变态的行凶，最后引起了妖怪管理局的人注意，大力逮捕。

<br/>　　原因是它生下来的一胎猫崽子，被宠物店贩卖。
<br/>　　不但宠物店老板被杀死，那些购买宠物猫崽子的主人，也全部被杀。
<br/>　　那只黑猫抓到，直接处以死刑。

<br/>　　这铐在自己手上的锁妖枷太变态了，他不但不能动用属性能力，就连意识力也不能使用。
<br/>　　他现在就是手无缚鸡之力，要是没了这具人形态，涂小安也就是条毒蛇而已。
<br/>　　说起人形态，涂小安为了救那个小女孩，被轿车一撞，还损坏了百分之五十的肉身强度。
<br/>　　这具才刚刚购买的化形术，一半没了。
<br/>　　好气，真的好气，这是典型的好人没好报吗。
<br/>　　当一条正蛇难呀，救人都救出事情。

<br/>　　涂小安的耳畔响起了一道戏谑调侃的声音，他扭头过去，既然看到角落边蹲着一只漆黑如墨的大鸟，声音正是这只大鸟的口吐人言。
<br/>　　细看之下，它的两只脚正是铐着比涂小安小很多型号的锁妖枷。
<br/>　　天底下，只有两种鸟是纯颜色的，一种是白鹭，一种是乌鸦。

<br/>　　显然，这漆黑如墨的大鸟正是乌鸦是也。
<br/>　　咦...一只乌鸦既然能口吐人言，达到二阶精怪的地步，不简单呀。
<br/>　　“你怎么知道我被下套抓的”
<br/>　　涂小安好奇的问。
<br/>　　那只乌鸦眸有慧光，高傲的说：“本鸦有顺风耳呀，刚看那位愚蠢的人类将你压进来，随后就听他跟同事炫耀，骂你蠢的一B”
<br/>　　涂小安闻言差点没一口老血喷出来。
<br/>　　至于这只乌鸦为什么知道涂小安是蛇精，这点就不需要问了。
<br/>　　气息代表一切，眼睛看到的未必是真。
<br/>　　不过这也需要一定的道行才行。
<br/>　　一般的修武者，是看不出涂小安是蛇精所化。

<br/>　　说来也奇怪，这只乌鸦虽然一身黑毛，但给人的感觉却并不丑，甚至看它让人产生一种鸡窝里出凤凰的错觉，很顺眼，属于那种越看越喜的。
