陆寒霆抬头就撞上了她的目光，今天她穿了一件淡黄色的长裙，清纯的长发掖在了耳垂后面，小圆圆乖软的趴在她的怀里，也睁着一双圆溜溜的眼睛望着他。

<br/>
<br/>        “知道了。”陆寒霆掀开身上的被子，这时发现安眠药还在手边，昨晚忘记放回抽屉里了。
<br/>
<br/>        这时夏夕绾走进来两步，“你…还好吗，现在睡眠怎么样，我帮你把个脉吧？”

<br/>        她要走过来了。

<br/>        陆寒霆不想让她看见安眠药，更不需要她给他把脉，不要再关心他了，这样只会让他更加的舍不得。

<br/>        “是不是听不懂人话，滚出去！”陆寒霆拿起了枕头，砸到了她的身上。

<br/>        夏夕绾没有避，真的被枕头给砸到了，其实一点都不疼的，但是心里好像裂开了一道血口，撕心裂肺的。

<br/>        白皙的眼眶再次一红，她转身就跑了出去。

<br/>        她走了！

<br/>        真的走了！

<br/>        喵喵喵~

<br/>
<br/>        这个小圆圆好像是他买回来的吧，但是它的心完全到夏夕绾身上去了，见他欺负自己的女主人，小圆圆变得奶凶奶凶的，跟它的女主人一样。

<br/>        ……

<br/>        夏老爷子醒了！
