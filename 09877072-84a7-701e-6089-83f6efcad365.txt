　　“呼——”
<br/>　　一颗青色星辰之上，元始长长呼出一口气，气息之中夹杂着璀璨星光，如同一道长长的匹练飞出很远，才消散于虚空。
<br/>　　元始站了起来，内视人体，只见他人体之中，各处玄窍之处，在闪闪发光，各种颜色的光芒，红色，青色，白色，金色，蓝色……
<br/>　　各种各样的光芒在神窍之中散发出来，使原本黑暗幽深的人体变得璀璨无比。每一颗玄窍，都是一颗星辰，无数星辰照亮了这个人体。
<br/>　　元始的肉体也变得强悍无比，他向着虚空一挥拳头，顿时，拳啸生风，空间破碎，时间仿佛都停顿了，可见力量极为强大。
<br/>　　数百年时间的修炼，元始体内被炼化打开的玄窍已经增到三百多了。
<br/>　　“只剩下太阳、太阴两颗至尊星辰了。”
<br/>　　元始看着远处，轻声低语道。
<br/>　　向着星空深处深深看了一眼，元始不再停留，转身向着太阴星飞去。
<br/>　　却说在星空深处，天道之力触及不到的一个空间内，有一个身穿银袍，满头银发，手握权杖的老者。老者全身上散发着苍茫亘古的气息，星辰流转，银光幻灭。他脚踏星辰，星光汇聚而来，涌入到他的身体之中。
<br/>　　此刻，老者看着元始离去的身影，大声骂道：
<br/>　　“可恶，若不是吾之伤势还没有养好，洪荒之中又有天道力量的覆盖，吾又怎会容忍一个小辈在吸取吾之星辰本源。”
<br/>　　“可恶！真是可恶！盘古！天道！啊！”
<br/>　　老者大叫，嘴里胡说八道，不久之后，他身上的气息变得萎靡许多。他不再多言，而是闭上双眼，默默地用星辰的力量疗伤。
<br/>　　正在前行的元始却是不知道这些，这些年在星辰之中修炼，刚开始他还不知道，直到后来，他就隐隐感觉到，有人在注视这自己。
<br/>　　但他也没有多管，只认为是星辰之中隐修的大能。只要不是来阻止自己的，管他注视不注视的，随便他。若是敢来阻止他，他也绝不会任人欺负。他才管他是谁，来了就好付出代价。
<br/>　　元始一路急行，星辰在他脚下飞过，很快便抵达太阴星。
<br/>　　经过一番寻找，元始找了一个地方，准备闭关接引太阴星辰之力。
<br/>　　太阴星可不是一般星辰可比的，太阴星为两大至尊星辰之一，元始此次不仅要突破《玄黄不灭》第五重，他还要顺便突破第六重。
<br/>　　第五重星辰入神窍，第六重阴阳筑神魂。第六重需要的阴阳之力，没有什么地方的阴阳之力比太阴星与太阳星还多了。
<br/>　　准备一番，元始在闭关之所布置了一个阵守护法，之后才开始接引太阴之力。
<br/>　　太阴星太大了，难保不会出现什么意外，闭关之时最忌被人打扰。若是被人打扰了突破，轻则受伤，重则走火入魔，乃至身死道消。故而，为了小心为妙，元始还是布置了一个阵法守护。
<br/>　　阵法之中，元始强横的神识之力顺着星球一直往下延伸，太阴星太大了，故而，星辰本源也极为难找。
<br/>　　一直搜寻良久，元始还是没有找到星辰本源。
<br/>　　“嗯？怎么会没有？这不可能啊。”
<br/>　　元始疑惑不已，但还是没有放弃，任然不停的搜寻，一寸一寸，不放过任何地方。
<br/>　　“嗯？”
<br/>　　良久，顺着神识，元始感受到一股极为至阴至寒的力量，甚至连元始的神识也能冻结。
<br/>　　至阴之力冻住元始的神识，元始也找到了那股至阴之力，这才是太阴星的星辰本源。斩断这一缕被冻住的神念，元始再次放出一道神念，向着之前感应到的方向看去。却见那星辰本源并不在太阴星深处，而是在一株神树之下。
<br/>　　顿时，元始便知道了那神树是什么了。太阴星至阴至寒，连个生物都极难见到，又怎么会有神树。若是有神树，那就只有可能是十大先天灵根之一的月桂树了。
<br/>　　相传月桂树乃盘古睫毛所化，到底是不是真的，元始亦不知晓。
<br/>　　想到此处，元始也就知道了太阴本源在何处了。
<br/>　　“嗯？谁？！”
<br/>　　就在元始将要接引太阴本源之时，一道强横的声音通过元始神识传到元始神海之中。顿时打断了元始的接引。
<br/>　　阵法之中，元始睁开双眼，眼中闪过一丝厉色，走出阵法，向着太阴星中心走去，那里，正是月桂树所在之地。
<br/>　　损失一道神念，元始倒是不甚在意，他元神强大无比，度过涅槃之境，只差一步便能突破到了第七转星空境。损失的这点神识，他还不放在眼中。倒是此处竟然有主人，这才是元始动容的理由。
<br/>　　有主人，那就意味着他想要接引太阴星辰本源不是一件容易之事了。可能要做过一场才是。
<br/>　　一路急行，很快，元始就到了月桂树之前。月桂树被一个秘境笼罩，但元始还是发现了。却是秘境之前，早已有人等待了。
<br/>　　目光所至，视线之中，赫然是一道倾城的身影，此时此刻，她正离地不足三尺，凌空而立。月华笼罩之下，一道身穿素色白袍的绝丽身影正站在秘境之前，遗世独立，场景绝美，如诗如画。
<br/>　　此时此刻，元始看到这道绝世身影，眼中闪过一道惊艳之色，不由为之一愣。饶是见惯了美女的他，此时此刻，也不由为之一阵呆滞。
<br/>　　精致的容颜仿佛玉质一般，让人无法从中找到一丝瑕疵，那遗世独立的缥缈气质，倾城绝世的傲人身姿，无不让人为之沉醉。
<br/>　　这一刻，元始只觉得浑身一震，内心深处涌起一股难言的感觉，如一道清泉闪电，瞬间达到四肢百骸，涌入心扉，平添了几分感慨。
<br/>　　好在，他修为深湛，道心坚稳，只是瞬间的失神之后，便就清醒过来。
<br/>　　“道友为何人，闯我太阴星所谓何事？”
<br/>　　一道清脆悦耳的声音传来，如同一汪清泉涌入元始心间，让他不由为之一荡。
<br/>　　但瞬间之后，元始就恢复了淡然之态，到了他如今的境界，早已能够控制自己的情感了，那些东西，已经不再能够影响自己的心情了。
<br/>　　“本座乃道印天尊元始，强闯你太阴星？太阴星又不是你私人之物，本座想来就来，想走就走，何来‘强闯’一词？”元始恢复了平静，淡言道。
<br/>　　“你——”那女子顿了顿，再次恢复平静，她感应不到对方的实力，神色有些凝重道：
<br/>　　“贫道自出生起便一直在太阴星之上修炼，此处可算是贫道的。”
<br/>　　元始笑了笑，毫不在意道：“你出生在此，就算是你的了？那本座出生在洪荒，那洪荒算不算是本座的。”
<br/>　　“你——”女子有些恼怒，大声喝道：“你这人好生无礼，我也不便与你废话，你今日来此，所为何事？”
<br/>　　“正好，本座也不喜欢废话，本座来此，只为了此处的太阴星辰本源。”
<br/>　　“不行！星辰本源不容有失！”女子坚决道。
<br/>　　“唔！看来还是要做过一场才行。”
