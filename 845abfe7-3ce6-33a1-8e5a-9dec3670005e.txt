　　秦龙少主的一身实力，几乎可以跟稍弱的半步天人境强横抗衡了。
<br/>　　当然单单只是抗衡，却是无法跟尹龙仙一般，越级击败半步天人境。
<br/>　　可纵然如此，他的全力一击，哪怕是这苍龙宗的老祖，都要好好应对，绝不可能做到这随手一击，就将他的龙爪打破。
<br/>　　“阁下也是龙族，为何要帮这小子来攻打苍龙宗？”
<br/>　　秦龙少主看着狂龙帝，低沉的说道。
<br/>　　他也看得出，狂龙帝也是一位龙族妖神，而楚风眠只是一个人类。
<br/>　　他有些不明白，狂龙帝为何要来为楚风眠出头。
<br/>　　“老朽现在只是公子面前的一位马前卒罢了，小辈，你资质不错，我不愿镇压你，退去吧。”
<br/>　　“看来龙帝跟苍龙宗，倒是关系不浅啊。”
<br/>　　楚风眠看到狂龙帝的表现，嘴角露出几分笑意道。
<br/>　　看来狂龙帝也是跟这苍龙宗关系密切，现在才会不让楚风眠，让他主动出手。
<br/>　　毕竟如果楚风眠一旦出手，那只怕是会掀起无边血雨腥风。
<br/>　　以楚风眠的性格，若是这苍龙宗，执意于他为敌，那便是只有灭亡这一条路。
<br/>　　“有些渊源罢了，若是公子下令，老朽愿意为公子出手，灭了苍龙宗。”
<br/>　　狂龙帝看着楚风眠，平静的说道。
<br/>　　“那倒不用。”
<br/>　　楚风眠随意的开口道。
<br/>　　他看的出来，不管是狂龙帝还是尹龙仙，都跟苍龙宗有着极为密切的关系。
<br/>　　楚风眠也不会做这种赶尽杀绝的事。
<br/>　　“这两个人，到底是什么来路？居然要灭了苍龙宗？”
<br/>　　那秦龙少主在一旁，听到楚风眠跟狂龙帝的交谈，汗如雨下。
<br/>　　他听得出来，楚风眠跟狂龙帝的交谈，绝不是虚张声势，楚风眠出手不出手放在一边。
<br/>　　单单是狂龙帝，如果真的要出手，只怕是苍龙宗就算是不覆灭，其中的老祖也要伤亡大半，倒时候苍龙宗也会名存实亡了。
<br/>　　“怎么回事？”
<br/>　　嗖！
<br/>　　就在这秦龙少主一时间不知道说些什么的时候，数道遁光，突然飞了过来，正是苍龙宗的一位位老祖。
<br/>　　随着这一位位老祖的到来，一道道强横无边的龙威，轰然从天凝聚，笼罩在了楚风眠等人的身上。
<br/>　　“破！”
<br/>　　楚风眠感受到这笼罩下来的龙威，嘴角露出几分不屑之意，一个破字喊出。
<br/>　　群龙咆哮，从楚风眠身上笼罩下来的龙威，轰然破碎开来。
<br/>　　“天龙八音？”
<br/>　　几位苍龙宗老祖，都是目光闪烁，看向了楚风眠。
<br/>　　“小畜生，就是你抽了恒龙的龙筋？你还敢来苍龙宗？看来你是自寻死路！”
<br/>　　其中一位老祖，目光看向楚风眠，语气之中蕴含着无边怒意，眼神之中的杀意，毫不掩饰的看着楚风眠说道。
<br/>　　“他是谁？”
<br/>　　楚风眠都懒得去看那老祖，而是转过头看向尹龙仙问道。
<br/>　　“他就是恒龙少主的师尊，撼龙王。”
<br/>　　太古时代之中，远古战龙一族，半步天人境的龙族，只能够称为龙王，至于龙帝，则是只有羽化登仙，踏入天人境的龙族，才有资格称呼的。
<br/>　　这苍龙星域之中，还继承了这古老的规矩，所有半步天人境的龙族老祖，都被称为龙王。
<br/>　　当然狂龙帝是一个异类，他之前虽然是半步天人境，可是他的一身实力，跟真正的天人境相比都毫不逊色了，也可以自称龙帝了。
<br/>　　不过现在狂龙帝度过了天人大劫，已经是一位真真正正的龙帝了。
<br/>　　“恒龙少主的师尊？怪不得跟那个废物一样的狂妄，没有几分实力，倒是一样的狂妄。”
<br/>　　楚风眠看着那撼龙王，不屑的说道。
<br/>　　“小畜生，你说什么？”
<br/>　　撼龙王听到楚风眠的话，大怒道。
<br/>　　“公子……”
<br/>　　狂龙帝刚刚想要开口，似乎是想要出手，可被楚风眠一把拦了下来。
<br/>　　“算了，此人就由我亲自对付吧，撼龙王是吧？正好，我今日就抽了你的龙筋，让你好跟你的弟子一起去作伴。”
<br/>　　楚风眠一步踏出，眼神中露出几分寒意道。
<br/>　　这秦龙少主楚风眠可以不杀，但是这撼龙王居然是敢辱骂楚风眠，那这撼龙王必定是一个死人了。
<br/>　　看着楚风眠出手，狂龙帝也是叹息了一声，他终究也是跟苍龙宗关系密切，不愿意出现伤亡。
<br/>　　可看到楚风眠要主动出手，他也知道这不是他可以阻拦的了。
<br/>　　楚风眠之前默认让他出手，已经是给了他一个机会了，算是给足了他的面子了，这一次他也没有阻拦楚风眠的理由了。
<br/>　　“笑话，要抽了老夫的龙筋？老夫倒是要看看，你有着几斤几两，敢说如此大话，给我死！孽龙入海！”
<br/>　　撼龙王听到楚风眠的话，暴怒长啸一声，整个人的身形从空中膨胀起来，化为了一条巨大无比的孽龙，充斥毁灭之力，向着楚风眠顿时轰击而来。
<br/>　　巨大无比的两道龙爪，铺天盖地之间向着楚风眠狠狠的轰了过去。
<br/>　　“孽龙武技！我们苍龙宗最为玄妙的武技之一，据说是威能最强，主杀戮的武技！”
<br/>　　“看来撼龙王也是动了真怒了，要取了这小子性命！”
<br/>　　一些苍龙宗老祖，都是目光闪烁，看着这一场战斗。
<br/>　　从寰龙城主的形容之中，楚风眠的实力简直是神乎其神，可他们却并不相信。
<br/>　　毕竟楚风眠只是一位九劫古帝罢了，他们可都是从九劫古帝之中晋升而来的，自然不相信一个九劫古帝，可以强横到了哪里去。
<br/>　　任由是天空之上两道巨大的孽龙之爪，向着楚风眠抓了过去。
<br/>　　楚风眠的身影，却是屹立在哪里，依然未动。
<br/>　　“一动不动。”
<br/>　　“难道是这小子还没有反应过来？”
<br/>　　“我当这小子到底是多厉害，原来只是这种货色？”
<br/>　　看着楚风眠的举动，几名老祖都是不屑的说道。
<br/>　　“小子，给我死！”
<br/>　　那撼龙王看到楚风眠一动不动，都是脸上露出了几分凶狠之色，他要杀了，为恒龙少主报仇。
<br/>　　可就当这两道孽龙之爪，轰击在楚风眠身躯之上的一刻，从楚风眠的身上，一层光华散发而出，一枚枚菱形晶体，从楚风眠的身上浮现而出，化为了一道屏障。
