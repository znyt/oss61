　　“循环？”鲁初雪被完全吸引，连记录都差点忘了。
<br/>　　“就是循环，从人到丧尸，再到人的循环。”江左无比认真，道：“这一推论被命名为循环论。”
<br/>　　看着听的目瞪口呆的鲁初雪，江左继续道：“所以人类和丧尸之间是有联系的，在基因上，解毒剂的最初猜想，就是定准了五阶丧尸之后产生的变异，那些超级丧尸的晶体之所以与众不同，就是因为包含了更多能与新人类基因有联系的东西。”
<br/>　　“丧，丧尸会进化成·人么？”鲁初雪的声音都开始颤抖，紧张问道：“另一种人？新物种？”
<br/>　　“这个我不知道。”江左摇着头，道：“但你们难道没有发现那些丧尸从一开始都肉·身腐烂，但渐渐随着等阶升高，身上的各个组织都逐渐完善起来，尤其是这一次南署基地大战中·出现的超级丧尸头领，不仅皮肤完好，还有了五脏六腑，甚至长出了头发！”
<br/>　　鲁初雪瞳孔剧烈收缩，心中涌·出了惊涛骇浪：“我听说了这件事，最新传来的战报中有说明，虽只有寥寥数语但也足够震惊了，等到南署那里的大战结束，我们会有更详细的战后情报！只是那只超级丧尸的小头领，不是才这两天第一次出现么？但这个循环论的推理……”
<br/>　　“循环论虽然是我们研究院的最新推理，但却是三个月前就开始的，院长本人当然不可能知道超级丧尸小头领的具体情况。”江左说着，眼神里带着佩服：“我也是在拿到了解毒剂成果之后才知道这一推论，联想到最新超级丧尸头领的状况，以及出现过多次的智力型丧尸，才理解了这循环论。”
<br/>　　“这一推论骇人听闻，当下还没几个人能接受，我的很多同僚还在消化中，但我想楚涵上将知道后，一定能立即接受或是判断真伪。”江左说着，发出一声感叹：“他的想法本身就独特，不被固化。”
<br/>　　鲁初雪握着笔的手在不停颤抖，努力将这一推理记录下来，而后震撼的说道：“难怪你说哪怕说出了关键点也没几个人能制造出解毒剂，能理解这想法的人恐怕就不多，更别说仿造解毒剂，根本就制造不出来啊！循环论，真是令我大开眼界！”
<br/>　　江左点头：“调配过程也非常复杂，环节很多，难度系数很高，真的数量很少。”
<br/>　　鲁初雪叹口气，眼中也不禁带上了钦佩：“你们院长真是个奇才，真是佩服楚涵，似乎世界上的厉害人物都被他招揽到了狼牙。”
<br/>　　江左眼神古怪的看着鲁初雪：“狼牙是有很多怪才大能者被楚涵上将招揽过去，但我们院长可不是那一类啊，她不可能去其他地方发展的。”
<br/>　　鲁初雪皱眉不解：“为何？楚涵上将招揽人才的手段众多，无所不用其极啊！”
<br/>　　江左好似看白·痴一样的看着鲁初雪：“我说姑娘，狼牙研究院的院长，名叫袁熙叶，是文明时代门萨组织排名第十一位的人，她当然只会站在楚涵这一方。”
<br/>　　鲁初雪瞪大眼睛，恍然大悟道：“难怪！门萨内的成员各个智力属于世界顶尖，内有排名一说还是第一次听到，世界第十一位的高智商，难怪可以研究出解毒剂，又退出循环论这一逆天想法。”
<br/>　　“可是……”说着鲁初雪还是不解，问道：“这和楚涵有啥关系？”
<br/>　　江左嘴角抽·搐个不停：“因为她是楚涵的老妈。”
<br/>　　“噗！咳咳咳！”鲁初雪顿时被震惊的被自己口水呛到，拼命的咳嗽到脸都憋红。
<br/>　　楚涵的老妈是门萨内智力排名第十一位的成员！
<br/>　　楚涵的老妈是狼牙研究部的院长！
<br/>　　楚涵的老妈是解毒剂的发明者，还提出了循环论！
<br/>　　这……
<br/>　　鲁初雪心中的震惊难以言喻，只觉得似乎有什么东西曾经被世人刻意忽视，从而搞错了某个至关重要的点。
<br/>　　“所以楚涵上将？”鲁初雪话说到一半，竟然发现自己有些不知道如何继续问下去。
<br/>　　江左却一目了然的明白她想问什么，点着头回答道：“楚涵上将的智力当然也很高，甚至不能用高这个词来形容，而是变·态级别，他的思维运转速度无人能及，只是他并没有在文明时代参与过门萨组织的考核，不然……”
<br/>　　说到这里江左摇摇头，笑道：“但这些都不重要，有着门萨智力排名第十一位的老妈，排名第十位的老爸，我们长官到底遗传了他父母的多少智商，虽然没有具体排名或数值，但他在这个末世纪元的表现，尤其是这么多奇迹的创造，其才华应当有与之相配的高智商吧！”
<br/>　　鲁初雪听着，忽然察觉到一丝不对劲：“等等，楚涵老爸也是门萨成员？还是第十位？！”
<br/>　　最后一句话，鲁初雪简直就是在尖叫，虽说当下的门萨组织已经不复存在，文明时代的一切都在末世纪元化为泡沫消失，但这种传说中的存在还是让鲁初雪大为震惊。
<br/>　　江左不解的看着她，呐呐道：“这事在狼牙人尽皆知啊，我以为你们都知道……”
<br/>　　“真不知道！”鲁初雪有些想吐血。
<br/>　　暗道要是早知道楚涵的双亲来头这么逆天，当初她的长官段江伟，就不可能多次试探还差点冒险。
<br/>　　毕竟谁也不会去招惹一个战力超绝不说，还有可能智商逆天的家伙！
<br/>　　望着表情千变万化的鲁初雪，江左挠了挠头：“科技部的那个和上，是门萨内部智力排名第一位，这事你不会也不知道吧？”
<br/>　　“啥玩意儿？！”鲁初雪震惊的霍然而起，紧接着就差点因为刺激过大而晕过去。
<br/>　　第一，那个光头胖子竟然是第一！
<br/>　　江左有些尴尬：“好吧，你又不知道，那你知道啥？不是我说，你们段氏基地的情报部门是不是个摆设啊？咋啥都不知道！”
<br/>　　鲁初雪彻底噎住，一个字也说不出来，只能铁青着脸道：“可能真是摆设……”
<br/>　　江左也不好继续刺激她，起身告辞道：“排名什么的又不重要，你反应这么大干什么，我该说的都说完了，说出来的都是可以公布的东西，当然你应该明白这些过于先进的推理……”
<br/>　　“我懂。”鲁初雪恢复了理智，道：“内容属于密报，只有牧司令及几名将领会知道。”
