　　“也只有这个解释了。”妙心道，“我不相信佛子花能凭空想象出那些字画的细节。”
<br/>　　梧桐想起什么，问叶少阳：“你是怎么知道我们之前是在幻境中的？”
<br/>　　叶少阳不着急回答，把阴阳镜拿出来，在上面按了一下，让罡气渗透进去，说道：“出来。”
<br/>　　镜面一闪，一道人影飞了出来，是一个身穿旗袍、身材婀娜的女子。
<br/>　　在场至少都是天师、禅师牌位的法师，短暂的发愣之后，立刻认出她的真身来。
<br/>　　“鬼域之魂！”梧桐叫起来。
<br/>　　“没错，她……是我的门人，名叫美华，认识一下吧。”
<br/>　　陈晓宇一听，顿时感觉自己找到了发挥的机会，冷笑道：“原本你也收的有仆人，还是个美艳的女鬼……”
<br/>　　叶少阳听出他话语中的恶意，望着他，冷冷道：“从现在开始，你莫跟我说一句话，不然就死战到底！”
<br/>　　“好的口气，怕你不成！”陈晓宇往这边走来，立刻被卢晓清等人拦住。
<br/>　　“这人好讨厌。”美华在叶少阳耳边说道，“要不要……”
<br/>　　“你不是他对手，不要找事，将来我会收拾他。”叶少阳小声说，随即问她：“你之前干啥呢，也不出来。”
<br/>　　“我？我在里面修行呢。不是说了我听你的召唤再出来吗？”美华笑了笑，“老大你这阴阳镜里是谁开辟的空间，里面种了一株紫菱，灵气充盈，是个修行的好去处。”
<br/>　　“哦哦，那可能是杨宫梓干的。”
<br/>　　“那是谁？”
<br/>　　“别问了，再等一百年你就知道了，你继续进去呆着吧，没事了。”叶少阳把美华又赶进了阴阳镜中，现在用不上她，又怕她被陈晓宇暗算。
<br/>　　“通灵佛子花最可怕的地方，不是模拟周围的环境，而是能够模拟一切，连法器和各种能力也能模仿，感觉这应该不是模仿，是某种渗透……像我这阴阳镜，在幻境之中，跟真实一模一样，但是佛子花却不能模仿另外一个空间的东西……我这阴阳镜里自有一方天地，刚那妹子就藏在里面，这个，是佛子花绝对无法模仿的。”
<br/>　　他这么一说，大家立刻也都懂了。
<br/>　　“也就是说，你发现你那鬼仆不在阴阳镜中，依次判断出刚才那是幻境？”
<br/>　　“没错，当时我们被困在里面，我想到的办法就是让她闯出结界，去外面搬救兵来救我们出去……”说到这，叶少阳想到什么，“现在想起来，幻境之中发生的一切都太巧合了，不过也足够真实，才让我们以为那就是幻境。”
<br/>　　众人远远望着通灵佛子花，个个都是颇有感慨：如果不是叶少阳发现的及时，他们到现在只怕还要困在幻境之中，只怕真的会饿死在那间并不存在的石室里。
<br/>　　一想到这个，众人都是有些脊背发凉。
<br/>　　“我有个疑问。”毛小方道，“既然这通灵佛子花有办法将我们困在幻境之中，那我们的身体，当时就在这里发呆是吧？它为什么不趁机杀了我们，这样不是更加轻松？”
<br/>　　“也许，它没有办法直接杀人。”叶少阳想了一下道，“制造幻境，可能也是用上了它一身修为，它没有办法对我们的肉身做什么，这个其实不用多想，如果它真能做到，我们现在已经死了。”
<br/>　　“那我们现在怎么办？”毛小方道，“再一次烧了它？”
<br/>　　叶少阳笑道：“你这话说得有问题，我们这是第一次烧它。”
<br/>　　一边说，一边从背包里拿出些黄裱纸，倒了一碗水，把黄裱纸点燃之后立刻扔进去，然后捞出湿漉漉的，撕成一小块一小块的，分给大伙，让他们各自都塞进鼻孔里，免得待会再次闻到通灵佛子花散发的香气，再一次进入幻境。
<br/>　　来到通灵佛子花前面，叶少阳先点了一张地火符打过去，接着梧桐拿出五宝金莲，再一次吹出了金莲业火。
<br/>　　火焰落在通灵佛子花的花叶上，立刻燃起大火，跟之前在幻觉中发生的那一幕差不多。
<br/>　　通灵佛子花将花叶收紧，保护着中间那团液体组成的膜，等到花叶完全枯萎的时候，中间的膜突然破裂，一股液体突然朝着叶少阳喷射过来，叶少阳狼狈躲开，骂道：“妈个蛋，干啥针对我一个！”
<br/>　　一颗液体中的孢子，落在叶少阳手中的阴阳镜上，一闪而逝，钻了进去，没有人看得见……
<br/>　　再没有任何意外发生，通灵佛子花被烧了个干干净净，跟幻觉中一样。
<br/>　　“这一次，不会再是幻觉了吧。”梧桐望着花坛中的一堆焦灰，喃喃说道。
<br/>　　为了验证之前的猜测，在叶少阳提议下，大伙打开了花坛，结果在花坛里面所看到的跟幻境中一模一样：至少有数十具尸体，像是被煮成了一锅肉。
<br/>　　这也印证了，通灵佛子花制造的幻境，是以现实世界为蓝本的。
<br/>　　总算搞定了通灵佛子花，大伙心中都是长长地松了一口气。那种反复被困在幻境中的感觉，几乎在每个人心头都留下了阴影。
<br/>　　叶少阳这辈子都不想再跟这种可怕的植物打交道了。
<br/>　　阴阳镜内，在一片氤氲之中，有一方池塘，池塘边上有一座造型别致的小楼，后面还有个花圃，长满了各种五颜六色的花朵。
<br/>　　这一块不属于人间的乐土，是当初由杨宫梓开辟，后来又经过雪琪和陈露两个人完善，形成了现在这个模样。
<br/>　　美华此刻就躲在花丛深处，欢快地吸着这里比外界充盈得多的天地灵气……这里的天地灵气，来自小楼后面的一口井里，美华第一次进来就好奇地查探过，这里有一道混沌之气，自演阴阳，不断生成天地灵气，因为完全与外界隔绝的缘故，这里的天地灵气没有杂质，要精纯地多
<br/>　　美华进来之后，就一头扎在水井边上开始修炼，中间被叶少阳拎出去一下，又被赶回来，继续愉快的修炼。突然，周围的气息产生了一阵异动，美华察觉到，抬起头看去，正好看到一个小家伙探头探脑地钻过未开辟的氤氲。
