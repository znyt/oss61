　　瓜瓜三言两语说了经过，叶少阳低头看那个被擒住的黑衣人，也是有些面熟，但一时想不起来在哪里见过，问他：“你是什么人？为什么要刺杀陈悦？”
<br/>　　黑衣人不说话。
<br/>　　叶少阳立刻就要动手，林三生道：“少阳，他是我的人。”
<br/>　　仿佛一道雷电，劈在叶少阳和瓜瓜头上，两人当场愣住了，傻傻地看着林三生。
<br/>　　林三生走过来，伸手拉了那黑衣人一把。“你们先回去吧。”
<br/>　　这些黑衣人果然听话地一起走了。
<br/>　　陈悦这时候已经醒了，也呆呆地看着这一幕。
<br/>　　“为什么，为什么要杀我娘！”瓜瓜冲上去要找林三生的麻烦，被叶少阳抱住。
<br/>　　“她不是你娘。”林三生淡淡说道。
<br/>　　叶少阳回想了一下经过，说道：“这是你谋划好的是不是，你拖着我说话，然后让手下对她行刺。昨天晚上，偷听我们说话的也是你的人。”
<br/>　　“不是我的人。”林三生看着他，“是我自己。”
<br/>　　叶少阳茫然看着他。
<br/>　　“我一直都跟着你们，跟你们一样，我也很想弄清她的身份，确定，她是不是我要找的人。”
<br/>　　叶少阳听着。
<br/>　　林三生摊了摊手，“没了。”
<br/>　　“没了？你为什么要杀他？”
<br/>　　林三生叹了口气，道：“叶少阳，这一切是为了你，我就知道，你一旦发现，我就再也没有机会行动了，但人算不不如天算，还是让瓜瓜发现了，是我太着急了，本来还是可以等机会的……”
<br/>　　“你别废话了行吗，赶紧说啊！”
<br/>　　叶少阳把瓜瓜放开，瓜瓜立刻又要冲过去，林三生望着他，淡淡说道：“我们这么久的兄弟，难道顶不了一个刚认识的人？”
<br/>　　瓜瓜怔了一下，道：“现在是你要杀她，如果她要杀你，我一样也要拼死保护你！再说我现在对你已经够客气了好吧！”
<br/>　　林三生微微一笑，“瓜瓜，如果，她跟少阳只能活一个，你选谁？”
<br/>　　瓜瓜一下怔住，怒道：“为什么要选！”
<br/>　　“因为，我已经选了！”林三生也罕见地怒起来，“我跟这个姑娘压根不认识，我为什么要杀她？还不是为了你家老大！这个恶人，就让我来当吧！”
<br/>　　突然袖子一挥，一道青色的人影飞出，极快得射向陈悦。
<br/>　　叶少阳反应也极快，一个箭步上前，捏了一枚铸母大钱，打了过去，击中了青衣人，爆发出一波灵光，但落在青衣人身上，完全没反应。
<br/>　　这么厉害？
<br/>　　来不及作法了，叶少阳飞身上前，勉强赶在青衣人前面、将半个身子挡在了陈悦身前，单手结印，朝青衣人打去。
<br/>　　青衣人挟带着一身极强大的气势撞过来，叶少阳站不住，抱着陈悦一起摔倒在地上，青衣人再度袭来，叶少阳一个翻身把陈悦压在身下，青衣人本来已经冲到他面前，猛然站住。
<br/>　　“道风！”叶少阳大声呵斥。
<br/>　　青衣人站住之后，被大伙看清楚，正是道风。
<br/>　　“原来你也有份！”叶少阳咬牙。
<br/>　　林三生走上来，拿出山河社稷图，对道风说道：“你还没复原，不能在人间久留，今日不成，将来再说！”
<br/>　　道风犹豫了一下，试图钻进山河社稷图，被叶少阳起身拉住，“说清楚，你们都给我说清楚，这是怎么回事，否则你们一个都走不了！”
<br/>　　林三生看了看道风，道风犹豫道：“直说了吧，事情到这一步，必须有个了断。”
<br/>　　林三生道：“去空界说。”
<br/>　　当下先打开了虚空裂缝，让叶少阳跟他一起进去。
<br/>　　叶少阳把陈悦拉起来。
<br/>　　陈悦一脸茫然，紧紧拉着叶少阳的手不敢松开。
<br/>　　“别怕，有我在，没人能伤到你！”
<br/>　　瓜瓜也走上来，现出人身，拉着陈悦另一只手，三人一起钻进了虚空。
<br/>　　林三生和道风也钻了进去。
<br/>　　虚空的那一头，居然连着青丘山，一进来，叶少阳立刻看到了小九，小九看了陈悦一眼，又看了看林三生他们，道：“我就知道你们得手不了。”
<br/>　　叶少阳愕然，原来这件事……小九也有份。
<br/>　　林三生不高兴地说道：“你要当好人，我也没办法。”
<br/>　　小九道：“那是我不支持你们这么做，但也不反对。”
<br/>　　然后看了一眼叶少阳，让他们跟着上山。
<br/>　　陈悦一路上都缩在叶少阳身边，不知所措。
<br/>　　到了小九的妖宫，小九让阿黄和阿紫去沏茶，叶少阳摆手道：“别忙活了，赶紧的，你们快告诉我真相！”
<br/>　　瓜瓜也在旁边附和。
<br/>　　小九起身去关了宫门，作法调集山上的妖气，将妖宫封锁起来，暂时形成了一方与外界完全隔绝的空间。
<br/>　　道风站在屏风前，欣赏上面的化作，不理他们。
<br/>　　“罢了，还是我来吧。”林三生无奈苦笑，望着瓜瓜说道：“瓜瓜，你告诉少阳，你来人间找少阳，除了我们已经知道的，你还背负着什么秘密？”
<br/>　　叶少阳心中一惊，猛然想了起来，瓜瓜身上的确还有一个秘密，是关于自己的，当时他不愿意说，自己也没逼他，不过他当时说的理由就是为了自己好……
<br/>　　当下朝瓜瓜看去，瓜瓜发着呆，冲林三生说道：“我的秘密，跟这件事有什么关系？”
<br/>　　“大有关系……我们要杀这个姑娘，为得就是你心中这个秘密。”
<br/>　　瓜瓜惊呆。
<br/>　　叶少阳道：“别打哑谜，就我一个蒙在鼓里是吗？”
<br/>　　林三生见瓜瓜不说，转头对叶少阳说道：“上一次，你父亲冒死见你，要你寻找与你差不多大的法术界天才，然后务必要杀死对方，你还记得吧？”
<br/>　　叶少阳怔了下，猛然明白过来，转头望着陈悦，“你说她是少年天才？”
<br/>　　“她有七宝妙树和七绝宝经，体内跟你一样有强大的戾气，一旦炼化，顷刻间就是个绝顶强者。”
<br/>　　叶少阳一时倒无话可说，想了想道：“那照这么说，我遇到强者就给杀了？”
<br/>　　“你自己想一下，人间法术界，还有谁有这样的天赋，跟你还是同年同月同日生，你以为是巧合？”
<br/>(本章完)
