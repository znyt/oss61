　　马特回头看见坚定跟着自己的苏重，他就觉得头疼。眼看和自己妹妹约定的时间快到了，他不能再耽搁。
<br/>　　“不管了，先去基地入口。”
<br/>　　和苏重这个小混混相比，马特更看重保护伞公司的黑材料。
<br/>　　“终于不再兜圈子啦。”苏重也长舒一口气，这个一个多小时转悠，他比马特还着急。要是不能到达地下找到疫苗，自己这具身体，说不好就要直接挂了。
<br/>　　马特不管苏重沉默疾走，不时注意周围环境，熟门熟路走向住宅区深处。
<br/>　　看着远处草坪中间那座别墅，马特长出一口气，还好赶上时间了。
<br/>　　“你真有眼光，这种房子，一看就油水充足。”冷不丁身后响起一道声音。
<br/>　　马特浑身一僵，他怎么把这个阴魂不散的家伙给忘了呢。
<br/>　　“我真不是小偷，我是世界维权公益组织成员，来这里是为了和线人接头，为的就是找到保护伞公司的黑幕，维护大众权益。你能不能别跟着我，这会让我暴露。”
<br/>　　“编，你接着编。反正我不信。”苏重面无表情。
<br/>　　马特：“…”特么说真话怎么还没人信了。
<br/>　　“快走！”苏重催促。
<br/>　　马特无奈：“你不能乱跑也不能乱动，这里布置的很多警报系统，一旦触发，咱们谁都跑不了。”
<br/>　　苏重斜睨着马特：“还说不是小偷，调查这么清楚，你不是入室盗窃是什么？！”
<br/>　　马特感觉已经无力分辨了。
<br/>　　“走吧。先进去再说。”他是管不了这个小混混怎么想了。他只要拿到资料，然后就拍拍屁股走人。到时候骗这个小混混去偷东西，随便触发点儿警报就能把这家伙甩掉。说不定还能给自己背锅。
<br/>　　马特想到这里，心情舒畅起来，站在门口道：“我告诉你哈，这家人格外富有，咱们也别五五分账了，你跟我一块进去，看中什么就拿什么，最后收成全凭个人本事，你说怎么样。”
<br/>　　“不怎么样，想让我留下指纹给你背锅？门都没有！你进去吧，我在外面等着。”苏重撇了一眼马特面无表情道。
<br/>　　马特：“……”现在小偷都这么谨慎的吗？有这脑子你干什么不行，非要当小偷！
<br/>　　马特摇头不管苏重，按计划翻窗户潜入房子。
<br/>　　刚进去，一道黑影猛然袭来，不等马特招架，后脖颈砰的重重挨了一下，脑袋一阵眩晕，还没来得及清醒，就被人反手铐了起来！
<br/>　　“姓名！”
<br/>　　“马特，马特·艾迪森。”
<br/>　　“干什么的！”
<br/>　　“入室盗窃。不是……是……”
<br/>　　“是什么！”一个黑光头瞪着两双牛眼，死死盯着马特。
<br/>　　马特：“……”这场景怎么莫名熟悉。
<br/>　　“是盗窃。”
<br/>　　“这家伙是你的同伙吗？”
<br/>　　然后马特就看到被人用枪指着，双手举过头顶的苏重。马特下意识的点头。
<br/>　　“不是，他是世界维权公益组织的成员，和我不是一路人。”苏重一脸嫌弃。
<br/>　　马特：“……”刚才你特么不信，这会儿怎么又信了，这不是坑人吗！
<br/>　　砰！
<br/>　　马特后脖子又挨了一下。
<br/>　　“来不及和他们扯皮了，我们必须尽快进入蜂巢，找到蜂巢关闭原因。”黑光头詹姆斯队长打断几人交谈道，他是保护伞公司派来的突击队，为了调查蜂巢异常而来。
<br/>　　“他们两个怎么办。”唯一女队员雷恩恶狠狠瞪了两人一眼问到：“要不要就地解决！”然后满脸恶意的吓唬马特。
<br/>　　“别闹了雷恩，带着他们一起。等解决完蜂巢事故，让公司法务部的人和他们谈判。”詹姆斯雷厉风行。
<br/>　　苏重舒一口气，刚才还在想怎么混进地下，现在不用了，直接就给带下去了。
<br/>　　咔嚓一声，苏重也得到了和马特相同的待遇，被铐上了手铐。
<br/>　　背着电脑包的卡普兰伸手结果苏重背包，结果一下没拿住，噗通一下掉在地上。
<br/>　　“什么东西这么重！”
<br/>　　“武器！”苏重一本正经。
<br/>　　咔嚓咔嚓！三杆枪同时指向苏重。
<br/>　　“这些钉子也算武器？”卡普兰拿出一包钢钉哭笑不得。
<br/>　　“你不懂，这是行业秘密。”苏重板着脸严肃道。
<br/>　　“快走吧，别听小偷先生胡扯了。卡普兰，把武器还给小偷先生，让他背着。”詹姆斯不屑的瞥了眼苏重，然后转头对一位大美女温和道：“爱丽丝主管，我们走吧。”
<br/>　　苏重不以为意，斜挎着双肩包，开始打量这位传奇女汉子。
<br/>　　他突然想到，这位女汉子，好像是保护伞公司某位高层的克隆体。生化病毒最初就是为了给那位高层治疗早衰疾病用的。
<br/>　　怪不得别人要么变成怪物要么没反应，偏偏就爱丽丝能够无损伤的完成强化。因为生化病毒完全就是给她量身定做的。
<br/>　　“那病毒血清到底还管不管用？”苏重突然想到了这个严重问题。要是不管用，自己可就死翘翘了。
<br/>　　跟着保护伞特种小队，打开密码门，进入轨道列车，几人呼啦啦冲向蜂巢基地。
<br/>　　苏重开始在车厢里溜达，他记得，好像解毒血清就在这节车厢壁橱内。里面似乎还藏了个人。
<br/>　　伸手拉开门，哗啦一声果然滚出一个人。
<br/>　　“这是谁？”詹姆斯拿枪指着昏迷在地人。
<br/>　　“我觉得是小偷。”苏重道。
<br/>　　“小偷先生，恐怕要让你失望了。他叫史宾斯，是保护伞公司保安人员，名义上是这位爱丽丝女士的丈夫，可不是你的同行。”卡普兰笑嘻嘻。
<br/>　　“那个箱子是怎么回事？他肯定是小偷。”苏重用下巴指向史宾斯身旁的手提密码箱。
<br/>　　“卡普兰，打开看看。”詹姆斯眉头紧皱。
<br/>　　小小密码箱，当然难不到电脑达人卡普兰：“这是什么？某种药剂？”
<br/>　　看着蓝绿相间的药剂，詹姆斯脸色难看。安保人员竟然出了问题。
<br/>　　“看吧，我就说他是小偷。”苏重满脸欣喜的盯着一管管绿色药剂，这可都是他的救命药！
<br/>　　列车很快到达目的地，走下列车，四周一片寂静，黑暗阴冷，让人忍不住汗毛倒竖。
<br/>　　“卡普兰，规划路线，我们需要去红后主机房关闭红后。”詹姆斯队长目标明确。
<br/>　　他们被雇佣过来，就是解决蜂巢异常关闭问题的。红后是蜂巢中控人工智能，只要关掉对方，就能重新开启蜂巢。
<br/>　　“这里阴森森的，赶紧完成任务回去，我可不想在这里多待。”彪悍女子雷恩警惕的打量四周。
<br/>　　苏重怜悯，还想回去，你们一个也回不去啦。
<br/>　　这几个人就是保护伞派来送死的。
