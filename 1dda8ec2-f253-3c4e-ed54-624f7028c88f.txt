　　看着杜天易的样子，涂小安嘴角微勾，眼中划过一丝玩味，含词轻吐道：“我杀的！”

<br/>　　这简单的三个字瞬间在杜天易的脑海中回荡，他的脸色阴沉一片，不好的预感愈发的强烈，他还是冷静道：“朋友，这话可不能乱说，杀死妖精管理局的组长，这是大罪，不单单是会被温城的管理局通缉，还会被国家颁布诛杀令，无论天涯海角，你都无藏身之处”
<br/>　　对方到底是太狂妄，对自己太自信，还是弱智无知，杀了妖精管理局的人，既然还敢公认的站出来承认。

<br/>　　“可就是我杀的有什么办法，明人不说暗话”涂小安浅笑吟吟的耸了耸肩膀。
<br/>　　杜天易真想骂娘，对方如此昭然若揭，是非逼着自己对他动手？
<br/>　　他总不能听到对方说杀了妖精管理局的人还无动于衷吧。
<br/>　　该死，杜天易有点骑虎难下，他是什么人，位高权重，妖精管理局局长之下的第一人，有生杀大权，杀死一个人，就跟捏死一只蝼蚁一样轻松。

<br/>　　不是他对自己的实力不自信，而且局势不在他这边。

<br/>　　另外在加上大厅中还有两只超凡脱俗的超等生物，虽然不过二阶，但要对付起来，也要费上一番手脚。
<br/>　　三人两兽，杜天易不是傻子，就算他是三阶生灵，在局势面前，也不得不低头。
<br/>　　这该死的牛妖王，既然成了别人的手下，这让他很尴尬，有种深入虎穴的感觉。
<br/>　　说好的狠狠的敲诈牛妖王一笔，现在弄的进退两难。
<br/>　　半晌...
<br/>　　“哈哈...牛妖王，你的这个主人真会开玩笑，果然是人中龙凤”杜天易抽搐下脸庞，哈哈大笑。
<br/>　　我特么堂堂妖精管理局的第一人，都这样做了，你差不多得了，不要得寸进尺，他心中咆哮。
<br/>　　涂小安笑眯眯的看着杜天易，他这是给自己台阶下，但这个台阶，涂小安还就不想下去。
<br/>　　浑身一震，他的体表外一股深蓝色气焰迸发出来，整个大厅的温度骤然下降，宛如寒冬腊月。
<br/>　　“这股力量，你应该不陌生吧”涂小安含笑的看着他。
<br/>　　杜天易脸色的假笑凝固，旋即愕然一片。
<br/>　　冰霜之力？
<br/>　　这是冷霜组长特有的元素冰霜属性能力，这怎么可能？？？
<br/>　　杜天易脸上的愕然之色变成了难以置信，惊奇得如五雷击顶，既说不出话，也没有力量。
<br/>　　就算冷霜是这个年轻男子杀的，但为什么冷霜的能力会转嫁到对方的身上，这怎么可能。
<br/>　　他是怎么做到的，他拥有什么诡异的手段？
<br/>　　能夺舍能力，这种事情闻所未闻，几乎只出现在小说中。
<br/>　　这个人神秘莫测，怪不得能成为牛妖王的主人，杜天易能看穿道士打扮的是妖灵化形，但那个年轻男子身上一点气息没有，让他没能看出深浅。
<br/>　　就是看不出来深浅，这点才是最可怕的。
<br/>　　最为糟糕的是，这年轻人分明对他有敌意，故意把事情放到明面上，就是逼着他杜天易不得不翻脸。
<br/>　　可这个脸，特么杜天易现在就不想翻。
<br/>　　他现在就感觉心中一万个草泥马在奔腾。
<br/>　　杜天易位高权重，什么时候受过这种鸟气，憋屈到不行。
<br/>　　他恨自己一个人来到这栋别墅，羊入虎口一样，更恨牛妖王不懂江湖道义。
<br/>　　你特么认了别人为主，也好歹跟我打一个招呼。
<br/>　　此时安静待在涂小安身边的牛妖王冲着脸色难看的杜天易摊了摊手，一脸的无辜：老牛也是身不由已。
<br/>　　他岂能想到，自己堂堂三阶妖灵会被一条二阶蛇精收为战宠。
<br/>　　他还想要一个解释呢。
<br/>　　“杜组长，你不准备动手抓我吗”涂小安见对方没反应，戏谑的看着他。
<br/>　　杜天易全身隐隐颤抖，双手握着拳头，对方这就是非逼着他动手，可越是如此有恃无恐，他就越不想撕破最后的脸面。
<br/>　　“哈哈...鄙人跟牛妖王是多年朋友，那个冷霜跟我虽是同事，但我们却向来不好对付，好比水火不容，杀的好，朋友你帮我出了一口恶气，杜某还要谢谢你”
<br/>　　咦...

<br/>　　涂小安闻言都迷了，一脸懵逼，他刚把窗户纸给撕下来，对方又给强行糊上去了。
<br/>　　这还妖精管理局的组长啊，一点血气放刚都没有。

<br/>　　忍，他不想吃亏，只能继续忍让，心中想着逃离之策。
<br/>　　杜天易发誓，要是让自己安然无恙的从这别墅离开，他一定带人夷平这里，将这些人都给抓回去，不然咽不下这口鸟气。

<br/>　　涂小安不咸不淡的开口。

<br/>　　跑，这场战斗完全没得打，打赢了二阶的生物，后面还有三阶妖灵等着他。
