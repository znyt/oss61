　　　　不管这只乌鸦是妖鸟，还是神鸟，涂小安二话不说，先一个意识洞察丢过去。
<br/>　　必须知己知彼才行。
<br/>　　锁妖枷虽然不凡，将涂小安的属性能力跟意识力全都束缚，但系统能力还是可以使用的。
<br/>　　只见涂小安的眼眸深深的看着这只老气纵横的大黑鸟，顿时间，这只乌鸦的一身黑毛泛起了一层深蓝色的光芒，好像被触碰到了什么禁忌似的。
<br/>　　“叮...扫描失败，且无法收服”
<br/>　　啥意思？
<br/>　　涂小安楞了下，还能扫描失败，是这只乌鸦的等级太高，还是逼格太高。
<br/>　　它口吐人言而未化形，说明只是二阶觉醒生物，不是三阶生灵，涂小安的洞察是不可能失败。
<br/>　　可现实就是他洞察扫描失败了。
<br/>　　至于逼格，那么就是血脉血统了，难不成这只乌鸦大黑鸟还能超凡脱俗了不成。
<br/>　　涂小安很震惊。
<br/>　　果然不愧是不祥之鸟，浑身上下透着浓郁的神秘气息，刚才洞察，涂小安就留意到它身上的深蓝色光芒，好像有某种不为人知的神秘力量，禁止别人窥探。
<br/>　　“你干嘛”
<br/>　　大黑鸟突然盯着涂小安警惕的问。
<br/>　　“我没干嘛呀”
<br/>　　涂小安眨了眨眼，天真烂漫的说。
<br/>　　偷窥是可耻的，他自然不会承认。
<br/>　　大黑鸟露出了高人一等的人性化表情：“蛇精，不错呀，被锁妖枷束缚，还能动用某种力量，对你黑大爷窥探”
<br/>　　黑大爷？
<br/>　　尼玛的是黑，但不是我大爷。
<br/>　　这只乌鸦不好惹话。
<br/>　　涂小安决心安分点，从来没有洞察别人，还会被发现的。
<br/>　　这让他很尴尬，就好像你拥有了透视眼，想将一位美女的衣服看透，但却被逮到了。
<br/>　　“说说你是怎么被抓的，说出来让我高兴下”
<br/>　　他机智的转移话题。
<br/>　　涂小安的被抓，是很冤枉也很悲凉，被人取笑也难免，你说你堂堂一条蛇精，不想着没事吞了人填饱肚子也就算了，还想着救人，救人跟你有一毛钱关系吗。
<br/>　　他每天最应该做的就是没事在心里默念三遍：我不是人，我是条蛇。
<br/>　　三省吾身，这点逼数要有的。
<br/>　　不过话说回来，大家都是被抓来蹲号子的，就谁也不要大傻子笑二傻子。
<br/>　　“哼...”
<br/>　　这大黑鸟很高冷的哼了一声，显然不愿意跟涂小安分享这段丢人的事迹。
<br/>　　“这妖鸟丧心病狂，居然冲着一个千人户的村子诅咒，诅咒那些妇女生不出儿子，照成那个村都快成了女儿国了”
<br/>　　一位路过的警员非常善良的帮涂小安解答了这段疑问。
<br/>　　闻言，涂小安猛然倒吸了一口凉气，随后目瞪口呆，撑着惊骇的眼神看着角落的大黑鸟，惊为天鸟。
<br/>　　那么牛逼？既然身负诅咒之力？
<br/>　　果然不愧是乌鸦呀，没埋没了它的品种。
<br/>　　生活中，你会经常听到一些骂人的话，比如我画个圈圈诅咒你，比如我诅咒你生儿子没**...等等。
<br/>　　但这种话也就骂骂而已，实现不了。
<br/>　　但这只大黑鸟不同，它直接诅咒你生孩子没小JJ，而且你还就生不出来了。
<br/>　　就问你服不服，怕不怕。
<br/>　　“傻吊，多管闲事，黑大爷的事情需要你说”
<br/>　　乌鸦一脸的鄙夷，视这些警员如粪土，逼格比涂小安高的多。
<br/>　　“妖鸟，我告诉你，你要是不解除了对那个村的诅咒，你这辈子就别想出去了”
<br/>　　那位警员恶狠狠的瞪了它一眼，不过显然对它有点忌讳，没上前动手。
<br/>　　“解除是不可能解除的，这辈子都不可能了，那些愚蠢的村民，让他们给黑大爷我建一个寺庙供奉，既然还不乐意，瞎了他们的狗眼”
<br/>　　大黑鸟傲娇的说，就应该再次诅咒，诅咒他们连女娃都生不出来。
<br/>　　妖怪的产生起初于远古人们对自然，动物的敬畏之心，人们把自己看不见、摸不着、无法控制的力量统统称为妖怪。
<br/>　　动物成精成妖，让很多人们把不少动物当成为了自己的图腾，崇拜跟信奉。
<br/>　　最为被世人认可的就是传说中的龙图腾，还有一些蛇图腾，再来就是狼图腾，牛图腾之类。
<br/>　　但凡信奉某动物图腾，地方人都会为其建筑寺庙供奉，香火不断，也禁止杀戮。
<br/>　　所以，只要那个地区信奉某动物为图腾，那么该动物在当地是非常的幸福，不会被惨遭恶意的杀戮，大量繁衍。
<br/>　　当然，这也有意外，一般来说对图腾要敬重，禁止伤害，但有时却有极其相反的情况。
<br/>　　曾经有个部落专门猎取图腾动物来吃，他们觉得图腾太完美了，吃了它，它的智慧、它的力量、它的勇气就会转移到自己身上来。
<br/>　　那该动物就悲剧了，在当地几乎绝种。
<br/>　　白镇就是一个已蛇图腾信奉的地区，涂小安这个蛇王当的也是顺风顺水，没有太多的波折跟坎坷。
<br/>　　但也不是什么动物都有那么好命，会被人类视为图腾的供奉。
<br/>　　比如这是乌鸦大黑鸟，它觉得自己的逼格很高，完全有资格让人们建筑寺庙供奉，可惜就是一直难以得偿所愿。
<br/>　　乌鸦这种鸟类，在古代人们对它或许还有一定的好感度，但在现代，就是人人嫌弃的不祥之物。
<br/>　　躲你都来不及，还想世人给你建筑寺庙供奉，美死你...
<br/>　　所以大黑鸟一气之下，就诅咒了一个村的妇女生孩子没小JJ，也被妖精管理局的人给抓了过来。
<br/>　　“哼...不解除诅咒，迟早拔了你的毛”警员瞪了乌鸦一眼，转身离开。
<br/>　　哎呀，这只大黑鸟也是个人才呀。
<br/>　　威逼不成，就降下诅咒，涂小安怔怔的看着旁边的黑大爷，充满了浓厚的兴趣。
<br/>　　大黑鸟冷笑一下：“黑大爷我可是一只神鸟，只怪世人太愚钝，不识货，连区区喜鹊都有人崇拜信奉，居然就不信奉本大爷”
<br/>　　它很不理解...有种天妒英才的感觉。
<br/>　　涂小安觉得它此时应该念一首跟自己逼格相近的诗，比如唐伯虎的，别人笑我太疯癫，我笑他人看不穿。
<br/>　　不过这只乌鸦想世人供奉，并不仅仅是为了一点虚荣心，它要的是人们的信仰之力。
<br/>　　涂小安的信仰之力直接被系统转化为了信仰点数充当货币。
<br/>　　但别的精怪要是有信仰之力，应该是可以提升实力的。
