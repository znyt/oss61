　　楚风眠的话，可所谓的嚣张至极，狂傲到了极点。
<br/>　　对新人来说，任何一尊人榜武者，都是庞然大物一般的存在，绝对是强者，不是这一届刚刚加入的新人可以对抗的。
<br/>　　武风云，更是人榜中的佼佼者，位列第十一位，只差一步，便是可以成为外门前十的人物。
<br/>　　这样的人物，已经是有着晋升内门弟子的实力。
<br/>　　如今楚风眠放言，居然是就连武风云都没有放在眼中，这等嚣张霸道。
<br/>　　“什么！楚风眠，你居然就连武风云师兄都敢辱骂！”
<br/>　　牛峰一愣，随后大声喝道。
<br/>　　“你是在找死不成！”
<br/>　　“找死的是你们！”
<br/>　　楚风眠一声惊喝，大手一挥，直接将那牛峰，马允二人，直接镇压。
<br/>　　“两个废物，敢来抢我的洞府，还敢出言威胁我？今日我就给你们一个教训，告诉你们，有的人不是你们可以招惹的起的！”
<br/>　　楚风眠毫不留情，大手一挥，直接拍了下去。
<br/>　　这一掌之下，直接打入这二人的背上。
<br/>　　“啊！”
<br/>　　牛峰，马允同时惨叫一声，身躯上的皮肤，猛然炸裂开来，无数圣者精血，随着楚风眠的力量之下，全部便是都被抽了出来。
<br/>　　足足是他们二人，九成的圣者精血，如今在楚风眠的手中，全部都被抽取了出来。
<br/>　　这些圣者精血，全部都是被楚风眠吞噬，融入到了远古杀神之中去，为他的杀戮剑术，再添一份力量。
<br/>　　被抽取了九成精血，牛峰，马允二人的脸色，苍白到极点。
<br/>　　他们毕竟都是圣者中的佼佼者，就算是被抽取了九成精血，也不至于会死，但是却也废了他们的半条命。
<br/>　　尤其是这九成圣者精血，更是将他们二人的大部分力量，全部都给夺走。
<br/>　　现在那二人身上的力量，可以说不足一成，资质彻底算是废了，不可能还恢复的过来。
<br/>　　从此之后，这二人便是只能作为圣者中的底层，最弱者，穷其一生都无法在进行任何突破。
<br/>　　在这北邙学院中，都再也没有晋升的可能，甚至是在外门，都再也没有任何的地位。
<br/>　　“这手段，好狠！”
<br/>　　“抽取了九成圣者精血，这二人，简直就成了废物，现在我们任何一个人，都足以轻易捏死他们。”
<br/>　　一些武者，看着这一幕，都是心有余悸。
<br/>　　尤其是那些，本来也是前来，打算要抢夺楚风眠洞府的武者，都是不由的庆幸，可算他们来得晚了。
<br/>　　不然这被抽取圣者精血的人，可就是他们了。
<br/>　　作为一代天才，雄心壮志加入北邙学院，若是直接被人废了资质，只能成为最弱的圣者，只怕是想死的心都会有了。
<br/>　　楚风眠的手段，实在是太狠，虽然没有杀了这二人，但是却将他们的力量全部抽走，更是直接废了他们二人的境界。
<br/>　　对他们这些一方天才，天之骄子来说，如此打击，简直是比起他们杀了他们，还要令他们痛苦的多。
<br/>　　“这楚风眠的手段，实在是太狠了，这一下，只怕是新人中，没人还敢招惹他了。”
<br/>　　“那是自然，这一届的新人王，肯定是归他莫属了。”
<br/>　　又一尊武者，开口道。
<br/>　　能够以一敌二，击败牛峰，马允二人，这等实力，肯定是这一届新人中的新人王了。
<br/>　　这一届的新人中，无人能敌。
<br/>　　“此人的实力，的确不错，但是性格实在是也是太狂傲了点，如此出手，只怕是风云会那边，不会善罢甘休啊。”
<br/>　　又一尊武者道。
<br/>　　“他既然敢骂武风云，武风云绝对不会放过他。”
<br/>　　“那又如何？这小子既然敢骂武风云，肯定是有着底牌才对，你看他对付牛峰，马允二人，似乎都没有动用全力。”
<br/>　　“哼，底牌，我看是这小子实在是太狂傲了，不知道人榜武者的可怕，在北邙学院中，只要是上了人榜的人，哪有那么容易对付……”
<br/>　　一声声议论声响起，楚风眠听在耳中，他却是毫不在意。
<br/>　　兵来将挡水来土掩，谁来对付楚风眠，楚风眠都不会任何惧怕。
<br/>　　楚风眠的双眼，看了一眼那牛峰，马允二人，冷喝一声道。
<br/>　　“还不快滚！”
<br/>　　“走！”
<br/>　　牛峰，马允的眼神中，都是露出了几分怨恨，看了楚风眠一眼，还是缓缓的互相搀扶起来，离开了。
<br/>　　“走，我们去禀告武风云师兄去！让武风云师兄为我们做主！”
<br/>　　“不错，此人实在是太过狂傲了，让武风云师兄来对付他，也要废了这小子！”
<br/>　　他们二人直接便是满心怨恨的离开。
<br/>　　“诸位，还不走吗？”
<br/>　　楚风眠看了一眼洞府外面还在聚集的武者，冷哼一声开口道。
<br/>　　“走，走。”
<br/>　　这些武者，这才纷纷离开，每个人离开的时候，都是目光闪烁，不知道是在思考这什么。
<br/>　　今日刚刚加入到北邙学院之中，便是就发生了这种事，对于如今局势的打破，众人的心中，都有着各自的想法。
<br/>　　解决了两个找事的蝼蚁，楚风眠自己倒是没什么感觉。
<br/>　　放他们一命，也只是因为这里是北邙学院，不好杀人。
<br/>　　从拓跋鼎哪里，楚风眠便是大约理解了北邙学院的规则，可以互相对战，但是不能闹出人命来。
<br/>　　只要不出现人命，一切都还好说，现在楚风眠刚刚进入北邙学院，也不好公然挑衅北邙学院的底线。
<br/>　　不过这也只是暂时的，只有等楚风眠在北邙学院中立足脚跟，有着足够的实力之后，这规矩，也就对楚风眠毫无用处了。
<br/>　　楚风眠回到洞府中，便是继续开始布置这禁制，阵法。
<br/>　　他所布置的无数禁制，阵法，就算是寻常生死境武者，都无法打破，准备好这一切后，楚风眠这才缓缓坐在了洞府之中，开始休息。
<br/>　　“不知道寒月漓那边如何了？”
<br/>　　楚风眠心中自言自语一句，脸色微微一笑道。
<br/>　　“她毕竟是跟随了一尊长老，只怕是比我这边要好得多吧，先天神体，在加上长老培养，只怕是实力会一日千里吧。”
<br/>　　“我也必须要尽快提升自己的实力才行了，免得追赶不上。”
<br/>　　楚风眠笑了笑道，便是坐在地上，安心的修炼起来。
