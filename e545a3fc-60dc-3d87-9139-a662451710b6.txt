<br/>    一个月之后。
<br/>    
<br/>        燕山现在正在迅速收服所有十万大山之中的势力，处处都是烽火狼烟，以前妖族的统治，就如同是一个个部落，这不是陈明想要的，陈明想要的，是燕山的统治方式。
<br/>    
<br/>        战争并没有结束，而是在继续，不过这一次并没有之前那么血腥，只要归降燕山，便可以留得性命，即便是妖王也不例外。
<br/>    
<br/>        诸位大妖王在这一个月之内，在燕山的赤潮蔓延至下已经死伤殆尽，妖族大君的陨落，同样告诉了所有的妖族，抵抗只有死路一条。、
<br/>    
<br/>        所有的大妖，都开始向着沙海在燕山的记名弟子安排下迁移，组建这些新任的燕山领主的势力。
<br/>    
<br/>        战火继续蔓延，燕山的战火并没有惊到周围的势力，因为现在整个大陆之上，战火都在不断地蔓延，王级的战场，厮杀的如火如荼，短兵相接，激烈非凡。
<br/>    
<br/>        各种可以让王级突破的禁地，秘境，一一开启，无论是魔宗，还是仙门正道，都是如此，他们在为最后的决战做准备。
<br/>    
<br/>        王级的战场蔓延至整个大陆，所有人都将目光凝聚在悟道榜上，这个榜单的争锋第一次变得激烈非凡了起来，变化速度之快让人瞠目结舌。
<br/>    
<br/>        群星仙门并没有将陈明的名字继续排在悟道榜上，毕竟以陈明的实力，死在他手下的大君都有两位了，还能算悟道境？
<br/>    
<br/>        群星老君人为操作，让陈明的名字离开了悟道榜，而新出现的悟道榜第一，是一个很多人都不熟悉的名字，李素衣。
<br/>    
<br/>        “这个李素衣到底是何人？”
<br/>    
<br/>        “没有听说过啊！”
<br/>    
<br/>        “曾经的入道榜第二好像就是他！”
<br/>    
<br/>        “那他这个实力增长也太可怕了吧！”
<br/>    
<br/>        就在这时，魔宗突然宣布，魔宗七劫之首战败于李素衣的刀下，李素衣，晋升魔宗大弟子。
<br/>    
<br/>        而更多关于李素衣的消息也传了出来，李素衣曾经是燕山山主门下三弟子，不过被魔宗大宗主强行掳走了，成为了魔宗麾下。
<br/>    
<br/>        又是燕山！
<br/>    
<br/>        怎么在这个风云变化之际，燕山的出场率怎么这么高？
<br/>    
<br/>        这个时候，所有人想起了魔宗大宗主说过的那一句话，“我取你一刀，你折我一刀，两不相欠！”
<br/>    
<br/>        他们终于是明白了这句话的意思，取那柄刀，叫做李素衣，折断的那柄刀，叫做解千愁。
<br/>    
<br/>        “燕山，到底是一个什么样的宗门，竟然有李素衣这般天骄！”
<br/>    
<br/>        “这般天骄又有什么用，还不是被魔宗带走了，燕山连屁都不敢放一个！”
<br/>    
<br/>        “不知道燕山对此有什么表示。”
<br/>    
<br/>        “还能有什么表示，带走李素衣的那位，可是魔宗的大宗主！”
<br/>    
<br/>        魔宗血海之前。
<br/>    
<br/>        李素衣在血海前磨刀，无疆走到李素衣的背后，淡淡道：“这血海魔刀，便是从这血海之中炼制出来的，血海魔刀，沉寂在血海之中一万三千年，而后吸收血狱邪念，成就仙器。”
<br/>    
<br/>        李素衣依旧是磨刀，并没有回头，过了一会儿，李素衣忽然道：“我听说，魔宗有很多人并不安分？他们是觉得我不配魔宗大弟子的身份吗？”
<br/>    
<br/>        无疆道：“家大了，什么样的人都有，你本来是仙门正道的弟子，刚加入我魔宗，就是魔宗大弟子，总有些人会不满。”
<br/>    
<br/>        李素衣道：“这样不满的人，便是对你不敬，你是魔宗大宗主，却是没有掌控整个魔宗！”
<br/>    
<br/>        无疆笑道：“魔宗几乎占据整个天下半壁江山，就像古仙门一般，他们何时真正有一个人能够一言九鼎的。”
<br/>    
<br/>        李素衣收起血海魔刀，背在背后血红色的刀棺之中，“给我一份所有不服的王级名单。”
<br/>    
<br/>        无疆挥手，一纸名单飘到李素衣的手中，“早就准备好了。”
<br/>    
<br/>        李素衣伸手接过名单，“正好突破到大君还需要一些食物，就是他们了。”
<br/>    
<br/>        无疆冷冷地笑，“他们在你的眼中，就是你的食物？”
<br/>    
<br/>        李素衣道：“所有人在我的眼中，都是食物，通向强大的食物！”
<br/>    
<br/>        无疆伸手拍向李素衣的肩膀，李素衣并没有躲，无疆的手落在李素衣的肩膀上，无疆看向李素衣，两双血红色的眸子对视，“如果说，所有人都只是食物，那么我呢？”
<br/>    
<br/>        无疆血红色的眸子正对李素衣的双眼，看不出李素衣的眼神有一丝变化，继续道：“你要仔细想想你的答案，我给你三年的时间去想清楚，如果让我不满意的话，我会杀了你的！”
<br/>    
<br/>        李素衣伸手将无疆的手强行拿了下来，朝着远方走去，“这个问题不需要思考，我现在就可以告诉你，你在我的眼中，是一块比较大，比较不好下口的食物。”
<br/>    
<br/>        李素衣的回答很快，甚至没有思考，他知道，无疆是这个大陆之上的最强者，同样的，面对弱者的时候，强者都很讲信用。强者对于弱者，没有必要毁约。毫无疑问，无疆是强者，而他李素衣，是弱者。
<br/>    
<br/>        那么，无疆说会杀了他，就一定会杀了他。
<br/>    
<br/>        李素衣继续前行，并没有回头，因为他坚信自己的答案是自己的心意，直抒胸臆，无论对错。自己的心意，哪里需要去管那些，魔修，只要四个字，逍遥自在。
<br/>    
<br/>        无疆看向李素衣前行的背影，他并没有动手，没有动手的原因也很简单，因为他很喜欢这个答案，无所顾忌，故而所向无敌，所向无敌，故而逍遥自在。
<br/>    
<br/>        李素衣的背影消失在他的眼中，他忽然哈哈大笑，“哈哈哈哈哈哈，和当年弑师时候的我，太像了！”
<br/>    
<br/>        “你可知道，若是你三年回答我，无论你回答我什么，我都会杀了你，因为那样的心，太乱了，还不够彻底入魔。”
<br/>    
<br/>        “这是一柄好刀，还没有出鞘，我就知道这是一柄好刀。”
<br/>    
<br/>        “你故意开口，让我给你名单，是因为你现在还不想直接对你师父所代表的正道出手吗？不过没事，你既然已经举起了屠刀，便是入了这苦海，既然已经入了苦海，便无需再计较那么多了，因为，苦海无边啊！”
<br/>    
<br/>        无疆转身，朝着血海播撒着解千愁的骨灰，骨灰在海面上落下，而后被血海中的血水浸湿，逐渐沉入粘稠的血海之中，“若是能够回头，何来无边二字？”
