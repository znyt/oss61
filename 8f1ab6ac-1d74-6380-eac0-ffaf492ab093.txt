　　古色古香的房间里，萧逸忽然醒了。
<br/>　　“咦，我没死？”这是他的第一个念头。
<br/>　　下一瞬间，他注意到自己正处于一个陌生的环境里。没有任何犹豫，抬手往身下一拍，准备跃起。
<br/>　　这是一个优秀杀手的本能反应。
<br/>　　手刚落下，剧烈的疼痛袭来，让他手臂一软。
<br/>　　我怎么会这么虚弱？
<br/>　　....
<br/>　　萧逸，华夏人，武学天才。
<br/>　　一位修炼出内力的形意拳宗师。
<br/>　　当那些无知的外国佬嘲讽着华夏武术只是花拳绣腿，只能用于观赏用途，毫无实战意义时...
<br/>　　他怒而出手，横扫世界各大地下拳坛，并创下三百连胜的奇迹记录。
<br/>　　对手不乏金腰带拳王，却从无一人能在他的拳头下撑过十招。
<br/>　　‘形意拳’三字，成为整个世界地下拳坛的禁忌与恐惧。
<br/>　　他还是个杀手，杀手榜排名第一，杀手界公认的无冕之王。
<br/>　　出道至今，任务成功率达到骇人的百分百。
<br/>　　想请他出手的人趋之若鹜，甚至不惜一切酬劳。
<br/>　　但，他有着杀手的职业道德，却也有着身为华夏人的一份底线和责任。
<br/>　　只要是对华夏不利的任务，一概不接。
<br/>　　他爱着他的祖国，敌视一切试图对华夏不利的势力和人物。
<br/>　　虽身在国外，却始终心系祖国。
<br/>　　当一个SSS级佣兵团无视他的警告，高调接受一份敌视华夏的秘密任务时....
<br/>　　他怒而出手，对这个名列世界佣兵团第一的团队展开了血腥的屠杀，仅仅一晚，全团队员无一幸免。
<br/>　　自此，华夏成为了雇佣兵的禁地。再臭名昭彰、凶名赫赫的佣兵，都不敢去华夏撒野。
<br/>　　当M国那些自视甚高的特工和密谍，打算潜入华夏窃密时....
<br/>　　他毫不犹豫地对这些特工进行了惨无人道的暗杀。
<br/>　　在那几位已经成为M国特工界传奇人物的特工齐齐死于非命后....
<br/>　　那些自视甚高的M国特工终于见识到他的强大与残忍。
<br/>　　自此，华夏成为了国外特工噤若寒蝉的恐怖之地。再训练有素、再出色的国外特工，都不敢踏入华夏半步。
<br/>　　理所当然地，他因此结下了数之不尽的仇家。
<br/>　　想杀他的人，多如牛毛。
<br/>　　在世界黑道悬赏榜上，他的追杀悬赏高得吓人，高居榜单第一。
<br/>　　然而，当一个个顶尖杀手、一个个传奇人物，在接了这项追杀任务，接二连三地人间蒸发后，再无人敢提起这项任务。
<br/>　　杀萧逸，成了黑道悬赏榜的一项禁忌。
<br/>　　悬赏空挂，却无人敢接。
<br/>　　数天前，他通过特殊途径收到一个消息。
<br/>　　一支国外的雇佣兵团队，伪装成考古学家，潜入华夏，在某遗迹出土了一件价值连城的文物。
<br/>　　并且，已经秘密运出了国外。
<br/>　　对此，萧逸自然不会坐视不管。
<br/>　　泱泱华夏重宝，岂能流落国外。
<br/>　　他当即追上了这支雇佣兵团队，并将他们击杀，将文物取回。
<br/>　　就在他准备潇洒离去时。
<br/>　　上百个手持枪械的人包围了他。
<br/>　　他一眼就认出了这些人，有的是M国精英特工，有的是凶名在外的雇佣兵，有的是顶尖杀手...
<br/>　　萧逸瞬间明白，这是一个阴谋。
<br/>　　一个针对自己的阴谋。
<br/>　　那支雇佣兵团队只是个幌子，为的就是引他出来。
<br/>　　多方势力联手，只是为了杀他。
<br/>　　任他再精明，在这多方联手布置的阴谋下，他还是中计了。
<br/>　　但，萧逸没有怕。
<br/>　　他有信心，拼着重伤的代价杀出重围，日后再慢慢找这群人算账不迟。
<br/>　　就在他杀出一条血路，准备逃离时。
<br/>　　伤口上的鲜血滴落到怀中华夏文物之内。
<br/>　　异变突发。
<br/>　　他如遭雷击，身体发麻，四肢无力。
<br/>　　他只能不甘地、眼睁睁地看着敌人狞笑着朝他走来。
<br/>　　没有人知道，萧逸身上一直有一枚威力强大的炸弹。
<br/>　　这颗炸弹，足以将方圆百米炸成灰烬。
<br/>　　哪怕是再简单的任务，萧逸都会带着这颗炸弹。
<br/>　　为的就是防止意外。
<br/>　　当他引爆炸弹的瞬间，敌人的狞笑变为了恐慌。
<br/>　　那剧烈的爆炸，将他以及所有敌人全部淹没。
<br/>　　萧逸语录，永远要有底牌，这样，哪怕自己死了，敌人也没有机会笑下去。
<br/>　　.....
<br/>　　“哈哈哈哈...”
<br/>　　萧逸不禁放声大笑。
<br/>　　值了，就算老子死了，能让你们所有人陪葬，够本了。
<br/>　　“额，好疼。”萧逸的笑声戛然而止，体内传来一阵虚弱之感。
<br/>　　“不对，我没死。”萧逸愣了愣，疑惑至极。
<br/>　　能感受到身体虚弱，证明自己没死！
<br/>　　下一秒，一股陌生的记忆涌入脑海....
<br/>　　“我草，老子竟然穿越了！”
<br/>　　哪怕萧逸早已看淡生死，经历过大风大浪。
<br/>　　此刻还是忍不住爆了句粗口。
<br/>　　这里已经不是地球了，而是一个名为炎龙大陆的异世界。
<br/>　　有些类似于地球的古代。
<br/>　　这个世界，强者为尊，武者横行。
<br/>　　只要拥有足够的实力，你便能凌驾于一切之上。
<br/>　　而自己之所以没有死，是因为穿越并附身到一个刚刚死去的家族子弟身上。
<br/>　　巧的是，这个家族子弟也叫萧逸。
<br/>　　这个萧逸，乃是紫云城三大家族之一，萧家，当代家主唯一的儿子。
<br/>　　只不过，他幼年丧母，父亲也在多年前神秘失踪。
<br/>　　年幼的他，由家族中一位长老抚养长大。
<br/>　　如今十六岁，却只有凡境一重的微末实力，远逊色于同龄人。顶着‘少家主’的头衔，却是整个家族的耻辱，终日受到欺凌。
<br/>　　在所有人眼中，他是个彻头彻尾的废物。
<br/>　　“同样叫萧逸，你咋混得这么惨！”萧逸撇了撇嘴。
<br/>　　至于他为什么会死，萧逸从记忆中得知...
<br/>　　数天前，他的未婚妻慕容娇儿约他到后山。
<br/>　　傻乎乎的他，还以为是美人相伴，约他幽会。
<br/>　　当他赴约时，才发现后山上等待他的，除了慕容娇儿外，还有他的表哥萧若寒。
<br/>　　而萧若寒，早就想除掉他。
<br/>　　在他不可置信且夹杂着恐惧和愤怒的目光中，慕容娇儿和萧若寒吻在了一起。
<br/>　　直到死前他才知道，原来表哥萧若寒早就与自己的未婚妻有私情。
<br/>　　萧若寒乃是萧家有名的天才，凡境七重武者，更是五长老之子。
<br/>　　结果可想而知，萧若寒一掌震碎了他的心脉，并将他推落山崖。
<br/>　　当他被人从山崖下救回时，已经奄奄一息，口不能言，眼不能睁。
<br/>　　在那严重的伤势下，身体和心理的双重折磨，他仅仅撑了三天，便在床上一命呜呼。
<br/>　　也是在这个时候，萧逸穿越并附身到已经死去的他身上。
<br/>　　“好歹毒的女子，好恶毒的表哥。”萧逸皱了皱眉头，眼中尽是冰冷。
<br/>　　“同样是死在阴谋之下，老子好歹让那群王八蛋全部陪葬了，你却死得那么窝囊。”萧逸叹了口气。
<br/>　　“罢了，既然占了你的身体，老子才得以重生，这仇，我便帮你报了。”
<br/>　　萧逸不喜欢欠别人人情，哪怕这个人情只是一个巧合，哪怕这个人已经死了。
<br/>　　心中想罢，萧逸再次整理了一下脑海中的陌生记忆。
<br/>　　这个世界的武者，有一种特殊的天赋，觉醒武魂。
<br/>　　觉醒的年纪越早，天赋越强，武魂也越厉害。
<br/>　　武魂又分很多种。
<br/>　　器武魂：刀、枪、剑、戟、箭、斧、、、；兽武魂：狮、虎、猿、蛇、、、乃至强大的妖兽武魂；植物系武魂：花、草、树、木、、、乃至珍稀而拥有特效的天材地宝。
<br/>　　当然，还有其他各种超乎人想象的奇异武魂。
<br/>　　总之，只要是这个世界拥有的事物，都能成为武者的武魂。
<br/>　　而武魂，从低到高，又分赤橙黄绿青蓝紫七阶。
<br/>　　级别越高，武魂自然就越强。
<br/>　　像赤阶武魂，大多是些寻常之物，比如普通的刀，猪狗牛羊等家畜，路边的野花杂草等等。
<br/>　　而高阶武魂，比如萧家有名的天才萧若寒，则是黄阶武魂火云蟒。一旦发动攻击，操纵火云蟒，火焰滔天，顷刻间便能将敌人吞噬并燃烧殆尽。
<br/>　　可以说，武魂是这个世界武者最最重要的东西。
<br/>　　它决定着武者的未来，修炼速度，攻击手段，战力强弱。
<br/>　　总之，觉醒出越厉害的武魂，代表着这个武者天赋越强，日后也越容易成为一方强者。
<br/>　　“武者，武魂。”萧逸嘴角咧出一道笑意，“有意思，虽然身死，但能来到这个世界，或许以后会很精彩。”
<br/>　　然而，下一秒，他嘴角的笑意瞬间定格，甚至是目瞪口呆。
<br/>　　因为，他稍稍感知了一下自己体内的武魂。
<br/>　　武魂，存在于武者体内丹田之旁。
<br/>　　而如今自己丹田之旁，正有一只像球一样的，呆呆傻傻的东西，浑身散发着微弱的赤色光芒。
<br/>　　赤阶武魂，控火兽。
<br/>　　“尼玛，这是在逗老子？”萧逸差点破口大骂。
<br/>　　控火兽，看这名字似乎很牛。
<br/>　　但实则，这是一种在荒郊野外最最低等的妖兽。
<br/>　　更严格来说，它是妖兽中的耻辱和废物，因为，它很可能连普通野兽都打不过。
<br/>　　整天呆呆傻傻，以兔子等弱小的动物为食。
<br/>　　哪怕是来一只凶恶点的残狼虎豹都能一口把它吞了。
<br/>　　甚至于，哪怕是一个没有修为的普通成年人，都能一脚把它给踢飞。
<br/>　　“我草，兄弟，你竟然觉醒出这玩意，难怪以前总是被人废物废物地骂着。”萧逸无语了。
<br/>　　控火兽，被誉为最垃圾的武魂。即使在最低等的赤阶武魂中，那也是垫底的存在，比猪狗牛羊等家畜武魂还要不如。
<br/>　　起码那些家畜武魂，还能给武者提供一定的力量增幅。
<br/>　　而控火兽，唯一赋予武者的，只有‘控火’的技能。
<br/>　　萧逸当场试验了一下，操纵体内武魂，掌心中猛然蹦出一小团火焰。
<br/>　　只是，这小团火焰，用萧逸前世的话来说，比打火机还不如，别说用来攻击了，用来烧火都嫌碍事。
<br/>　　别的武者，16岁起码有凡境三重的修为，他却只有最弱的凡境一重。
<br/>　　别的武者，像萧家中的子弟，普遍都是橙阶武魂，就是资质较差的，也是赤阶中比较有用的武魂。
<br/>　　他却觉醒出最没用的控火兽。
<br/>　　难怪他有着‘少家主’的头衔，却终日被家族子弟欺凌，被所有人所看不起。
<br/>　　武魂，几乎决定了武者一生的成就。
<br/>　　“算了，靠山山崩，靠人人倒。”萧逸抹去心中的不快，眼中尽是自傲之色，“凭我一身形意拳绝学，就算不靠武魂，也能在这世界闯下属于我自己的一片天。”
<br/>　　萧逸收回了掌心那团‘可怜’的火焰，懒得再看体内的武魂一眼。
<br/>　　恰在此时，他却猛地一窒。
<br/>　　“嗯？”萧逸皱了皱眉头，随即脸色大变。
<br/>　　因为，在体内丹田之旁，除了那弱小的控火兽武魂外；竟还有另外一团东西。
<br/>　　那东西，呈‘剑形’，浑身散发着耀眼的紫色光芒，而且还是深紫色。
<br/>　　“这...这...这是冰鸾剑，它..它怎么成为我的武魂了，而且还是紫色武魂。”萧逸惊喜交加。
<br/>　　冰鸾剑，不是这个世界的东西，而是让萧逸在地球死亡的罪魁祸首。
<br/>　　不错，萧逸当初要抢回的华夏文物，就是这把冰鸾剑。
<br/>　　“怎么会这样？”哪怕萧逸见识过人，却想不明白。
<br/>　　数天前，当萧逸收到消息，并出手前，曾经查过这件文物。
<br/>　　然而，以他这个杀手界无冕之王的能耐，却连冰鸾剑一丝一毫的消息都查不出。
<br/>　　只知道，这把剑的来历极为神秘，连历史记载都没有，反倒是留下了一些神话传说。
<br/>　　据说，这是一把上古时期，无数仙人大神都拼命争夺的宝物。
<br/>　　当时，萧逸还对这些神话传说嗤之以鼻；但现在，他却改变了想法。
<br/>　　“看来，你的来历果然不凡。而且，我能安然穿越到这个世界，重获新生，也肯定少不了你的功劳。”萧逸猜测着，但其实心中已经无比肯定。
<br/>　　“双生武魂，其中一个还是最强大的紫色武魂，啧啧。”萧逸原本郁闷的心情转变为一丝期待。
