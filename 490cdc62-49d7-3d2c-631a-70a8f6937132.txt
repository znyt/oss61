　　听了真贤道圣之言，苏莫和姜神夜均是眼眸闪亮，既然这神灵液有如此效果，他们当然不会拒绝。
<br/>　　嗖！嗖！
<br/>　　毫不犹豫，苏莫和姜神夜纷纷跳进了神灵液的水潭中。
<br/>　　进入水潭，身体被神灵液包裹，苏莫感觉到神灵之中，有一股奇异的力量开始进入体内。
<br/>　　这股力量柔和、温暖，舒适，进入体内之后，开始洗條他的身体。
<br/>　　“好舒服！”
<br/>　　苏莫轻轻的呼了一口气，闭上了眼眸，一边吸收神灵液，一边暗暗思索能不能带走一些。
<br/>　　不过，那真贤道圣居然没走，就在水潭盘坐了下来。
<br/>　　他不敢询问对方，因为对方是绝对不可能同意的，若是人人都可以带走，那这神灵液早都没有了。
<br/>　　神灵液的吸收非常的缓慢，进入身体之后，就开始融入血肉骨骼和玄力之中。
<br/>　　苏莫能清晰的感觉到，这神灵液的力量，不仅能洗條他的肉身，让得肉身更加的纯净，还能淬炼玄力，巩固神丹，效果奇佳。
<br/>　　最主要的是，就连体内的特殊血脉，都被洗礼净化，血脉之力缓缓的升华。
<br/>　　这神灵液仿佛能调理身体的本质，让得身体更加的适合修炼，更加的与天地灵气契合。
<br/>　　稍稍运转玄力，玄力在体内奔流，更加的顺畅自如，没有任何的隔阂。
<br/>　　“果然玄妙！”苏莫暗暗心惊，难怪说在神灵池洗礼之后，将来必成武圣，看来并不是多夸张。
<br/>　　经过长时间的洗礼，整个人都会升华，无论是肉身，还是修为玄力，都在无形中得到了升华与提升。
<br/>　　苏莫暗自思索，如何偷偷收取一些神灵液，日后带出去给苍穹神宫的人用。
<br/>　　就在此刻，只听真贤道圣开口了：“这神灵液你们能吸收多少是多少，自己吸收就好，也不用想着带出去，因为只要离开了万界山，神灵液就会变成普通的灵气，没有任何效果！”
<br/>　　苏莫闻言一怔，这神灵液离开万界山就会失去效用，这么神奇！
<br/>　　只不过，难道真贤道圣看出了他要打神灵液的主意，但下一刻，苏莫就知道怎么回事了。
<br/>　　只见一旁的姜神夜嘴角一抽，面带尴尬之色，道：“真贤师兄，我只是想弄点出去研究研究！”
<br/>　　苏莫无语了，他还没有动手，原来姜神夜已经动手了，但还被真贤道圣发现了。
<br/>　　“无妨，任何一个来到神灵池的弟子，都想要带点出去，这很正常！”真贤导师摇了摇头，并没有怪罪姜神夜，因为这是人之常情。
<br/>　　试问，谁没有一些亲人朋友呢？
<br/>　　神灵液有如此神效，谁不想带点给亲人朋友享用？
<br/>　　他当年还偷偷带出去了，结果出去之后，都变成了普通的灵气，没有半点效用。
<br/>　　姜神夜点了点头，随即不再打歪主意，静心吸收神灵液。
<br/>　　苏莫亦是放弃了收取神灵液的打算，无论真贤圣师所言的真假，但他想要瞒过对方，根本不太可能。
<br/>　　随即，他也安心的吸收神灵液，不再多想。
<br/>　　水潭边，真贤圣师目光转动，看了看苏莫，又望了望姜神夜，目中露出期待之色。
<br/>　　“不知道他们嫩能洗礼多长时间？”
<br/>　　真贤道圣暗暗猜测，这神灵液能洗礼的时间越长，才能真正的代表天赋。
<br/>　　可以说洗礼的时间越长，将来武道之路越是顺畅。
<br/>　　基本上任何人都能接受一日的洗礼，而只要洗礼了一日的时间，将来必成武圣。
<br/>　　洗礼时间达到两日，将来达到武圣境大圆满，不会有什么问题。
<br/>　　而洗礼的时间达到三日，将来晋升虚神境的机会极大，而时间若是超过三日，前途不可限量。
<br/>　　所以，真贤圣师很期待，期待苏莫和姜神夜能够洗礼多长的时间。
<br/>　　时间缓缓的流逝，苏莫和姜神夜盘坐在神灵池中，身形接受这神灵液的不断洗礼。
<br/>　　一日!
<br/>　　两日！
<br/>　　三日！
<br/>　　三日之后，姜神夜站了起来，随即走出了神灵池。
<br/>　　“真贤师兄，我的身体已经无法再吸收神灵液了！”姜神夜对真贤道圣说道。
<br/>　　“嗯！我们静等苏莫吧！“真贤圣师点了点头，对姜神夜非常满意，洗礼了近三日的时间，对方将来晋升虚神的希望极大。
<br/>　　同时，他对于苏莫的天赋，更加的好奇了起来，因为此刻，源源不断的神灵液，以及在不停的进入苏莫体内，完全没有停止的迹象。
<br/>　　姜神夜点了点头，便也盘膝而坐，等待苏莫。
<br/>　　他内视体内，暗暗欣喜，这神灵液果然不同凡响，他感觉整个人都得到了极大的升华，与这方天地，与天地灵气更加的契合，资质得到大幅度的提升。
<br/>　　时间继续流逝，又过去了一日，苏莫睁开了眼眸，他的身体已经无法再吸收丝毫的神灵液。
<br/>　　“真贤师兄！”苏莫走出了神灵池，向真贤道圣微微抱拳。
<br/>　　“好！很好！”真贤道圣面带微笑，连连点头，苏莫接受了四日的神灵液洗礼，将来基本上必成虚神，这种资质非常不凡了。
<br/>　　当然，这种事情，他并没有告知苏莫和姜神夜，因为修炼之路变化万千，每个人日后的成就，都不是绝对的。
<br/>　　“你们现在已经得到了奖励，等会就该离开万界山了，有些事情，本道还要嘱咐你们。”真贤道圣面色郑重的说道。
<br/>　　“师兄请说！”
<br/>　　“第一，关于万界山的一切，你们出去之后，严禁向任何人述说，违者必死；第二，无论你们是什么身份，是什么背景，倘若日后敢背叛万界山，就算修为达到了虚神境，同样是必死！你们可明白？”真贤道圣严肃的说道。
<br/>　　“明白！”苏莫和姜神夜点了点头，心中暗凛，就算修为达到了虚神境都要必死，难道万界山真有这么恐怖？
<br/>　　“嗯！”真贤道圣微微颌首。
<br/>　　“师兄，我们万界山的界主，是什么修为境界？”姜神夜好奇的问道，这个问题他一直就想问了。
<br/>　　“界主？”
<br/>　　真贤导师闻言，眸中充满崇敬之色，道：“界主功参造化，已然通神，修为已经不是我等能揣摩的地步了！”
<br/>　　已然通神？
<br/>　　苏莫和姜神夜闻言一怔，这是什么意思？难道已经达到了虚神境的巅峰？
