　　昆仑山中，三清圣地。
<br/>　　有三座巍峨浩大的宫殿群，三座宫殿群分布在这昆仑的三个地方，三者相互连接，遥首相望，形成一个有机的整体。
<br/>　　三者之间，一个巨大的阵法笼罩其中，将三者守护起来。阵法之中，烟云缭绕，氤氲生烟，仙鹤飞舞，仙音缥缈，树木丛生，百草丰茂，灵气横溢，仙光闪闪。
<br/>　　好一座仙家福地！
<br/>　　其中，位于昆仑之东，三宫之左的一座宫殿群，是三座宫殿群中最为庄严隆重的一座，宫殿碧丽辉煌，霞光绽放，瑞气冲宵，乃三宫之首。
<br/>　　宫名为玉虚宫！
<br/>　　玉虚宫的一座宫殿之中，有一身穿白袍，头插白玉簪的中年道人，他盘腿坐于虚空，一头黑发随意披散，身形好似一座巍峨大山，让人看了不可高攀。
<br/>　　此刻，他闭着双目，全身散发着一种极为浩瀚磅礴的气息，这气息堪比大海一般宽阔，如若星空一般浩瀚，形同宇宙一般磅礴。
<br/>　　他好似化身成为一片绚烂的星空，照耀了整片宽阔大地。
<br/>　　他，就是元始。
<br/>　　此刻，元始盘立虚空之中，感受着一种比不周山虚空更为浩瀚的空间。一种难以言喻的愉悦和感悟在他意念之中环绕。
<br/>　　身旁，悬浮于虚空之中的一颗颗细小的微尘，变得越来越小，到了后面甚至连微尘都可能会消失，而同样的，这微尘在他眼中却又同时变得越来越大，到了最后，或者会与一颗星辰一般浩大。
<br/>　　微尘不以小而小，不以大而大。就和这浩瀚无边，恢宏而不见边际的虚空一般，你不知道这宇宙是大还是小。
<br/>　　说它小，是因为它与亘古苍茫的混沌海相比，他的确是小到不可明见，说它大，它对于在这宇宙之中生存的天地万物来说，它的确是大到无边无际。
<br/>　　元始忽然有所明悟，在他眼中，这宇宙或者漫漫无边。但当有一天他的修为达到一定境界之后，这看似浩瀚无边的宇宙虚空在他眼中，也会与他眼前的微尘一般，变得微小无比。
<br/>　　若是这浩瀚宇宙在他眼里只有一颗微尘后，那这一颗微尘之外的世界又是怎样的世界？
<br/>　　这一刻，元始放松一切，完全身无外物，精神意志已经与这浩瀚的宇宙融为一体，似乎想要将这无边无际的浩瀚也融入到他的大道之中。感悟这种浩瀚无边，融入到自己的道韵之中。
<br/>　　无数的道念随之而来，融入到他的道韵之中。脑海之中，这些年游历洪荒、感悟星空、感悟宇宙、幽冥界一行……各种各样的感悟随之而来，他的气息变得越来越磅礴浩大。
<br/>　　随着时间的流逝，眨眼间，洪荒已过数千年。
<br/>　　在这数千年时间之中，元始一直默默无闻，盘立虚空，保持这个姿态未曾动过一分。但没人看过后会觉得他这样不妥，仿佛一切本该如此。
<br/>　　下一刻，元始身上气势忽然在不断上升，一种浩瀚到无边无际的气息从元始山上扩散开来。无穷无尽的道韵气息慢慢的在他周身环绕，在这一刻，若是有人看见他，那么他一定会将元始看做是整个宇宙，是整片虚空。
<br/>　　突然，他的周身虚空之中升起一个浩大的黑白阴阳图，阴阳二气流转，扩大到整片虚空，甚至后来扩大到这个三清圣地，整个昆仑山。
<br/>　　这样的异象，自然是吸引了无数人的眼光，人们纷纷猜测这是不是有什么逆天的宝物现世了，他们纷纷向着昆仑山赶来。
<br/>　　“那是什么？宝物吗？”
<br/>　　“那是什么？”
<br/>　　“有宝物出世，快走！”
<br/>　　……
<br/>　　此时，无数修士纷纷向着昆仑山赶来。
<br/>　　就连正在修炼之中的太清与上清，都被这浩大的异象惊醒。
<br/>　　“是大兄？这是宝物，还是道法？这股阴阳大道太深奥了，比我的还强？”
<br/>　　太清宫之中，老子被这浩瀚的阴阳图惊醒，感受到这其中浩瀚的阴阳道韵，轻声感叹道。
<br/>　　“大兄实力又增强了？”
<br/>　　上清宫中，通天亦感叹道。
<br/>　　同时，昆仑周围的无数生灵都被惊醒，纷纷向着三清圣地赶来，只是，这三清圣地早已被元始三人布下了无数大阵，他们能否进来还是两说，就算进来，能否活着亦是一个问题。
<br/>　　他们注定不能进来一观惊天“宝物”的样子了。
<br/>　　元始并不知道他修炼的异象已经掀起了如此大的动静，吸引了无数人的眼光，他此刻正一心一意地沉浸在修炼之中，没有丝毫的心思再去管其他的。他就算知道了，亦会毫不在意。
<br/>　　在元始的识海之中，他的元神世界已经形成了一个巨大的黑白蛋，大蛋在缓缓旋转，阴阳之气弥散开来，弥漫道整片元神空间，甚至整片玉虚宫，整片昆仑山。
<br/>　　但更多的阴阳之气却是与元始融入到他的身体之中，每一寸血肉，每一寸经脉之中，都有阴阳的道韵产生。
<br/>　　“咔擦——”
<br/>　　不知过了多久，突然，那巨大的黑白阴阳蛋随着“咔擦”一声，从中分裂成两半，白色阳气上升为天，黑色阴气下降为地。天地相互交织，环绕，阴阳元神化为一片无边无际的虚空。
<br/>　　同时，元始之前融入的那些大道法则，神通道法，在这一刻，在这浩瀚的虚空之中，形成一颗颗巨大的星辰，青色的星辰代表木系法则，赤色代表火系法则，蓝色的水系，黑暗，光明，毁灭，吞噬，黑洞……
<br/>　　其中，最大的两颗星辰，分别代表着太阳星与太阴星，它们是至阳与至阴法则所化。
<br/>　　各种各样的星辰在元始的识海之中快速形成，星空瞬间变得多姿多彩，璀璨无边。
<br/>　　这一刻，元始发现，他的识海完全变样了，这哪里是识海啊？这简直就是一个无边无际的星空。
<br/>　　无边无际，从清楚到模糊，却再也没有边界的禁锢。就好像是他的神识可以无穷无尽的扩展，终究有一天，他的识海可以与浩瀚的星空融合起来，整个星空都是他的神识范围。
<br/>　　之前的元神世界终究是有界域边界的，有边界的禁锢，就算元神再大，亦有了限制。如今边界已经消失，却是如同打破禁锢，挣脱束缚，变得正真的无边无际。
<br/>　　有界的界域化为无边星空虚空，这就是一个巨大的进步。
<br/>　　此刻，他的元神更甚至与肉身相融合，阴阳元神有很大一部分融入到肉身之中，使得他的肉身变得更加强大。一个个细胞空间之中，神窍空间之中，都多了那么一丝丝神性，使得空间更加神奇了。
<br/>　　也许有一天，这些空间都蜕变成了世界，那么，这一丝丝神性便可能上升为这些世界的天道。
