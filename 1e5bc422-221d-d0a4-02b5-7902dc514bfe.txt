　　大帝出手，天崩地裂！
<br/>　　七位至尊联手攻击，自古以来，前所未有。
<br/>　　“轰隆隆！”
<br/>　　天崩地裂，虚空坍塌，无尽法则光华爆射，光辉闪耀，爆发出来，比一颗太阳还要璀璨耀眼。
<br/>　　七位至尊联手攻击，真的是逆天了，不能并存的道合在一起，他们之间发生剧烈爆炸。
<br/>　　每一个至尊都立身于一片星空之中，被一挂挂星河缭绕，每一个都是宇宙的主宰者，一个人堪比一片真正的宇宙，他们用帝兵将这里打通。
<br/>　　古之大帝的人物大作战一切都是为了成仙，到了这一刻，几乎都没有保留，激烈向前搏杀。
<br/>　　数十万的虚影出现，疑似仙影，像是羽化飞仙般，铺天盖地向他们而来，阻挡他们！
<br/>　　那是一片光雨，化作了永恒，摧毁一切，大圣级别的强者，只要被光雨触及，必然形神俱灭，这是专门为了阻挡古皇而生的法则。
<br/>　　七位至尊各展身手，崩裂法则，毁天灭地。然而，这七位至尊的出身更让人震惊。
<br/>　　开创了九秘之一的者字诀的长生天尊，自神话时代经历了太古时代、荒古时代一直活到如今，他经历了帝尊、不死天皇、神皇、狠人大帝、无始大帝的时代，一直活到了现在，而有的大帝却是人族的古之大帝。
<br/>　　“苟延残喘至今，为了一场虚无缥缈的成仙，真的值得吗？”域外，众多轮回者震撼，仅仅出现的至尊就有这些，还有许多没有出现的，那又该如何？
<br/>　　“即便杀入仙域，也永世不能成为真正的帝与皇，这般惨重的代价，只为了验证成仙之法，这到底值不值？”众多轮回者有的都还年轻，不能体会那种感觉。
<br/>　　“仙又何尝不是人修炼而来，他们从自斩道基那一步开始，便已经与仙绝路了。仙又怎会是一个自斩的至尊？”青帝讽刺道。
<br/>　　“这是一场注定失败的成仙路！”众多轮回者点头道，同时心中也在暗自决定，定要好好利用自己的机缘，成就真正的仙。
<br/>　　百万神魔，千万仙灵，全部轰杀，根本就不能阻挡七位至尊，他们太强大了。
<br/>　　不死山石皇手中黑戟轰出，震动八方，直接将那城门轰碎，其他至尊也出手，将整座阻拦在仙路前的雄关生生轰碎，震撼世间。
<br/>　　七位至尊联手轰杀进去，不管遇到什么，全部轰杀，无可匹敌，神挡杀神，佛挡杀佛。
<br/>　　“路呢？”
<br/>　　“路在哪里？”
<br/>　　雄关粉碎，巨城已平，但是……前方已经无路，成仙路断了。
<br/>　　七位至尊看上去悲凉不已，空具一身盖世修为，却无用武之地。
<br/>　　就在此时，一身钟鸣，随后有霞光飞起，在那虚无的尽头，有一方庞大的世界若隐若现一股长生的力量吸引着众人。
<br/>　　七位至尊的眼眸一下子亮了，他们找不到路，路断了，便自己打出一条。
<br/>　　“没有路，那就打出一条路！”
<br/>　　七位至尊动手，继续轰杀向前，一条条璀璨的大道形成，由大道符文构成，直接通向远方。
<br/>　　域外星空之中，混沌道君的眼眸一直在闪烁不断，外面过去一秒，在他的世界内，已经过去了数十年，这些至尊在交手，是他们大道显化最为激烈的时候，数个呼吸，道君便能够领悟许多大道，这都是众至尊的大道。
<br/>　　每个至尊，都是能够镇压时代的主角，他们的大道，已经衍生到了此界巅峰，代表着一个极致，值得道君学习。
<br/>　　另一边洪荒之中，元始本尊亦在参悟着这其中的大道，虽然他们修为没有达到混元，但大道并不算弱。他的元始大道不论是什么，都来之不拒。
<br/>　　闯过无尽界海，七位至尊终于来到对面，古路尽头，是一对大门，门上血淋淋的大字，是用帝血所书。
<br/>　　一扇门上写着：过去！
<br/>　　另一扇门上写着：未来！
<br/>　　最上方一个匾额，也是两个字：现在！
<br/>　　七位至尊向里面闯去，可惜，都被轰了出来，他们明显是不符合。
<br/>　　这时，七位至尊开启北斗法阵，一时间，中州、北原、南岭、西漠、东荒全部在发光，形成一座五色祭坛，而后演变，成为一座巨大法阵，吸引万界力量，轰击仙门。
<br/>　　“就没有其他至尊动手吗，禁区内的至尊恐怕远不止如此吧！”叶梵问道。
<br/>　　“怎么可能只有这几位至尊？其他至尊都谨慎的很，不到最后一刻，验证成仙路的真假，他们又岂会出现？”青帝冷笑一声，道。
<br/>　　“这样么，那道便宜他们了，等先除去这几位至尊，再去禁区灭杀其他至尊。”
<br/>　　众多轮回者不急，此时的至尊们受伤越重，对他们越有利。
<br/>　　“阿弥陀佛！”
<br/>　　突然，一声佛号震荡天地。
<br/>　　须弥山上，无量信仰之力如怒涛汹涌，一尊大佛盘坐山巅，身为通天彻地。
<br/>　　无量信仰之力汹涌澎湃，比海洋还要浩瀚。
<br/>　　“我若成仙，诸天万界，无量众生，得无量光，无量寿，无量福……”
<br/>　　浩荡的誓愿声传遍整个宇宙。
<br/>　　在这一刻，星空彼岸，宇宙边荒，但凡佛教信仰之地，无尽信仰之力凝聚而来，铺天盖地。
<br/>　　如同漫天光雨，信仰之力自宇宙各个角落汇聚，落在须弥山上光雨无数，漫天飞洒。
<br/>　　“我佛慈悲，渡尽一切苦厄，普度众生脱离苦海，尔等可愿追随？”
<br/>　　“阿弥陀佛！”
<br/>　　亿万信徒，众口一声，口诵佛号，举步踏入须弥山。
<br/>　　“轰隆！”
<br/>　　须弥山拔地而起，散发出滔天佛光，携带着西漠亿万佛门信徒，呼啸着冲进仙路。
<br/>　　“那是阿弥陀佛？”另一边，轮回者们吃惊不已。
<br/>　　“善哉，那只是阿弥陀佛的信仰之身。”释迦牟尼开口道，他脸上带着无尽慈悲，看着阿弥陀佛带着意外佛众冲进仙路，心中不忍亿万佛众无辜牺牲。
<br/>　　亿万佛众大多都是普通人，成仙路上只要随便一震，便会死去无数，佛家的慈悲，阿弥陀佛为了成仙，早已不顾一切了。
<br/>　　“哀哉！”释迦牟尼痛呼一声，从某种意义上来说，他也是阿弥陀佛，只不过是帝尸，那信仰之身也不是真正的阿弥陀佛，乃是信仰之力所生出的神邸。
<br/>　　仙门被轰开，众多至尊全部杀了进去，冲入混沌古洞，消失不见。
<br/>　　突然，就在这一刻，宇宙颤抖一声，这天地溢出一缕缕掀起，神秘莫测，让所有人心头一颤。
<br/>　　“我有种预感，他们不是成仙了，而是要遭遇不测，宇宙将发生莫大变故。”
<br/>　　“这个世界怎么了，不知为何，让我心惊肉跳，浑身寒毛倒立，要发生什么吗？”一位准帝惊悚。
<br/>　　另一边，诸多轮回者们纷纷严阵以待，各自将手中兵器那好，准备战斗。
<br/>　　“准备好了，需要我们出手的时刻到了，记住，不能让任何至尊逃了出去，否则定然会发动黑暗动乱，生灵涂炭，到时候他们的实力便会恢复，我等要对付便更加困难了。”青帝嘱咐道。
<br/>　　“放心吧，我等定要一会古帝之锋芒！”一众轮回者跃跃欲试。
<br/>　　“轰！”
<br/>　　青帝突然将荒塔打入北斗，荒塔爆发出璀璨光辉，化成一座巨大当初虚幻之塔，将整个北斗笼罩在巨塔之内。
<br/>　　整个北斗瞬间被封了，外人不能入，北斗之人更是不能出。
<br/>　　一场大战，就此展开！
