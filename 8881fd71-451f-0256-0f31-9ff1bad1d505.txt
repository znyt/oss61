　　不过，最后苏莫还是选择了魔云海。
<br/>　　因为魔云海的魔甲族非常的多，而且实力并不是太强，更没有武尊境的魔甲族，连高阶武帝的魔甲族都寥寥无几。
<br/>　　最主要的是，魔甲族均是拥有强大的魔魂。
<br/>　　苏莫虽然没有吞噬过魔魂，但想必那魔魂应该不会比兽魂差。
<br/>　　至于和他人结伴，等到了魔云海之后，再分开便是。
<br/>　　选择好了试炼之地，苏莫立刻走出了阁楼，向南苑万通大殿走去。
<br/>　　万通大殿，是通向各个试炼之地的地方，其内拥有传送大阵，或者是空间通道。
<br/>　　这些，神武录中都有记载。
<br/>　　没过多久，苏莫便来到了万通大殿。
<br/>　　这是一座青色的石殿，占地千丈，高达三百余丈，古朴大气。
<br/>　　苏莫走进万通大殿，发现殿中刻画着数十个小型的传送阵法，每一个阵法之中，都刻有目的地的名称。
<br/>　　除此之外，还有数十个悬浮在空的光门，每一个光门之下，都竖立着一块石碑，石碑上同样书写着光门通往的目的地。
<br/>　　这个万通大殿的情形，和当初天灵宗的乾坤大殿有些类似。
<br/>　　看到那些光门，苏莫知道那定是通往一个个小世界，不由的他又想起了苍穹世界。
<br/>　　这些小世界的命运，估计和苍穹世界差不多，是神武学府弟子的试炼之地。
<br/>　　这让他不禁心中感慨，他为苍穹世界感到悲哀，也为这些小世界感到悲哀。
<br/>　　弱小地方的武者，只能被强者奴役，无法改变。
<br/>　　当然，这只是苏莫的一些感触，说不定这些光门通向的小世界，并没有人类也说不一定。
<br/>　　此时，大殿中除了两名学府的导师之外，有着数十名南苑弟子，或三两扎堆，或单独一人，分部在整个大殿之中，嘈杂的声音此起彼伏。
<br/>　　苏莫的到来，顿时引起了所有人的注意，所有的目光都全部向他望了过去。
<br/>　　当然，这倒不时别人多么重视苏莫，而是众人都在选择队友。
<br/>　　有弟子到来，众人自然要查看一番。
<br/>　　不过，看到苏莫之后，几乎所有人都失去了兴趣，虽然他拒绝火元圣师一事，已经在外府传开了，但他低弱的修为，根本不是众人选择的对象。
<br/>　　苏莫扫视了众人一眼，便在大殿中逛了起来，很快他便发现了通向魔云海的传送阵。
<br/>　　此时，在通往魔云海的大阵周围，有三名弟子。
<br/>　　这三人为二男一女，女子容貌颇为娇美，身穿翡翠色长裙，皮肤白嫩，身材丰腴，凸凹有致，非常的惹火。
<br/>　　而那两名男子，其中一人面色黝黑，身形壮硕，显然憨厚老实。
<br/>　　而另一人身材高大，相貌堂堂，器宇轩昂，整个人显得气度非凡。
<br/>　　“三位，你们是不是要去魔云海，我们结伴如何？”苏莫扫视了一眼三人，沉声说道。
<br/>　　他发现这三人的修为都不高，女子是武帝境一重的修为，两名青年男子都是武帝境二重的修为。
<br/>　　不过，那名气度不凡的男子，气息更强，貌似修为达到了武帝境二重的巅峰。
<br/>　　“就你？”
<br/>　　高大青年闻言，瞥了苏莫一眼，顿时嗤笑出声，道：“你这么废，到了魔云海，是要我们保护你吗？”
<br/>　　青年满脸的不屑之色，他们三人都是武帝境界，岂会和苏莫这种准帝境界的人组队！
<br/>　　苏莫闻言脸色一沉，不组队就不组队，对方的言语未免太侮辱人了吧！
<br/>　　“阁下未免太狂妄了吧！”苏莫冷冷的说道。
<br/>　　“嗯？”
<br/>　　高大青年闻言，眸光顿时一凝，眸中闪过一道寒光，冷冷的道：“你真以为有火元圣师撑腰，就能无法无天了吗？”
<br/>　　上次广场授课之时，高大青年也在广场，所以知道苏莫就是传言中拒绝火元圣师的人。
<br/>　　苏莫心中冷笑，他和火元圣师可不熟，从来也没想过让对方撑腰，他正欲再次开口，另一名面色略显黝黑的青年却先开口了，
<br/>　　“廖兄息怒，我看这位兄弟气息深邃，就算没有到武帝境，想必实力也不弱，我们就一起吧！”
<br/>　　黝黑青年从中打圆场，不想苏莫两人闹的太僵，而打搅了这次的历练。
<br/>　　“聂乾，魔云海极为危险，你不怕他拖累我们后腿？”高大青年转头向黝黑青年问道。
<br/>　　他们去魔云海，起码也要找武帝境以上的队友吧！不说能给予他们多大的助力，但最起码不能拖后腿。
<br/>　　“这……！”黝黑青年聂乾闻言一滞，随即转头问向丰腴女子，道：“韩盈姑娘，你觉得呢？”
<br/>　　丰腴女子沉吟了一番，柔声道：“那就让他一起吧！反正我们已经等了一个时辰，都没有其他人去魔云海，多一个人就多一分力量！”
<br/>　　“不错！四个人总比三个人强，廖兄，你觉得如何？”聂乾点了点头，又征求高大青年的意见。
<br/>　　高大青年见聂乾和韩盈均是同意苏莫加入，顿时大感脸上无光，眸中冷芒一闪而逝。
<br/>　　扫了苏莫三人一眼，高大青年心中冷笑，面上却是没有丝毫表情的点了点头，道：“既然如此，那我们就走吧！”
<br/>　　“好！那我去请导师开启阵法吧！”聂乾点了点头，随即转身去找大殿中的导师。
<br/>　　苏莫静静的站在一旁，眼角余光瞥了一眼高大青年，眉头不禁微微一皱。
<br/>　　此人如此狂妄，居然没有丝毫多言，便同意让他加入，不会中途使什么幺蛾子吧？
<br/>　　不过，他也没有丝毫惧意，如今他修为达到准帝之境，实力再次大增，就算对方是神武学府的超级天才，也不一定是他的对手。
<br/>　　很快，聂乾带着一名中年导师来了！
<br/>　　苏莫几人立刻全部走进了阵法之中，等待对方开启传送阵法。
<br/>　　“进入魔云海，随时都会遭遇危险，若是无法应对，你们切记要立刻返回！”中年导师面色郑重的向几人纷纷道。
<br/>　　“是！”四人点了点头。
<br/>　　中年导师微微颌首，随即大掌向阵法凌空一按，掌中玄力如天河倾泄，疯狂的涌入阵法之中。
<br/>　　霎时之间，这小型的传送阵法便立刻运转了起来，爆发出冲天的光芒。
<br/>　　苏莫见此微微意外，他没想到这名导师，居然直接用体内玄力催动传送阵，当真厉害！
<br/>　　嗡嗡~~
<br/>　　空间剧烈波动了起来，两个呼吸之后，阵法之光缓缓消散，苏莫四人的身影消散无踪。
