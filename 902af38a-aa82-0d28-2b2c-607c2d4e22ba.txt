　　第286章
<br/>　　蛤蟆岛腹心出现一座堡垒，垒以高墙，筑以箭楼，将山寨的聚义厅、医馆、民居包裹其中。
<br/>　　此时四方箭楼上飞箭如雨，劈头盖脸向楼下围攻的贼人射去。
<br/>　　程大雷立在箭楼上，发现这些贼人的临敌反应根本无法和戎族相比，挨了好几阵箭雨，死了诸多人后，才想起寻找庇护所。
<br/>　　不过，今日的堡垒也没办法和青牛山的城门相比，久守必失。
<br/>　　望着这一幕，程大雷发动了点将技能。
<br/>　　脑海中的点将台图案浮起，篝火明亮，一个个人影从篝火中走出。
<br/>　　嘟，获得普通游侠……
<br/>　　嘟，获得糟糕铁匠……
<br/>　　嘟，获得优秀武将……
<br/>　　……
<br/>　　一连串的系统提示，数据流疯狂涌入程大雷脑海，一条系统提示来不及显示一秒，便会被下一条系统提示取代。
<br/>　　青牛山一战，程大雷水淹十万大军，得到了堪称恐怖的恐惧值。只不过，在失去山寨之后，他的恐惧值就被冻结，也只有现在，二级山寨建成，点将台功能开启，这些被冻结的恐惧值才能再次被激活。
<br/>　　每一万兑换糟糕至普通人物1，每十万兑换优秀人物1，每百万兑换顶级人物1，每千万兑换绝世人物1 。
<br/>　　只不过，绝世等级的人物需要三级山寨才能召唤，目前程大雷只局限在普通到顶级这三个等级。他也并没有召唤顶级人物，樊梨花的事情提醒他，顶级人物都自带支线任务，一时半会儿未必会听自己的。
<br/>　　现在山寨缺的是士兵，而不是大将，有刘关张赵秦等人顶着，如今以足够用了。
<br/>　　这次程大雷召唤了一千糟糕到普通的喽啰。
<br/>　　以木板挡在头顶，下面的山贼向堡垒发起冲锋，看这情况，大门岌岌可危，随时都有可能失守。
<br/>　　窦竹仝抡着一柄大刀，看向程大雷的眼睛里能喷出火来。
<br/>　　“姓程得，你有胆量就打开大门与我一战。”
<br/>　　“快别扯淡了，还真以为自己有与我一战的资格。”程大雷看着远处，心中也有几分焦急，自己的人究竟什么时候才能到。
<br/>　　“杀呀杀！”
<br/>　　墙外杀声震天，人人都存了血洗蛤蟆岛的意思，人人奋勇，个个争先，冒着箭雨冲锋。
<br/>　　“杀，杀，杀！”花皮虎手持一杆花枪，口中大喝不止：“谁先抢破大门，重重有赏。”
<br/>　　重赏之下，必有勇夫，在花皮虎的率领下，大门已岌岌可危。
<br/>　　“杀呀杀！”
<br/>　　忽地，从众贼人身后传来阵阵呼声，花皮虎一愣，回头看去，只见岸边是密密麻麻的大小船只，这些人三五成群奔上岸，手里有拿大刀的，有拿斧头的，有拿铁叉的……一时间如蚂蚁成群，也分辨不出有多少人，上来岸后，也不废话，拿出兵器就砍杀。
<br/>　　祁珲坡大惊失色，口中叫道：“什么情况，什么情况，不是说蛤蟆寨只有一二百人么！”
<br/>　　这突然来的变故，令所有人都摸不准状况，无知大多是从恐惧起，人心惶惶。
<br/>　　箭楼上，徐神机眼睛睁大：还真有神兵天降啊！
<br/>　　“我说什么来着，我说什么来着。”程大雷用力拍拍徐神机的肩膀：“走啊，主动出击。”
<br/>　　一直攻不破的大门此刻主动打开，程大雷骑着高头大马立在队伍最前方，手中握着一大斧。
<br/>　　窦竹仝一见程大雷，眼睛立刻红了，拿着大刀向程大雷扑来。
<br/>　　“狗贼，受死吧！”
<br/>　　程大雷一马当先，带着弟兄们冲杀出来，骑兵冲击，虽然只有几十骑，但在堡垒前的平地上却无人可挡。
<br/>　　来回几下冲锋，就宛若绞肉机一般，程大雷手中大斧抡开，擦着死，挨着伤。这些人的战斗意志已经被彻底击溃，人心涣散，此刻已经思退。
<br/>　　只有窦竹仝与程大雷有刻骨深仇，此刻宁拼了一条性命不要，也要杀了程大雷报仇。
<br/>　　他握着大刀向程大雷冲来，大刀向程大雷砍来。程大雷出手的速度比他更快，虽然后发，但是先至，一斧砸在刀柄上。窦竹仝感觉双臂巨震，身子往后倒退几步。
<br/>　　程大雷再进，就要取他性命，可此刻突然有条花枪扎来，逼得程大雷不得不退。
<br/>　　花皮虎揽过窦竹仝臂膀，口中大叫道：“走啦！”
<br/>　　“狗贼，狗贼！”窦竹仝握刀还要找程大雷拼命。
<br/>　　“走啦，走啦。”花皮虎心慌意乱。
<br/>　　一个要杀，一个要逃，前后对冲，冷不防窦竹仝的刀扎入花皮虎小腹。
<br/>　　窦竹仝一愣，见花皮虎嘴角汩汩流出血来，看向自己的眼神又恨又怒。窦竹仝也是一惊，忽然心底冷笑一声：你这贼婆娘杀了吕春花，还让她死无全尸，这次正好杀你报仇。
<br/>　　他一狠心，将刀拔了出出来。花皮虎身体踉跄，一张脸变得煞白无比。
<br/>　　“贼汉子……你快走……你快走啊，你再不走……就走不成了。”花皮虎气息越来越微弱，终于一头跌倒，就此香消玉殒。
<br/>　　窦竹仝愣在原地，只感觉魂飞魄散，浑不知身在何处。睁眼看眼前，周围砍杀一片，到处都是程大雷的人，己方能逃的逃，逃不过也就是一命呜呼的下场。
<br/>　　他脸上神色变幻，时而悔恨，时而愤怒，时而得意。最后猛地一咬牙，自己杀了花皮虎，此地也没有自己存身之所在，只能逃之夭夭，等日后时机到了，再来寻程大雷报仇。
<br/>　　他抓住大刀，一路冲杀出去，岸边有登岸的小筏子，他跳上一张筏子，用刀滑水，等觉得已脱离危险区域，回头看了一眼。
<br/>　　岛上杀声未决，河面上漂浮的都是尸体，有些如自己一样，逃得了就是逃了。而逃不了，也就难逃刀下之鬼的命运。
<br/>　　十八家头领联手，以三千人打一座小岛，谁都觉得是轻而易举的事情。可最后的结果，却是谁也没有想到的，败便是败了，还败得如此惨。至于自己，前后两个女人都因程大雷而死，今日前，自己还是一方枭雄，但今日后，自己只能如一条丧家犬般逃命。
<br/>　　立在筏子上，窦竹仝忽然发出三声大笑。
<br/>　　“程大雷，若有来日，我必杀你！”
<br/>
