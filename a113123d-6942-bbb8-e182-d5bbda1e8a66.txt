　　“萧逸公子。”
<br/>　　顾怜星紧张地看着星光牢笼内的萧逸，脸色有些不安。
<br/>　　“萧逸公子，抱歉。”
<br/>　　“我替我父亲替你道歉…”
<br/>　　萧逸救她一命，历经不少危机才替她续命数年，本就是欠了恩情。
<br/>　　她恳请萧逸来天星府做客，萧逸卖她面子，便也来了。
<br/>　　可现在，萧逸却被强行禁锢留下，还被相逼，这算什么事？
<br/>　　顾怜星心头自是万分过意不去。
<br/>　　若是因此惹起误会，是她万万不愿见到的。
<br/>　　“不必了。”星光牢笼内，萧逸淡漠地摇了摇头。
<br/>　　顾怜星见萧逸语气淡漠，脸色再次一急，“萧逸公子放心，怜星即便拼了命，也必让父亲收回成命。”
<br/>　　“怜星的朋友，并不多，萧逸公子是其中一个，怜星绝不会让萧逸公子在我天星府受这般委屈。”
<br/>　　说罢，顾怜星咬了咬牙，转身便要离去。
<br/>　　“怜星姑娘不必自责。”萧逸摇了摇头，轻淡一笑。
<br/>　　“怜星姑娘的朋友不多，在下的朋友，同样不多，恰好，怜星姑娘算一个。”
<br/>　　萧逸淡淡一笑。
<br/>　　顾怜星见状，知晓萧逸并未与她计较，顿时松了口气。
<br/>　　恰在此时，公孙火舞拉了拉顾怜星的衣袖，“怜星，还有一事呢。”
<br/>　　“什么事？”顾怜星疑惑地看着公孙火舞。
<br/>　　公孙火舞跺了跺脚，低语道，“你忘了，你和姓叶那小混蛋有婚约的，萧逸他…”
<br/>　　顾怜星闻言，霎时掩嘴一笑，打断道，“若萧逸公子对我有意思，也不必被我父亲困在这星光牢笼下。”
<br/>　　“也是。”公孙火舞闻言，美眸一亮，松了口气。
<br/>　　“萧逸公子。”顾怜星看向星光牢笼内的萧逸，道，“无论如何，怜星去寻父亲解释一番，希望能让父亲解开牢笼。”
<br/>　　“真的不必了。”萧逸摇了摇头。
<br/>　　“我已与二位道别，亦没有理由留在天星府了。”
<br/>　　“三天后，我会自行破阵，随后离去。”
<br/>　　“怜星姑娘和火舞姑娘不必管我，也不必留在此了。”
<br/>　　萧逸的话语中，充斥着无比的自信。
<br/>　　“可是，这…”顾怜星迟疑着。
<br/>　　公孙火舞直接道，“小贼，我和怜星总不能干看着你被困吧？”
<br/>　　“我们能做些什么？”
<br/>　　“这…”萧逸思索了一下，道，“二位离去后，在大堂之外布一道禁制吧。”
<br/>　　“我不希望我参悟时被打扰。”
<br/>　　“好。”顾怜星点了点头。
<br/>　　“萧逸公子参悟期间，怜星保证不会有人能打扰。”
<br/>　　顾怜星的话语，同样充斥着无比的自信。
<br/>　　她实力虽不强，也无法帮萧逸破开星光牢笼。
<br/>　　但这里是她天星府内。
<br/>　　除了府主，无人敢让她蹙眉半分。
<br/>　　“火舞，我们走吧，不要打扰了萧逸公子参悟。”顾怜星沉声说道。
<br/>　　“哦。”公孙火舞点点头，转过身后，又转过头，看了星光牢笼内的萧逸一眼，之后才与顾怜星离去。
<br/>　　……
<br/>　　大堂之外。
<br/>　　嗖…嗖…
<br/>　　顾怜星二人，同时布下一道禁制，封锁了大堂的大门。
<br/>　　以她二人修为实力，加之顾怜星此时乃是武魂血脉尽皆在巅峰状态，二人布下的禁制，天星府上下，只有长老能打破。
<br/>　　但，即便长老亲至，她二人也不惧。
<br/>　　因为，她二人已打算在这大堂之外长等。
<br/>　　除却府主，任何人休想进入大堂打扰。
<br/>　　“怜星，你说小贼他真能堪破牢笼，脱困而出吗？”公孙火舞面露忧色地问道。
<br/>　　“世伯他该不会真打算困小贼一年半载吧？”
<br/>　　顾怜星摇了摇头，“以萧逸公子之傲气，怕是一年半载也休想让他折服。”
<br/>　　“不过。”顾怜星笑笑，道，“你觉得萧逸公子能脱困吗？”
<br/>　　“一定能。”公孙火舞连连点头。
<br/>　　“这小贼本事强着呢，各大险地，包括两大洞府在内，尚且困不住他分毫。”
<br/>　　“区区星光牢笼，他必能破去。”
<br/>　　“那不就得了？”顾怜星笑道。
<br/>　　“萧逸公子，是个总能创造奇迹，给人惊喜的绝世妖孽。”
<br/>　　“往往别人认为他做不到的事，他往往猝不及防便完成了。”
<br/>　　“总之，三天后，我们静等萧逸公子佳音便是。”
<br/>　　“嗯。”公孙火舞重重地点了点头。
<br/>　　二人，就这般守在了大堂之外。
<br/>　　……
<br/>　　大堂之内。
<br/>　　星光牢笼，散发着点点冷白气息。
<br/>　　这与正常的绚烂耀眼星光不同，这些星光，更像是寒冷白光，似极了一个白色的寒冷牢狱。
<br/>　　“冷星一道吗？”萧逸笑笑，随后脸色一凛，陡然闭上了眼睛，开始参悟。
<br/>　　据顾怜星所说，天星府主修的，乃是冷星一道。
<br/>　　冷星一道，位列顶尖武道。
<br/>　　但要与他的星幻剑道相比，还是差了一个层次。
<br/>　　也是因此，他才敢说三天后能脱困而出。
<br/>　　若是换了别的武道，萧逸倒是没辙。
<br/>　　当然，天星府主毕竟修为强横，有着挥手便禁锢下十数武道大能的恐怖战绩。
<br/>　　单单是这巨大的修为和实力差距，就足以让萧逸望尘莫及。
<br/>　　所以萧逸也没指望能彻底打破这个星光牢笼。
<br/>　　但若强行撕开一丝星光，足够他逃离，这一点他还是能做到的。
<br/>　　萧逸边感知着星光牢笼，边参悟着。
<br/>　　一个时辰后。
<br/>　　萧逸微微睁开了眼睛，眼中，尽是喜意。
<br/>　　“果然如此。”
<br/>　　在他的感知中，事实上完全感知不出这些星光力量到底达到何等层次。
<br/>　　这证明着，天星府主的修为实力，比他高了极大的距离。
<br/>　　可同时，他的星幻剑道感悟，明显能与这些冷白星光产生共鸣。
<br/>　　天星府主到底掌控了多少条完整武道，又达到何等武道境界，萧逸不得而知；但可以肯定的是，绝对远在武道大能之上。
<br/>　　但另外一点，萧逸的星幻剑道层次，达到了掌控九万三千道星光的地步。
<br/>　　单论星幻剑道，他的武道境界，绝不比掌控九千三条完整武道的强者差。
<br/>　　而星幻剑道与冷星一道，同属星光一道。
<br/>　　也就是说，单论此，他与天星府主的差距；只是圣皇境巅峰以上与府主级别的差距。
<br/>　　加上他的星幻剑道比冷星一道层次更高，以及他的阵法知识，仅仅是破开牢笼一丝，脱困而出，并不难。
<br/>　　三天时间，足矣。
