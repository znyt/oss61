        &lt;h3&gt;第三卷第三十章  无意之语&lt;/h3&gt;
<br/>
<br/>        江俞轩见了傅海生。
<br/>
<br/>        那个在晚上喝醉了的傅海生已经恢复过来了。
<br/>
<br/>        他穿着得体的工作服，在宽大的老总办公室里接见他，他谈起项目来如行云流水，对待工作上一丝不苟，哪里还看得到抱着陈子昂痛哭的傅海生。
<br/>
<br/>        他也并没有问陈子昂的任何一个字，只是针对项目和江俞轩讨了一个上午，中午热情的邀请江俞轩和他一起在他们单位共进午餐，仿佛昨晚的一切都不存在。
<br/>
<br/>        “傅总，没想到你对信息建设上有这么独到的见解。陈总要是知道了，肯定要大呼人才了。”
<br/>
<br/>        “我大学就是学的这个专业，平时没事了也爱钻研，西城的项目我一直都在关注，没想到还是你们公司做的。”
<br/>
<br/>        “你应该是没想到是陈总吧？”
<br/>
<br/>        傅海生愣了一下，微微一笑：“确实，她消失了很久，给了我一个惊喜。”
<br/>
<br/>        ”哈哈，陈总给我们的惊喜无处不在。我就没有见过像她这么拼的女人。“
<br/>
<br/>        “子昂一直很努力，在上学期间，都是出名的样样通。”
<br/>
<br/>        “样样通？”
<br/>
<br/>        “每门功课都是a，包括体育一样也没有拉下，别的女孩子一上体育课就藏起来，她上体育课生龙活虎，篮球比我们男生打得还溜，八百米跑和我齐头并进。”傅海生提前陈子昂的事迹，就像是他自己的事迹，满脸的骄傲，眼神温柔。
<br/>
<br/>        “怪不得呢，原来是学霸。听说傅总经常发表论文和给别的单位讲课呢，我们陈总也经常发表论文，经常出去讲课，你们同学还都是厉害啊。”
<br/>
<br/>        “你们陈总也发表论文？”
<br/>
<br/>        “难道她就是那个陈子昂？”傅海生也看过陈子昂的论文，可以说在这里她如果不是第一人，那么没有人敢说第二。
<br/>
<br/>        “是的，再没有第二个陈子昂了。”
<br/>
<br/>        傅海生心里已经震动得无以复加，原来，陈子昂已经在几年前都比他们上了一个层次。
<br/>
<br/>        他和夏颜也看陈子昂的文章，但是他们谁也没有想到那个陈子昂就是他们的同学。
<br/>
<br/>        记得第一次发现陈子昂的文章的时候
<br/>
<br/>        “傅海生，看，你初恋情人的文章。”夏颜阴阳怪气的递给他那本有陈子昂发表论文的书。
<br/>
<br/>        “你一天胡说什么呢？”傅海生还是忍不住接过了书，这一看，他被吸引住了，文章中阐述的思路和观点是那样的新颖，业界枯燥的词汇被她细致委婉的写出来，竟然生动了很多。
<br/>
<br/>        傅海生觉得似曾相识。
<br/>
<br/>        “傅海生，此陈子昂非彼陈子昂，人家这个陈子昂可是在北京啊，你的初恋情人连大学都没有上过能写出来这样的论文？”夏颜嘲笑着。
<br/>
<br/>        傅海生想了想觉得夏颜的话也对，陈子昂杳无音信这么多年，也不知道是不是早都嫁人了，况且她连大学都没有上过，对他们这个专业知识不了解，又如何能写出这样的论文来？
<br/>
<br/>        可能就是同名同姓吧，世界这么大，同名同姓的人多了去了。
<br/>
<br/>        傅海生后来每期都会去买这本书，他每次读到陈子昂的文章，都觉得给了他很大的启发。
<br/>
<br/>        他能有现在的成就，也跟看了陈子昂的书有关。
<br/>
<br/>        傅海生笑着摇了摇头：“你们陈总啊，总是会给人惊喜，我经常读她写的文章。给了我很大的启发。”
<br/>
<br/>        “哈哈，我们都叫她疯子和怪人。”江俞轩想到同事们对陈子昂的评价就笑了。
<br/>
<br/>        俩个男人聊起天来也不逊色任何俩个女人。
<br/>
<br/>        “傅总，我们陈总很久没有回家了，因为时间紧迫，她想回去看看她妈妈。你不介意吧？”
<br/>
<br/>        “怎么会介意呢，我也去过她家，山清水秀，好地方。”
<br/>
<br/>        “你这一说我也想去看看了，你们是大学同学吗？”
<br/>
<br/>        “我们是高中同学，本来说好了，上同一所学校，只是造化弄人，她学习比我好，却没有考上大学。”
<br/>
<br/>        “她学习比你好，竟然没有考上大学？”江俞轩很奇怪，他一直都不知道陈子昂是什么学历，但是就那种霸气的论文和讲课现场，让他这个研究生毕业的人都汗颜。
<br/>
<br/>        有时间他感觉很奇怪，明明陈子昂和他年龄差不多大，她怎么就能比他早出来工作，而且还发表论文？
<br/>
<br/>        他一直认为她是大学毕业，或者是研究生毕业。
<br/>
<br/>        这，让他这一会接受不了了，手中的大烩菜已经不愿意进嘴了。
<br/>
<br/>        一个高中生做得都比他好，这让他情何以堪？
<br/>
<br/>        “是啊，她当初可是我们学校的学霸呢，几个年纪她都是稳居第一。”傅海生一脸的骄傲，似乎就是他是年级第一一样。
<br/>
<br/>        江俞轩对他的这个老总以前的过往不是很了解，只是有点好奇她的过往，凭着陈子昂的智商竟然考不上大学？他还以为陈子昂大学毕业认识的张函呢。
<br/>
<br/>        怎么会没有上过大学，那么她那么多的专业知识从何而来？
<br/>
<br/>        “你们高中会学到信息上面的知识吗？”江俞轩很好奇，莫不是他们学校另外开了课？
<br/>
<br/>        “怎么会呢？我们在泉城的重点高中上学，那个时候交通不便利，哪有什么信息课可上的？我报考的学校还是子昂说的，她说她听别人说的，她要做时代的弄潮儿。”傅海生见江俞轩一副见鬼的表情接着解释。
<br/>
<br/>        ”子昂当初学习好，她报考什么学校，我就想报考什么学校，我考上了，她落榜了。“
<br/>
<br/>        “好奇怪啊，稳居第一都没有考上大学？那是个什么鬼？”听了傅海生的解释，江俞轩充满了不相信。
<br/>
<br/>        傅海生也一愣，他从来都没有想到过陈子昂是学霸却没有考上大学。
<br/>
<br/>        “是啊，她学习那么好，怎么可能考不上大学？连夏颜都考上了。”落榜？好像当初都没有陈子昂的名字，当时他可是仔仔细细的找了大家的名字，还看了分数线，就是没有陈子昂的名字。
<br/>
<br/>        夏颜稳居第一！
<br/>
<br/>        “夏颜是谁？”
<br/>
<br/>        “我老婆。”
<br/>
<br/>        “你老婆学习很差吗？”
<br/>
<br/>        “何止是差？简直是惨不忍睹。”傅海生的手一抖，手中夹的菜掉到了衣服上。
<br/>
<br/>        夏颜稳居第一，对，没错，当初确实是夏颜位居榜首。
<br/>
<br/>        江俞轩的那一句“什么鬼”让他有了一些模糊的想法。
<br/>
<br/>        “对不起，我衣服脏了，我回去换一换衣服。”傅海生站起来急急的对江俞轩说。
<br/>
<br/>        “.......有洁癖？”江俞轩看着急匆匆离开的傅海生。
<br/>
<br/>        傅海生回到办公室，拿起汽车钥匙就走。
