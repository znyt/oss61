　　龙腾和苏莫一样，一直在观察着龙族众强者的神态，尤其是龙帝的神态，因为这决定了他以后的地位。
<br/>　　龙帝没有表态，这让他心中暗暗恼怒，但是他也只能忍着。
<br/>　　只有龙帝表态了，他日后才能在龙族享有崇高的地位，才能有足够的力量帮助苏莫。
<br/>　　所在，在这众目睽睽之下，他当众询问龙帝，就是心中敢断定，对方不敢公然忤逆洪荒祖龙的意志。
<br/>　　闻听龙腾之言，无数龙族之人，纷纷将目光投向吞虚龙帝，个个惊疑不定。
<br/>　　众人也知晓，龙腾想要成为新的祖龙，最主要的是得到吞虚龙帝的认可，不然有名无实。
<br/>　　毕竟，吞虚龙帝目前，乃是龙族的掌控者，以及龙族的最强者。
<br/>　　若是吞虚龙帝不认可龙腾，此事估计还有很大的变数。
<br/>　　面对龙腾的询问，吞虚龙帝面无表情的脸庞之上，缓缓的露出了一丝笑容。
<br/>　　“呵呵，祖龙留下的意志，本帝怎么会反对，你有资格成为新的祖龙。”吞虚龙帝微笑着说道。
<br/>　　只不过，其虽然面带微笑，但是，笑意却是给人一种莫名的冷意。
<br/>　　大长老闻听此言，微微松了口气，若是龙帝不认可龙腾，那还真的有些棘手了。
<br/>　　好在，龙帝遵从了洪荒祖龙的意志，虽然不是心中真正的认可，但也算是不错了。
<br/>　　毕竟，龙腾修为还低，按现在展露的修为气息，不过是四品的修为。
<br/>　　苏莫和龙腾、司空炎闻言，心中也是暗暗的松了口气，若是龙帝仗着实力强大，无视龙腾的身份，他们还真的不好办。
<br/>　　“龙帝，事关重大，我觉得族中应该好好商议一番，龙腾日后在族中的地位。”大长老提议道。
<br/>　　虽然龙腾可以成为新的祖龙，但是，具体在龙族之中的地位，还要好好商议一番。
<br/>　　“嗯！”
<br/>　　龙帝点了点头，随即，略一沉吟，他环视四方，朗声道：“所有长老，前往天龙大殿易事。”
<br/>　　言罢，龙帝的目光又落在了苏莫的身上，道：“屠霄，你也前去！”
<br/>　　话音一落，龙帝一摆衣袖，转身离去。
<br/>　　龙帝离去，约莫有十数名龙族长老，也紧跟着龙帝离去，前往天龙大殿。
<br/>　　唰！
<br/>　　这是，龙腾再次化为了人形，落到了大长老的身边。
<br/>　　“大长老，多谢了！”龙腾向一头金发，体型高大的大长老拱了拱手，今日，若不是有对方，他也不会轻易得到众多长老的认可。
<br/>　　虽然那些长老只是认可他新的祖龙的身份，并不是真正的臣服与他，但是，只要是认可便已经足够了。
<br/>　　“这是你的造化！”
<br/>　　大长老叹了口气，随即沉声道：“走吧，前去天龙大殿！”
<br/>　　“嗯！”龙腾点头，随即向苏莫示意，便和大长老等一众长老，穿过分开的人群，向天龙大殿而去。
<br/>　　苏莫跟在龙腾的身边，他不清楚龙帝为何要他也前去，不过，也能大概猜出来。
<br/>　　……
<br/>　　天龙大殿，是龙族一座最雄伟，最庞大的宫殿，占地方圆数千丈，大气磅礴。
<br/>　　这里是龙族商议大事的地方，也是龙族接待其它神朝强者的地方。
<br/>　　大殿之中，高高的主座之上，一身月白色长袍的龙帝大马金刀而坐。
<br/>　　下方，共有三十七位龙族的神王境长老，分坐在两侧。
<br/>　　其实，龙族的神王境长老，并不止三十七位，只不过，有些长老人物，并不在龙都城，或者说不再天龙神朝。
<br/>　　龙腾端坐在右侧的首位，对面便是龙族的大长老。
<br/>　　而苏莫，则是坐在右侧的最后一位，他不是长老，更不是真正的龙族之人，自然地位较低，只能坐在最后方。
<br/>　　“诸位，今日，龙腾血统进化成祖龙，是我族的一件大事，值得庆贺！”
<br/>　　吞虚龙帝环视下方众人，朗声开口，继续道：“不过，现在首要的事情，是商议龙腾日后在本族的地位，大家有什么看法，尽管之言。”
<br/>　　“龙帝，龙腾成为新的祖龙，这是洪荒祖龙的意志，按规矩来说，他要执掌龙族，成为天龙神朝之主！”大长老沉声说道。
<br/>　　“大长老，此言差矣！”
<br/>　　这时，一位身材高瘦，面色略显黝黑的龙族老者站了起来，沉声道：“龙腾虽然可以成为新的祖龙，但是，目前毕竟修为较低，而且，将来能否成为至强者，还未可知，暂时不宜执掌龙族。”
<br/>　　此言一出，不少龙族长老纷纷点头，表示赞同。
<br/>　　这位高瘦长老，乃是龙族的二长老，所言在理，龙腾可以成为龙族新的祖龙，享有无上的荣誉，但是目前而言，执掌龙族就有些不妥了。
<br/>　　大长老闻言，略一沉吟，道：“二长老言之有理，但是，这是洪荒祖龙的意志，我们不能违背。”
<br/>　　大长老此言，也是得到了很多长老的认可，的确，这是洪荒祖龙的意志，不能违背。
<br/>　　洪荒祖龙如今并没有陨落，虽然数百万年没有出现了，但是在龙族之人的心中，永远是最崇敬的始祖。
<br/>　　龙腾面色平静，他并没有发表什么意见，因为，他就算是争取，也不可能有实质的效果。
<br/>　　苏莫静坐在最后方，静静的看着这一切，静看事态的发展。
<br/>　　“龙帝，您的意思呢？”有人向龙帝问道。
<br/>　　吞虚龙帝闻言，沉默了片刻，环视众人，声音低沉的道：“龙腾若是日后，能成为至强者，当然可以执掌龙族，统领天龙神朝，但是目前还太弱小了……！”
<br/>　　说到此处，龙帝稍稍停顿，继续道：“本帝的意思是，龙腾成为新的祖龙，在我龙族之中，享有崇高的荣誉，名义上的地位与本帝相当，但他目前最主要的是提升修为，将来若是实力超越本帝，便可执掌龙族！”
<br/>　　吞虚龙帝的声音，响彻在大殿之中，威严无比，态度明确。
<br/>　　其态度就是，目前还不会将龙族大权交于龙腾，一起需等龙腾有足够的实力之后。
<br/>　　众人的心中，如明镜一般，他们都知晓，龙帝目前不会交权。
<br/>　　毕竟，无论是出于私心，还是龙族大局，也不能将大权交于一个毛头小子。
<br/>　　苏莫和龙腾对视了一眼，从对方的目光之中，他也看出了对方的无奈。
<br/>　　现如今在龙族，吞虚龙帝势大，无法忤逆。
<br/>　　新的祖龙，怕是也只是名义上的祖龙，很难掌控龙族多少实权。
<br/>　　“诸位，你们觉得本帝的安排如何？”吞虚龙帝环视众人，眸光如电，面带威严的问道。
