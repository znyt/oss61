　　第140章
<br/>　　银子还是钢刀，并不是一道太难做对的选择题。
<br/>　　况且乔野目前的确很缺银子，而在牢里对程大雷照料一二，也不算太难的事情。
<br/>　　几个匪人起身，为首之人道：“乔将，再会了，江湖路远，咱们改日再聊。”
<br/>　　如此就要走了，他们倒也没伤自己家人。乔野突然问：“你们不怕你们走后，我出尔反尔，吞了银子不做事？”
<br/>　　那匪人笑笑：“只有千日做贼，没有千日防贼的道理。假若乔将拿了钱不做事，我们下次登门就得带走点什么了。”
<br/>　　带走什么？胳膊耳朵眼睛腿……还是自己刚出生儿子的脑袋。乔野忽然一身寒意，身体情不自禁打个哆嗦。
<br/>　　“走啦！”
<br/>　　几名匪人大摇大摆离去，他们刚走，乔野便立刻扑过去，将家人身上的绳索解开。
<br/>　　“你们都没事吧，宝儿呢，宝儿受伤没有。”
<br/>　　“老爷，幸亏你回来了，可吓死我们了。”
<br/>　　屋里哭哭啼啼声一片。
<br/>　　乔野检查各人情况，发现他们最严重着不过是擦破点皮，至于自己的宝贝儿子更是安然无恙。
<br/>　　如此，他才长长松了口气。
<br/>　　“今天的事谁也不许说出去，懂么！”乔野的虎目从每个人身上扫过。
<br/>　　说罢，目光落在那两口箱子上，白花花的银子让人眼花缭乱。乔野忽然觉得，今天的经历也未必是坏事。
<br/>　　刘悲带人离开乔家，转过几个街道见身后无人跟踪，方才回到湖底捞。
<br/>　　陆陆续续还有人回来，在这几天内，还有蛤蟆寨的人潜入落叶城，人数大概在百人左右。
<br/>　　大家分成好几股潜伏在城内，每到夜里各家头目在湖底捞汇合，分享各自的情报。
<br/>　　“今天，已经有几个兄弟故意闹事，被官兵抓进大牢去照顾大当家。”高飞虎。
<br/>　　“这样一来，大当家在牢里就不会无依无靠了。”刘悲道：“钱已经给乔野送去了，他拿了钱自然就会办事，大当家应该就不会受苦了。”
<br/>　　张肥拿出一份草图：“牢周围的环境我看过，自从大当家被抓进去后，大牢外就加了重兵。想要劫牢不太现实，除非他们把大当家押送到别的地方时，我们在路上动手。就怕，他们将大当家就地处决。”
<br/>　　“以大当家的所作所为，不是没有这个可能啊。”徐神机叹了一声，又道：“今天已经找人买通了牢头，据牢头所说，大当家在牢里是单独一个牢饭，目前还没上刑，他已经答应照顾大当家。”
<br/>　　想要保证程大雷的安危，自然不可能只靠一个乔野，事实上，这几天蛤蟆寨的银子大把花出，大棒加甜枣收买了更可能多的人。
<br/>　　“咱们还有多少银子？”刘悲问。
<br/>　　“……没剩多少啦。”徐神机无奈的摇摇头。
<br/>　　缺钱是如今的现实，倒不是说蛤蟆寨穷得揭不开锅了，单纯就是没银子，没有通用方便的货币。
<br/>　　蛤蟆寨倒有几十万斤稻米，可做这样的事情，也不能拉着几百斤稻米到处去逛。
<br/>　　“对了，今天我见那牢头时，从他嘴里听到一个消息，似乎还有人在上下打点……”徐神机。
<br/>　　……
<br/>　　上下打点的人是李如，一方在想方设法救程大雷，而李如则是想方设法让程大雷死在牢里。
<br/>　　这几天，她都在办这件事，然而过程中，她隐约发现，自己送出的钱，那些人好像不是很敢接。
<br/>　　怎么回事，什么时候这帮吸血鬼变得这么清廉了。
<br/>　　今日，李如决定亲自见一见落叶城城主，一定要在牢里做死程大雷。
<br/>　　“夫人，小姐来了？”有仆人过来。
<br/>　　“小姐，那个小姐？”
<br/>　　“大……苏樱。”仆人恍然意识到，苏樱如今已经不是苏家的小姐。
<br/>　　李如目光若有若无的在仆人脸色扫过，但已足够令对方如坐针毡。她心里也琢磨着：苏樱此刻来见自己做什么？
<br/>　　“让她在堂屋等我。”李如想了想道。
<br/>　　苏樱和小蝶等了好大一会儿，李如才从外面进来。
<br/>　　“樱儿，你可是让我好想，找就想让人找你，咱们娘俩说说话。你爹虽然不认你这个女儿，我可是一直把你当亲闺女的。可你不登我这个门，也不能让当娘的去给你赔不是。”李如刚进来就是一连串的话。
<br/>　　“有意思么？”苏樱。
<br/>　　李如脸色的表情忽然顿住，二人都穿着孝衣，目光相会的刹那，针尖对麦芒。
<br/>　　李如抬起茶杯，微微低头。
<br/>　　“来干嘛？”
<br/>　　“苏家家产我不要，我要一笔银子。”
<br/>　　“当初你父把你赶出去，这个家已经没你份了，你要家产从何说起。”
<br/>　　“我父死前把苏家家产都给了我，家里一名老奴已经把这件事告诉我。我父生前留下一份遗嘱，这份遗嘱现在应该在你手里。我今天来，不是要分家产，是把苏家全部家产放弃，只要一笔银子。”
<br/>　　李如暗暗晦气，这样的事究竟是那个混蛋传给苏樱的。她顿了顿，道：“酒楼也是苏家的产业。”
<br/>　　“我可以放弃。”
<br/>　　“你要多少？”
<br/>　　“一万两？”
<br/>　　“没有。”
<br/>　　“八千？”
<br/>　　李如摇头。
<br/>　　“五千。”
<br/>　　李如继续摇头。
<br/>　　“苏家的田产，城里的产业，码头上南来北往的生意，这些还不值五千两银子。”
<br/>　　“没……有也和你没多大关系。”李如笑笑：“最多三千。”
<br/>　　苏樱盯着李如：“好！”
<br/>　　“口说无凭。”
<br/>　　“立字为据。”
<br/>　　李如差下人取来纸笔，当场让苏樱写下字据，声明自愿放弃家产。
<br/>　　“银子？”
<br/>　　“你帮我立份收据，三千两银子不是小数目，你也拿不走，我让下人给你送过去。”
<br/>　　“苏樱为李如写下一份收据，换来李如手书的一份欠条，她咬着贝齿：“人在做，天在看，你若不把银子送来，小心老天收你。”
<br/>　　“放心，老天还没收你这个土匪的女人，证明老天也有瞎眼的时候。”李如平静道：“放心，我还不会贪你这三千两。”
<br/>　　“小蝶。我们走！”
<br/>　　苏樱起身，拂袖离去。李如拣起桌上的字据，仔细看过一遍，脸上浮现得意的笑意。
<br/>
