　　这简直匪夷所思。
<br/>　　大伙讨论了一下没有结果，也只能暂时丢下这件事不管了，叶少阳让瓜瓜好好照顾陈悦，潜在目的是好好盯着她。
<br/>　　不知道为什么，叶少阳总觉得她跟自己之间，有着某种奇妙的关联，或者说将来会跟她产生什么联系。
<br/>　　至于皇帝……记忆中是一片空白。当然没有人是没有记忆的，就算是白痴也有记忆碎片，什么都检测不到，更加让他们怀疑，或许是圣灵会用了什么秘法，洗去了他的记忆。
<br/>　　虽然听上去不太可能，但也没有更好的解释。
<br/>　　早上吃了宫里的御膳，叶少阳带着小马一干人前去寻找自己的法器，张无生则带着自己的几个弟子（从清醒过来的众人之中找到的）留守皇宫，等他们回来。
<br/>　　叶少阳能感知到法器的大致位置，离京城应该不远，几个人步行出发，后面跟着大批邪物。
<br/>　　这些都是法器丢了的。
<br/>　　经过统计和总结，他们发现，凡是被圣灵会吸收进武装的——像张无生，这些人的法器多半都没丢，而且在他们的记忆中，这些都不是自己的法器，而是因为立了军功什么的，上级奖赏的，要么就是师门的长辈给的，反正总有原因。
<br/>　　而那些成为老百姓的，法器基本上都被没收了——包括吴嘉伟的藏锋。
<br/>　　这其实也很好理解：既然他们被灌输了记忆，成了老百姓，家里有法器肯定不合身份，而那些兵士，因为本来就需要他们来维持秩序，人手一个刻了符文的兵器，其实也相当于法器了，再有更好的法器，只要有合理的来源，完全不会引起怀疑。
<br/>　　这里头唯一的例外，就是小马，他本来也没什么法器——当初芮冷玉给他的碎魂杖，后来因为用不上还还给冷玉了。他唯一的本名法器，就是那块玄铁板砖。
<br/>　　想不通为什么没有被没收，后来吴嘉伟想到一种可能：也许是因为这东西实在长得不像是法器，被筛选的人给忽略了。
<br/>　　这本来是一句戏言，但是大伙仔细一想发现，似乎很有这种可能啊。小马沾沾自喜起来，越发觉得自己这法器特别。
<br/>　　出了城之后，大部队浩浩荡荡地往北而行，去寻找叶少阳感觉中的地点。
<br/>　　本来叶少阳也怀疑自己的感觉会不会错了，因为这个世界就这么几个城市，彼此之间全是大片的树林和平原相连，并没有什么东西，大批的法器，难道就放在平地上不成？
<br/>　　“还有一点，我倒是不明白，你说法术公会得了这么多法器，为什么不送走呢？留在这个世界干什么？”吴嘉伟道出疑惑。
<br/>　　“这个问题我想过了，有两个可能，一种可能是，这个世界跟外界隔绝，他们没法把这些法器带出去，只好暂时堆积在这里。还有一种可能……他们根本就看不上这些法器，你要知道星月奴是轩辕山的，他那些嫡传弟子，用的也都是轩辕山法器，人间的普通法器，真的不入他们的法眼。”叶少阳给出自己的猜测。
<br/>　　吴嘉伟想了一会，问道：“他们不是收买了人间很多门派的法师吗，为什么不分给他们武装起来。”
<br/>　　“傻啊，法器都是有特征的，法术公会一向不愿意承认他们就是圣灵会，不管人间有多少人怀疑，只要没有证据，他们就可以不认，这种落人口实的事情，他们怎么可能去干。”
<br/>　　“这倒也是。”大伙纷纷点头。
<br/>　　小马又问道：“但是有一点，小叶子，别的法器要说星月奴他们看不上，倒有可能，但是你那几件法器，尤其是龙泉剑，就算是星月奴也不可能不感兴趣。”
<br/>　　碧清接过去说道：“星月奴很贪的，我的炼尸缸就被她收了，更不要说七星龙泉剑。”
<br/>　　叶少阳沉吟不语，这个问题，他也一直想不明白。
<br/>　　一大群人往北穿过了几片树林，前方是平原了，离老远能看到一排的建筑物，小马派人去查探，回来的人说，这里曾经是一座军营，有部队在这边驻守，后来部队调去了北大营，这里就成了物资仓库，有白虎堂的人曾经在这边带着一堆士兵看守，而今白虎堂的人都不见了，这里的士兵昨晚也遭到了邪物的袭击，早就没人了。
<br/>　　被派去调查的士兵还带回来一个消息，说是昨晚，有不少邪物在驱散追杀了这里的守兵之后，试图进入仓库，但中间最核心的一块地方，四面都贴着很多灵符，还有一些镇宅的法器，让他们根本没法进去，就放弃了。
<br/>　　叶少阳一干人互相望去，眼睛都在发光。
<br/>　　大概是没来错地方。
<br/>　　仓库的造型，就像一座巨大的瓦房，上个是个尖顶，大概是为了方便物资进出，大门修的很高很阔，小马看到这仓库第一眼就叫起来：“这特么不是CS里的仓库地图里那个仓库吗？”
<br/>　　“人家本来就是仓库，仓库都是一样的吧。”叶少阳白了他一眼，左右看去，在这座大仓库的对面，有一道高高的木头栅栏，还有一排营房，大概只给之前值守在这里的人住的，现在人去楼空，但是营房前面的地上，跟别处一样，到处都是残肢断臂，地上还流着一滩滩的血，都渗入了黄沙之中，还有很多残缺不全的盔甲和兵器，地上也是一片狼藉，显示着昨晚的战斗有多激烈。
<br/>　　两扇朱红色的大门，紧紧关闭。
<br/>　　叶少阳率先投过去，一抬头就看到了大门上方的正中间、悬挂的一块八卦镜，镜面上用朱砂写着某些符号文字，叶少阳走到跟前，八卦镜立刻亮起来，发出金灿灿的灵光。
<br/>　　叶少阳立刻感到浑身灼热，心中吃了一惊，往后退了少许，到了栅栏门的位置，镜面上的灵光立刻就消失了。
<br/>　　“这是一道法阵，是针对所有生灵的，一旦有人靠近，法阵就会激活，而且法力很强，怪不得昨天那些邪物没法靠近了。”
<br/>　　（今天去农村看大仙了，回来晚了，赶写出来）
