　　据他所知，CCG内部隐藏的怪物，可不在少数，就连明面上，也有有马贵将这样一个强大的对手。
<br/>　　如果一直能保持现在这种状态，即使引来了这些人他也能勉强对抗，但状态消退之后，他的战力也就和一般的SS级喰种大致相同。
<br/>　　不说那些隐藏的家伙，就连有马贵将来了，他也很难将其打败，毕竟对方也是疑似SSS级的存在。
<br/>　　找个安全的地方，抓紧将芳村艾特吞噬了，然后将所有的积分消化，大概也就能突破SSS了，那时，他就不用这样束手束脚了。
<br/>　　对接下来的行程稍稍计划了一下，刘哔路过金木身边，顺手将他提起，从入口处走了出去。
<br/>　　……
<br/>　　蹲在角落的一堆杂物旁，什造倚着墙壁，脸色无比苍白。
<br/>　　她觉得自己身上的伤势似乎比刚才更严重了，甚至她都已经察觉到，身体机能的逐渐下降。
<br/>　　不过也不全是坏事，身体并没有一开始那样疼了，但却比之前冷了很多。
<br/>　　这就是那些人所说的死亡吗，似乎也没有想象中的恐怖呢。
<br/>　　她不畏惧死亡。
<br/>　　或者说她对生命并没有一个标准的定义。
<br/>　　在什造的看法中，死亡这件事情，和喝水睡觉玩耍并没有任何不同。
<br/>　　人都会死，这是谁都摆脱不了的事实，早死一点或者晚死一点，又何必那么较真呢。
<br/>　　反正没人会在乎，她死了，对大家来说也是不值一提的事情。
<br/>　　只是觉得有些遗憾，再也品尝不到宿舍楼下那家店的甜甜圈了。
<br/>　　大叔应该是骗了她的吧，说什么她也会像大多数人一样，有一个美满的家庭，也值得展望的未来。
<br/>　　算了吧，肯定是没了。
<br/>　　她有点想妈妈了，
<br/>　　虽然妈妈对她做了很多残忍的事情，
<br/>　　但她还是喜欢妈妈。
<br/>　　真是的，那些家伙，为什么非要把她从妈妈身边抓走，然后告诉她应该做什么，不应该做什么！
<br/>　　说起来，你们这种才叫虐待好吧！
<br/>　　虽然已经很努力了，但还是无法理解很多事情。
<br/>　　她压根不知道怎么才能顺着那些人的心思，活下去。
<br/>　　连好孩子标准都没有的地方，到底该让她怎么办啊！
<br/>　　不过还好，她再也不用去想这些复杂的事情了，她马上就要死了。
<br/>　　真开心。
<br/>　　朦朦胧胧之间，她似乎听到了有什么声响，哦，好像是大门打开了，还有脚步声，是有人来了吗？
<br/>　　她努力的睁开眼睛，想要看清到底是谁，但眼前的视野实在是太模糊了，根本看不清楚。
<br/>　　那个人手里好像提着什么东西，大概是个人，咦，他为什么要扔掉呢。
<br/>　　他好像有说什么，但是听不清。
<br/>　　他冲自己跑过来了，大概认识自己？
<br/>　　他在喂自己吃东西，呸，怎么会这么苦，比不加糖的咖啡还苦。
<br/>　　他把自己抱起来了，是要带她去哪里呢。
<br/>　　……
<br/>　　两只手抱着什造，顺便还倒提着金木，刘哔觉得自己现在的状态一定丑爆了。
<br/>　　但没办法，他到达什造藏身的地方，这家伙就已经快要昏迷过去了。
<br/>　　目前这种危险的情况，他没有时间等待什造自己苏醒过来，只得喂完药之后，将她抱出去了。
<br/>　　不过好消息是，什造的性命是保住了，她身上被自己打出来的伤口，正在以一个肉眼可见的速度愈合。
<br/>　　果然是四千多积分一颗的疗伤丹，价格是贵了点，但效果是真的不错。
<br/>　　带着什造和金木，刘哔顺着艾特进来时破坏掉的出口，挥动翅膀飞了出去。
<br/>　　这里似乎是处于城市的边缘，周围建筑不多，也没有什么人。
<br/>　　或许是之前有人，但大多数都死掉了，他还记得艾特下来时，落下的那场尸雨。
<br/>　　四周道路上都能看见残留下来的血迹，以及一些残肢断骸，刘哔只是扫了一眼，对此并不怎么关心。
<br/>　　死就死吧，管他屁事，反正他也不认识。
<br/>　　或许是时间原因，CCG那边还没有什么反应，并没有继续增派人手，刘哔很顺利的就逃离了出去。
<br/>　　但是去哪，可是个十分蛋疼的事情，失去了搜查官的身份之后，他可没有固定的居所。
<br/>　　况且，他也不知道应该怎么躲开CCG的追查。
<br/>　　消除踪迹，对他来说很容易，但如果动用科技的手段，他就很难躲开了。
<br/>　　毕竟这个世界的科技明显比末世前的地球更为发达，难免会有一些能够追查倒他的东西。
<br/>　　因此，谨慎起见，刘哔并没有盲目的寻找落脚点，而是从空间戒指中掏出了手机，拨动了几下。
<br/>　　“喂，哪位。”接通电话的是一个男子，听声音非常年轻。
<br/>　　“我被CCG通缉了。”刘哔将金木扔在地上，然后把什造小心放下，方才拿起手机，回答道。
<br/>　　电话那边没有回音，似乎是在犹豫，或者是在做什么其他的东西。
<br/>　　“额，先生你是，我们很熟？”年轻男子的声音再度响起，语气充满了试探，毕竟，被CCG通缉可不是什么好事情。
<br/>　　摘下了手上的拳套，刘哔握着手机，脸上见不到半点慌乱：“你不认识我。”
<br/>　　“那关我屁事！嘟嘟嘟……”电话那头的人似乎有些愤怒，直接将电话挂断，手机中传来一阵盲音。
<br/>　　刘哔皱了皱眉头，继续拨通了那个号码，挂断，继续拨通，连续数遍之后，对方再次接听。
<br/>　　“你到底要干什么，我告诉你，你这样是骚扰，在这样我就报警……”电话那头的声音充满怒气，喋喋起来。
<br/>　　“金木在我这里。”没再说什么废话，刘哔直接进入了主题，他现在的时间并不多了。
<br/>　　电话那头的声音戛然而止，停顿了一段时间，年轻男子方才开口：“你在哪里。”
<br/>　　辨认了一下位置，刘哔给他报了这条路的名字，随后挂断了电话，他知道，这个人肯定会来。
<br/>　　而且，因为金木的身份，他也不可能去报警，相反，他还会想办法将他们摆脱接下来CCG必定会有的追踪。
<br/>　　这个男人，也是他接下来的计划中，非常重要的一环，这关乎到，整个计划是否能顺利实施。
