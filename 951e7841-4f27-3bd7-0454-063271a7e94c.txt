她温软无骨的身体从后面贴上来，陆寒霆不用回头看都知道是谁来了，他将修长的手指一根根的穿梭进她柔白的指腹里，和她十指相扣，“陆太太，我发现你真是越来越大胆了，在我工作的时候叫我出来给你当司机，恩？”
<br/>
<br/>        夏夕绾从后面紧紧的抱住他，像八爪鱼一样黏在他身上，“陆先生，人家打不到车嘛，再说了我们要赚那么多钱干什么，难道你还想养一个后宫啊，我可是很好养活的！”
<br/>
<br/>        陆寒霆没有回头，就这样被她温软而有力的抱着，半响他缓缓敛上俊眸绾绾，你这样会让我变得越发贪婪。

<br/>        ……

<br/>

<br/>
<br/>        李玉兰的脸色十分苍白，“妍妍，这件事非同小可，如果在医术上你再比夏夕绾给压下去，那以后很难再有翻身之日了。”
<br/>
<br/>        夏妍妍怎么会不明白这个道理呢，玩男人她玩不过夏夕绾，现在搞事业又搞不过夏夕绾，在夏夕绾面前，她简直就是一个彻头彻尾的loser了。

<br/>
<br/>        夏振国和李玉兰迅速起身，也跟了过去，“妍妍，我们跟你一起去，我们倒要看一看这个夏夕绾玩的什么把戏！”
<br/>
<br/>        ……

<br/>
<br/>        李文清院长看了一下钟表，问向身边的周萍，“周主任，时间就要到了，你去看看绾绾有没有到。”
<br/>
<br/>        “是，院长。”周主任准备转身。
<br/>
<br/>        但是这时耳畔传来了一道清丽悦耳的嗓音，“院长，周主任，不好意思，我来迟了。”

<br/>        夏夕绾跑了过来。

<br/>
<br/>        “绾绾，你到了就好，时间还没有到，你不算迟到，走，我们现在就上主席台。”

<br/>        “好。”

<br/>        李院长，现在夏夕绾学历成谜，请问真相究竟如何？
