<br/>				城主差了人来询问，问她要否要出来吃饭，都被她给拒了。<br/>　　<br/>　　其实，像他们这种修为的修仙者，就算是十天半个月不吃都没事的，若是进入冥修其间，就算是数年不吃不喝也不会有什么事。<br/>　　<br/>　　更何况，凤九空间里吃的东西也不少，更不会在这时说了出去吃饭了。<br/>　　<br/>　　找了一天的书，也看了一天的书，在这夜深人静之时，月光依稀从上方的窗口洒落进来，光线柔和，让她不禁有了一丝困意。<br/>　　<br/>　　她打了个哈欠，伸了伸腰缓步在这里面走着，在这藏书楼中，四面都有夜明珠当灯照明，只不过有的书架后面被挡到，光线照不亮罢了。<br/>　　<br/>　　缓步走上二楼，在二楼中转了一圈，随手拿起一些书藉看着，见有的高阶功法书藉却是施了印记无法打开的。<br/>　　<br/>　　“原来如此，原来有的书是施了禁制的，这样一来，就算是我想看，也看不到这书中的内容了。”<br/>　　<br/>　　她笑了笑，并不奇怪。毕竟，若不是有施禁制印记，那这里面一些高阶功法岂不是被她看完平白便宜了她？<br/>　　<br/>　　将手中的一本施了禁制的书放了回去，她又往上走去，来到第三楼看了一圈，依旧如二楼一般，有的书随手可翻，有的书却是看不得的。<br/>　　<br/>　　走了一圈后，她又往一楼走去，一天的时间，她连一楼的书都没看完。三天的时间里，当真能从中找到记载登仙梯的线索？<br/>　　<br/>　　来到一楼的楼梯处坐了下来，她闭上了眼睛，突然间，脑海中又响起了白天那老者说的那几句话。<br/>　　<br/>　　“登仙梯，登仙梯，一步登仙一步虚，仙路之巅天外天，梯下人间多少年？这是什么意思呢？”她轻喃着，目光睁开往一楼看去，当视线落在某一处时，目光微闪。<br/>　　<br/>　　她当即起身往一楼走去，来到后面几排的书架处，那摆放着椅子的地方，看着那白天老者站过的椅子，她抬脚站了上去，在上面的书架上找了找。<br/>　　<br/>　　“没有？是我想多了？”她心下疑惑，抬头再看，视线落在最上面架子上，于是将那最上面的那些书全拿了下来，一本本的放在地面上，继而，坐在椅子一一查找着。<br/>　　<br/>　　“咦？”<br/>　　<br/>　　当手中拿着一本被施了封印的书时，她看着那上面的几个字，眼中不由浮现喜意。<br/>　　<br/>　　“蓬莱杂记？这是记载着蓬莱仙岛杂记的书藉，会不会，在这里面就有关于登仙梯的资料呢？”她自言自语的说着，看着上面的印记不由的一笑。<br/>　　<br/>　　无论是解封印还是结界或者是阵法，对她来说都不是问题，像这种印记，只要她解开了看完之后再给它封印起来也就是了。<br/>　　<br/>　　当下，她一手托着书，一手凝聚着灵力气息，嘴里喃喃念着破印的术法，只见她手掌灵力气息在那书藉上面如同火纹般拂过，封印着书藉的那道印记便如水纹般消失。<br/>　　<br/>　　翻开书藉一页页的看着，见这上面记载着多为蓬莱仙岛发生过的一些事情，以及四大城的形成……<br/>
