　　离开了武斗场之后，苏莫也没有回到住处，而是向地灵塔而去。
<br/>　　三天时间，只是个拖延的幌子而已，他只是要利用这个时间，继续精深混天魔功而已。
<br/>　　他现在的混天魔功，连功法转换都没有完成，更别说是第一层境界了。
<br/>　　不过，这功法的确强大，苏莫甚至都想让本体也修炼这门功法了。
<br/>　　此事暂时不做考虑，等巫族之事完成之后，再行决定。
<br/>　　不多时，苏莫再次来到了地灵塔，这一次他来到了第二层的空间中。
<br/>　　第二层空间和第一层空间一模一样，只是修炼室略少一些，人则是要少的多。
<br/>　　苏莫到来之时，正好有空余的修炼室，便直接进入了修炼室之中。
<br/>　　关闭修炼室的大门，他盘膝而坐。
<br/>　　手掌一番，他手中出现了两枚储物戒，正是巫烈和巫玉儿的储物戒。
<br/>　　之前他眼疾手快，将两人的储物戒都收了起来，包括巫玉儿的长剑，都被他收取了。
<br/>　　检查了一番两人的储物戒，苏莫非常满意，下品灵晶足有两千余块，其它杂七杂八的宝物也有不少。
<br/>　　懒得仔细查看，随意检查了一番，他便将两枚储物戒都收了起来。
<br/>　　随即，苏莫取出了十几枚下品灵晶，放入下方的阵法凹槽，立刻便启动了阵法。
<br/>　　这修炼室的阵法，较为消耗灵晶，第一层空间修炼室中的阵法，一枚下品灵晶能支持一整天。
<br/>　　而这第二层修炼室中的阵法，一枚下品灵晶只能支撑两个时辰的时间。
<br/>　　阵法启动之后，磅礴的灵气立刻从阵法内涌了出来，充斥在整个修炼室之中。
<br/>　　以免再闹出乱子，这一次苏莫不敢再大力催动吞噬战魂，就只是微微催动一丝。
<br/>　　磅礴的灵气如体，苏莫立刻闭上双眼，运转了混天魔功，开始了修炼。
<br/>　　少倾之后，他身上魔威涌动，整个人身上都冒出滚滚黑气，霎时惊人，仿佛他彻底化为了邪魔。
<br/>　　不过，刚刚修炼了片刻，苏莫又突然睁开了眼眸，停止了修炼。
<br/>　　苏莫面露沉吟之色，这修炼室内灵气如此浓郁，自己的本体不修炼一番，真是浪费了。
<br/>　　本体才是他的根本，至于这巫尘的肉身，随时都有可能舍弃的。
<br/>　　只是，两个身体，能不能一起修炼呢？
<br/>　　苏莫手臂一挥，将自己的本体放了出来，与他相对而坐。
<br/>　　随即，他分出一缕意念进入本体的体内，然后尝试着控制自己的本体修炼。
<br/>　　但是，尝试了片刻之后，他不禁暗叹一声，根本行不通啊！
<br/>　　意念，根本无法支撑身体正常的修炼。
<br/>　　“算了！还是先让这具肉身的实力提升吧！”苏莫摇了摇头，便准备将自己的本体收起来。
<br/>　　“苏莫，汝这般是行不通的！”这时，阎魔鬼圣的声音突然在苏莫的脑海中响了起来。
<br/>　　“鬼圣前辈，你有没有办法让我本体也能修炼？”苏莫沉声问道。
<br/>　　“有！”阎魔鬼圣肯定的说道。
<br/>　　“哦？什么方法？”苏莫闻言顿时大喜，若是能两个身体同时修炼，那真是再好不过了。
<br/>　　“将汝现在这具肉身，炼化为分身！”阎魔鬼圣说道。
<br/>　　“分身？”
<br/>　　苏莫立刻眼眸一亮，急忙问道：“如何炼化为分身？”
<br/>　　若是炼化为分身，那真是再好不过了，对于他有莫大的好处。
<br/>　　这可是真正的分身，不是魔影分身那种虚幻之物，能一直存在。
<br/>　　阎魔鬼圣闻言，沉默了片刻，道：“吾有一种炼制分身之法，但汝的情况略有不同，倒是可以进行一种特殊的尝试！”
<br/>　　“什么特殊的尝试？”苏莫好奇了起来。
<br/>　　“你有大量的战魂，可以尝试炼制独立的分身，拥有独立的战魂！”阎魔鬼圣沉声说道。
<br/>　　苏莫闻听此言，顿时心中微震，他立刻便明白了对方的意思。
<br/>　　对方的意思，是要他将巫尘的肉身炼制成分身，而且拥有独立的战魂。
<br/>　　“这……可行吗？”苏莫不确定的问道，这若是可行的话，他就能炼制大量的分身，而且每一个都强大不已。
<br/>　　我去！这要是与人对战，几十个乃至几百个分身一起冲上去，这不是欺负人吗？
<br/>　　“不清楚！但想来应该可行，汝告诉的方法，你可以一试！”阎魔鬼圣不确定的说道。
<br/>　　苏莫闻言沉默了，这种分身他当然想要，他在思索阎魔鬼圣的用意。
<br/>　　此魔传他珍贵无比的圣级上品功法混天魔功，又让他炼制这种独立的分身，每一种都是能让他实力暴增，分明就是在为他走向巅峰强者铺路。
<br/>　　对方会这么好心？
<br/>　　无私的奉献？
<br/>　　苏莫不太相信，对方必有目的，绝对不可能这么简单。
<br/>　　见苏莫沉默不语，阎魔鬼圣问道：“怎么？汝不信吾？”
<br/>　　“不是不信，我只是在考虑这种分身的可行性而已！”苏莫没有承认。
<br/>　　“苏莫，吾告诉你炼制这种分身的方法，待你离开巫族之后，便放吾自由如何？”阎魔鬼圣郑重的说道。
<br/>　　“放你自由？”苏莫闻言有些迟疑，阎魔鬼圣对于他来说，就是个知识的大宝库，他并不放了对方。
<br/>　　“不错！吾教了汝不少东西，传了汝最巅峰的功法，再教你炼制独立分身的功法，应该足够换取自由了吧！”
<br/>　　阎魔鬼圣声音低沉，言罢，又继续道：“汝大可放心，吾传你的一切都毫无问题，汝放心修炼便可！”
<br/>　　苏莫沉默了片刻，随即重重的点了点头，道：“好，我答应你，离开巫族之后，便放你自由！”
<br/>　　苏莫虽然不完全相信对方，但也没有多少担心，他虽然学习了对方的魔功，但只是巫尘这具肉身修习了而已。
<br/>　　他的本体，可没有学习对方的任何武学，就算有问题也不会波及到本体。
<br/>　　他同意了对方的要求，对方帮了他很多，他也不想将对方永远囚困。
<br/>　　“好！希望汝言而有信！”
<br/>　　阎魔鬼圣心绪略有激动，随即又道：“吾现在便告诉你炼制这种分身的方法，你自己尝试着修炼！”
<br/>　　“这种分身，是要靠分化神魂来掌控，而你战魂众多……！”
<br/>　　阎魔鬼圣徐徐的将如何炼制分身的方法，详细的告知了苏莫，其中的要点也一一说明。
<br/>　　这种分身炼制难度并不大，就是通过分化神魂的方法，让得分身有独立的思想，而又完全受本体控制。
<br/>　　当然，因为苏莫拥有很多战魂，要炼制拥有战魂的独立分身，其中方法需要有一些改变。
<br/>　　听完了阎魔鬼圣的话，苏莫对于这种独立分身，已经有了清晰的了解。
<br/>　　随后，他也没有急着炼制分身，收起本体之后，便继续修炼混天魔功。
<br/>　　眼下，一切事情都要放一放，三日之后他还要面对众多巫族天才的挑战呢！
<br/>　　这是他在巫族彻底扬名的时候，也是他在巫族崛起的契机。
<br/>　　时间在修炼中缓缓而逝，三日时间眨眼即过。
