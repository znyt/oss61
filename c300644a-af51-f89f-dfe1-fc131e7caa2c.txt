        三年后……
<br/>
<br/>        滨江城，私宅。
<br/>
<br/>        夜晚，偌大的卧室里只亮着一盏落地灯。
<br/>
<br/>        傅夜沉躺在床上，目光炯炯地看着站在床边正在脱衣服的女子，女子的身材虽然纤细，却是性感迷人，他看不清楚女子的脸，但似乎听到浅笑声。
<br/>
<br/>        她在对他笑吗？
<br/>
<br/>        接着女子一丝不挂的走到他身边，他伸手将女子抱上床，压制在身下……
<br/>
<br/>        “筱颖，我们再来一次，嗯？”
<br/>
<br/>        “筱颖？”
<br/>
<br/>        傅夜沉猛地清醒，明亮的阳光从落地窗外洒进室内，让他不由得微眯起眼。
<br/>
<br/>        天亮了？
<br/>
<br/>        他又左右看了看，这里是他的房间，而床上……只有他一个人。
<br/>
<br/>        傅夜沉躺在床上思考了好一会儿，就算不想承认，但却是不争的事实。
<br/>
<br/>        昨晚，他作了个暧昧的梦，最后他听到自己喊着怀中女人的名字时，人也因此从梦中惊醒。
<br/>
<br/>        “该死！”傅夜沉低低骂了自己一声，高大的身躯坐了起来，他揉了揉额际，试图让脑袋清醒一点。
<br/>
<br/>        他不明白自己为什么会作这样的暧昧的梦，明明他已经很努力地去尝试要忘掉对筱颖所有的爱。
<br/>
<br/>        这三年来，他一直忙于家企里的事情，把傅氏集团推向了巅峰，旗下囊括各大领域，甚至还涉猎了娱乐圈。
<br/>
<br/>        极有魄力的他，自爷爷傅炎廷手中正式接管傅氏集团之后，其营业额翻了数倍，个人资产还上了富豪排行榜前三名。
<br/>
<br/>        网络上一时间因他的上榜而炸开了锅。
<br/>
<br/>        一个极有财力和实力的黄金单身汉，长得有型又有能力，自然吸引异性的注意，想和他沾上关系的人不少，想受他青睐的女人更是多如过江之鲫。
<br/>
<br/>        只是直至今日，他身边仍然没有一个交往对象，却也因此让更多女人想要抓到他的心。
<br/>
<br/>        或许，是因为他作为一名成年男性，从未交过女朋友，又禁欲多时，才会做那种梦的缘故吧！
<br/>
<br/>        可是，他梦里的对象，为什么会是筱颖？
<br/>
<br/>        筱颖……
<br/>
<br/>        不知道，她这三年来，在澳洲过得好不好？
<br/>
<br/>        自从那日，他偷偷地目送她的离开，就不曾再听过有关于她的任何消息了。
<br/>
<br/>        他一直在强忍着自己不去打听有关于她童筱颖的任何事情，连带着，他也有三年未和弟弟傅昼景联系过。
<br/>
<br/>        傅夜沉下了床，走出房间，进入浴室冲澡。
<br/>
<br/>        早上八点多，傅夜沉穿戴整齐后，从楼上下来，来到餐厅时，坐在旁边餐桌前吃早餐的连华生对他道了声：“早上好。”
<br/>
<br/>        傅夜沉走到餐桌前坐下，准备吃早餐。
<br/>
<br/>        桌上除了吐司和水果拼盘，还有荷包蛋和培根。
<br/>
<br/>        傅夜沉才刚坐下，吃没几口，就听到连华生问：“阿沉，筱颖最近过得好不好？”
<br/>
<br/>        闻言，也不知道是不是因为心虚，傅夜沉冷不防地被刚吃进嘴里的吐司噎着了，猛咳了好几下。
<br/>
<br/>        “你没事吧？慢点吃，又没有人跟你抢！”连华生关心地说道。
<br/>
<br/>        傅夜沉缓了缓内心的紧张后，抬眸看着连华生，心想着连华生虽然是医生，但又不是心理医生，更不会读心术，一定不知道他昨晚作了个暧昧的梦的事情。
<br/>
<br/>        但是，这一大清早的，他又为何会突然问起筱颖？
<br/>
<br/>        “你突然提她做什么？”
<br/>
<br/>        “这三年来，你对‘童筱颖’这三个字，是只字不提。而作为一个男人，你都二十七岁了，还从未交过女朋友，也不找床伴满足一下生理需求。我怕你这三年来在心里憋出什么毛病来，所以，才跟你提一提‘筱颖’，好让你整个人活得有灵魂点！”连华生打趣的笑容中带着刚起床不久的惺忪。
<br/>
<br/>        傅夜沉不禁微眯起黑眸，冷冷地反问：“你很无聊？”
<br/>
<br/>        “阿沉，三年前，你没抓住筱颖，现在，就应该学会放下，去发现另一段值得你去拥有的感情。这世上，可不止童筱颖一个女人。还有更多的女人，比童筱颖更优秀，更加值得你去爱。所以……”
<br/>
<br/>        连华生那滔滔不绝的话语都还没说完，傅夜沉没吃几口早餐，就从椅子上起身。
<br/>
<br/>        “够了，以后不要再在我面前提起‘筱颖’，我自己的私事，我会自己看着办。”
<br/>
<br/>        傅夜沉冷冷地说完后，随之走向门口，出门前，他看了眼坐在餐桌前的连华生。
<br/>
<br/>        只见连华生向他微微耸肩，抬起手来做了一个“OK”的手势。
