Party上的众人已经惊呆了，本来夏夕绾穿着这条正品星空长裙出现时，大家就够震惊的了，刚才又听老太君一口一个“奶奶”的自称，他们都怀疑自己出现幻听了。
<br/>
<br/>        现在夏夕绾拿出了一个猫耳朵的耳套作为生日礼物送给老太君，大家当即群起而愤之，她这个礼物也太敷衍了吧！
<br/>
<br/>        要知道他们送的礼物可都是价值连城的珍珠玛瑙，已经在那边堆积如山了，像老太君这种身份的人，连那些珍珠玛瑙都没有多看上一眼，夏夕绾竟然拿了一个猫耳朵耳套就过来献丑了。
<br/>
<br/>        众人纷纷看向厉老夫人：老太君，将她的猫耳朵耳套收下，然后用力丢掉，让她知道江湖的险恶！
<br/>
<br/>        厉老夫人看着这个猫耳朵耳套，迅速收下了，众人双目一喜，对，就是这样的，好样的！
<br/>
<br/>        但是下一秒厉老夫人就将猫耳朵耳套戴在了自己的头上，她开心的咧嘴笑，还献宝似的转了一圈给围观的众人看，“绾绾丫头给我亲手编织的猫耳朵耳套真的好暖和哦，你们都来看一看，好看吧？”
<br/>
<br/>        厉老夫人抓着两个猫耳朵连着做了好几个可爱卖萌的动作。
<br/>
<br/>        众人被雷到了，说好了用力丢进垃圾桶里的呢，说好了教夏夕绾做人的呢，为什么他们反而被狠狠的打了一个耳光。
<br/>
<br/>        众人迅速赔笑，
<br/>
<br/>        好看！真好看！
<br/>
<br/>        绾绾小姐的手好巧啊！
<br/>
<br/>        有了绾绾小姐亲手编织的这个猫耳朵耳套，这个冬天老太君再也不怕冷了！
<br/>
<br/>        夜滢和厉嫣然僵在原地，已经忘记了呼吸，厉嫣然长这么大都没有受过老夫人这样的疼爱，现在在整个帝都上流贵族的面前，老夫人这样抬举夏夕绾，简直是将她们母女当成了空气。
<br/>
<br/>        “绾绾丫头，你送的生日礼物我真是太喜欢了，我一刻都舍不得摘了。”厉老夫人拉着夏夕绾的小手笑道。
<br/>
<br/>        夏夕绾也没有想到厉老夫人会这样喜欢她送的礼物，她嘴甜道，“老夫人，你喜欢就好。”
<br/>
<br/>        “绾绾丫头，我现在要许愿了，过来，我们一起去。”厉老夫人迅速将夏夕绾拉到了蛋糕边。
<br/>
<br/>        李嬷嬷将蜡烛点了起来，厉老夫人迅速双手合十，对着蜡烛许了愿。
<br/>
<br/>        现在所有人都围了过来，将厉老夫人和夏夕绾簇拥在了中间，夏夕绾拍着小手，开始唱生日快乐歌happy..to.you，happy..to.you…
<br/>
<br/>        大家也伸出手跟上了夏夕绾的节奏，一起给厉老夫人唱生日快乐歌。
<br/>
<br/>        现场的气氛十分热络，但是夜滢厉嫣然母女已经被遗忘在了角落里，尤其是厉嫣然身上还穿着那件试衣样品，整个人就是个大笑话。
<br/>
<br/>        厉嫣然幽怨的看着现在被簇拥在火光中央的夏夕绾，她现在的位置本来是属于她的，但是她已经抢走了她所有的光环。
<br/>
<br/>        厉嫣然心里的一切小算计都被打破了，现在夏夕绾反而像那只凤凰，而她才是野鸡。

<br/>        曾经心里有多期待，现在就有多打击。
